# Adjoint of a Matrix

**Input:** None.

**Goal:** Return the adjoint of the following matrix:

$$\begin{bmatrix} 1 + 5i & 2 \\ 3 - 6i & 4i \end{bmatrix}$$

## Solution Notes

To get the adjoint, you perform both transpose and conjugate operations on the input matrix, in any order.
The easiest way is to start with the conjugate of the matrix that you found in the previous exercise, and then transpose it.

Source: Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function MatrixAdjoint() : Complex[][] {
        return [
            [Complex(1., -5.), Complex(3., 6.)],
            [Complex(2., 0.), Complex(0., -4.)]
        ];
    }
}
```
