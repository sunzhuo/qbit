# Measurement in XX basis

**Input**: Two qubits stored in an array which are guaranteed to be either in a superposition of the states $\ket{++}$ and $\ket{--}$ or in a superposition of states $\ket{+-}$ and $\ket{-+}$.

**Output**: 0 if qubits were in the first superposition, 1 if they were in the second superposition.  
The state of the qubits at the end of the operation should be the same as the starting state.

## Solution Notes

This time, you need to do the $X \otimes X$ measurement on both qubits.

The state $\alpha \ket{++} + \beta \ket{--}$ is an eigenstate of the $X \otimes X$ operator with the eigenvalue $+1$, and the state $\alpha \ket{+-} + \beta \ket{-+}$ is an eigenstate with the eigenvalue $-1$.
Hence, you can use this joint measurement to recognize which of the superposition states you were given while preserving the initial superposition state.

Source: Solution.qs

```qsharp
namespace Kata {
    operation XXMeasurement(qs : Qubit[]) : Int {
        return Measure([PauliX, PauliX], qs) == Zero ? 0 | 1;
    }
}
```
