# Bit Flip Code: Encode Codewords

**Input**: three qubits in the state $\ket{\psi} \otimes \ket{00}$, where $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$ is the state of the first qubit, i.e., `qs[0]`.

**Goal**: prepare a state $\alpha \ket{000} + \beta \ket{111}$ on these qubits.

## Solution Notes

You can do this using $CNOT$ gates with the first qubit as control. This will keep the term $\alpha \ket{000}$ unchanged and flip $\beta \ket{100}$ to $\beta \ket{111}$.

Source: Solution.qs

```qsharp
namespace Kata {
    operation BitflipEncode (qs : Qubit[]) : Unit is Adj + Ctl {
        CNOT(qs[0], qs[1]);
        CNOT(qs[0], qs[2]);
    }
}
```
