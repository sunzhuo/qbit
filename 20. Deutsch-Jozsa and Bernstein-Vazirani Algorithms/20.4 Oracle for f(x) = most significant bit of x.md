# Oracle for f(x) = most significant bit of x

**Input:** An array of $N$ qubits in an arbitrary state $\ket{x}$.

**Goal:** Apply the phase oracle $U_f$ for $f(x) = \text{most significant bit of } x$ to the qubits.
That is, apply a relative phase $(-1)^{f(x)}$ to each basis state $\ket{x}$.
$x$ is encoded using big endian, with the most significant bit stored first.

## Solution Notes

The binary representation of $x$ is $x = (x_{0}, x_{1}, \dots, x_{N-1})$, with the most significant bit encoded in the first bit (stored in the first qubit of the input array). Then, you can rewrite the function as

$$f(x) = x_0$$

and its effect on the quantum state as 

$$U_f \ket{x} = (-1)^{f(x)} \ket{x} = (-1)^{x_0} \ket{x} = (-1)^{x_0} \ket{x_{0} } \otimes \ket{x_1} \otimes \cdots \otimes \ket{x_{N-1}}$$

As you've seen in the previous exercise, this can be achieved by applying a $Z$ gate to the first qubit.

Source: Solution.qs

```qsharp
namespace Kata {
    operation PhaseOracle_MostSignificantBit(x : Qubit[]) : Unit is Adj + Ctl {
        Z(x[0]);
    }
}
```
