# |0〉 或 |1〉？

**输入：** 保证处于 $\ket{0}$ 或 $\ket{1}$ 状态的量子位。

**输出：** 如果量子位处于 $\ket{1}$ 状态，则为 `true`；如果处于 $\ket{0}$ 状态，则为 `false`。操作结束时量子位的状态并不重要。

## 解决方案注释

输入量子位保证处于基本状态 $\ket{0}$ 或 $\ket{1}$。这意味着在计算基础上测量量子位时，测量将毫无疑问地报告输入状态。

在 Q# 中，操作 `M` 可用于测量计算基础中的单个量子位。测量结果是 `Result` 类型的值 - 如果输入量子位处于 $\ket{1}$ 状态，则操作 `M` 将返回 `One`；如果输入量子位处于 $\ket{0}$ 状态，则操作 `Zero` 将返回。由于我们需要将第一种情况编码为`true`，第二种情况编码为`false`，因此我们可以返回测量结果与`One`相等比较的结果。

来源：Solution.qs

```qsharp
namespace Kata {
    operation IsQubitOne (q : Qubit) : Bool {
        return M(q) == One;
    }
}
```
