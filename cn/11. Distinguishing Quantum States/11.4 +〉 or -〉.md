# |+〉 或 |-〉？

**输入：** 保证处于 $\ket{+}$ 或 $\ket{-}$ 状态的量子位。

**输出：** 如果量子位处于 $\ket{+}$ 状态，则为 `true`；如果处于 $\ket{-}$ 状态，则为 `false`。操作结束时量子位的状态并不重要。

## 解决方案注释

两个输入状态都是叠加状态，两个基态的振幅绝对值相等。这意味着如果在泡利 $Z$ 基础上测量状态，则有 50-50 的机会测量 `One` 或 `Zero`，这不会为我们提供必要的信息。

为了确定输入量子位处于哪种状态，我们希望将量子位转换为相对于我们执行测量的基础不存在叠加的状态。

考虑如何准备输入状态，从基本状态开始：$H\ket{0} = \ket{+}$ 和 $H\ket{1} = \ket{-}$。也可以通过再次应用 $H$ 门来撤消此转换（请记住，$H$ 门是自伴的，即，它等于其自身的逆）：$H\ket{+} = \ket{0}$ 和 $H\ket{-} = \ket{1}$。

一旦我们有了 $\ket{0}$ 或 $\ket{1}$ 状态，我们就可以使用与先前任务 $\ket{0}$ 或 $\ket{1}$ 相同的原理来测量状态并报告结果。请注意，在此任务中，返回值 `true` 对应于输入状态 $\ket{+}$，因此我们将测量结果与 `Zero` 进行比较。

来源：SolutionA.qs

```qsharp
namespace Kata {
    operation IsQubitPlus(q : Qubit) : Bool {
        H(q);
        return M(q) == Zero;
    }
}
```

#### 替代解决方案

另一种可能的解决方案是在泡利 $X$ 基础（$\ket{+}, \ket{-}$ 基础）中进行测量，这意味着在测量之前不需要使用 $H$ 门进行转换。同样，测量结果 `Zero` 将对应于状态 $\ket{+}$。

在 Q# 中，可以通过 `Measure()` 操作以另一个泡利基进行测量。

来源：SolutionB.qs

```qsharp
namespace Kata {
    operation IsQubitPlus(q : Qubit) : Bool {
        return Measure([PauliX], [q]) == Zero;
    }
}
```
