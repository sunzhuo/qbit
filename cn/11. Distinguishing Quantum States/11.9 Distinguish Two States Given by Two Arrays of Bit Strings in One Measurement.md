# 区分一次测量中两个位串数组给出的两种状态

**输入：**

1. $N$ 量子位（存储在长度为 $N$ 的数组中）保证处于给定位串数组描述的两种叠加状态之一。
2. 两个位串数组，表示为 `Bool[][]`。
数组的维度为 $M_1 \times N$ 和 $M_2 \times N$
 分别，其中 $N$ 是量子位的数量，$M_1$ 和 $M_2$ 是每个数组中的位串的数量。请注意，一般来说$M_1 \neq M_2$。
位串数组 `[b₁, ..., bₘ]` 定义的状态是由位串 $b_1, ..., b_m$ 定义的所有基本状态的相等叠加。
例如，位串数组`[[false, true, false], [false, true, true]]`定义叠加状态$\frac{1}{\sqrt2}\big(\ket{010} + \ket{011}\big)$。

您可以保证存在一个量子位 Q 的索引，其中：

* 第一个数组中的所有位串在此位置具有相同的值（所有 `bits1[j][Q]` 都相同），
* 第二个数组中的所有位串在此位置具有相同的值（所有 `bits2[j][Q]` 都相同），
* 第一个和第二个数组的这些值不同。

**输出：**

* 0 如果量子位处于第一个数组描述的叠加状态，
* 1 如果它们处于第二个数组描述的叠加状态。

> 例如，对于数组 `[[false, true, false], [false, true, true]]` 和 `[[true, false, true], [false, false, true]]`，返回 0 对应状态 $\frac{1}{\sqrt2}\big(\ket{010} + \ket{011}\big)$，返回 1 对应状态 $\frac{1}{\sqrt2}\big(\ket{101} + \ket{001}\big)$，您可以通过测量第二个量子位来完美地区分这些状态。

**您可以仅使用一种测量。**操作结束时量子位的状态并不重要。

## 解决方案注释

与之前的解决方案一样，我们正在寻找两个位串不同的索引 Q。让我们定义一个函数 `FindFirstSuperpositionDiff()` 来搜索具有两个属性的索引 Q：

1. 索引 Q 处的 `bits1` 中的所有数组的值为 `true` 或 `false`，索引 Q 处的 `bits2` 中的所有数组的值为 `true` 或 `false`。如果不是这种情况，您无法确定测量相应的量子位总是返回相同的结果。
   > 例如，如果给定状态 $\frac{1}{\sqrt2}\big(\ket{010} + \ket{011}\big)$，并且测量第三个量子位，时间 $50\%$，您将得到时间 $0$ 和 $50\%$，时间 $1$，因此要获得可靠的信息，您需要测量前两个量子位之一。

2. `bits1` 和 `bits2` 的该值不同。
   > 事实上，如果您想区分状态 $\frac{1}{\sqrt2}\big(\ket{010} + \ket{011}\big)$ 和 $\frac{1}{\sqrt2}\big(\ket{000} + \ket{001}\big)$，有两个量子位将为每个状态产生固定的测量结果 - 第一个和第二个。然而，测量第一个量子位将为两种状态提供 $0$，而测量第二个量子位将为第一个状态提供 $1$，为第二个状态提供 $0$，从而允许区分状态。

为此，我们将迭代所有量子位索引，并且对于每个量子位索引，我们将计算 `bits1` 和 `bits2` 中该位置 1 的数量。

1. 第一个条件意味着该计数应等于 0（如果所有位串在该位置都有 0 位）或位串数组的长度（如果所有位串在该位置都有 1 位）。
2. 第二个条件意味着 `bits1` 和 `bits2` 的计数不同，即其中一个计数应等于 0，另一个计数应等于相应位串数组的长度。

第二步与上一个练习非常相似：给定我们刚刚找到的索引，我们测量该位置上的量子位。
这里我们使用库函数`ResultAsBool(M(qs[diff]))`，如果测量结果是`One`，则返回`true`，如果结果是`Zero`，则返回`false`；对此库函数的调用相当于上一个任务中使用的比较 `M(qs[diff]) == One`。

```qsharp
namespace Kata {
    import Std.Convert.*;

    function FindFirstSuperpositionDiff(bits1 : Bool[][], bits2 : Bool[][], nQubits : Int) : Int {
        for i in 0..nQubits - 1 {
            // count the number of 1s in i-th position in bit strings of both arrays
            mutable val1 = 0;
            mutable val2 = 0;
            for j in 0..Length(bits1) - 1 {
                if bits1[j][i] {
                    set val1 += 1;
                }
            }
            for k in 0..Length(bits2) - 1 {
                if bits2[k][i] {
                    set val2 += 1;
                }
            }
            if (val1 == Length(bits1) and val2 == 0) or (val1 == 0 and val2 == Length(bits2)) {
                return i;
            }
        }

        return -1;
    }

    operation SuperpositionOneMeasurement(qs : Qubit[], bits1 : Bool[][], bits2 : Bool[][]) : Int {
        let diff = FindFirstSuperpositionDiff(bits1, bits2, Length(qs));

        let res = ResultAsBool(M(qs[diff]));

        if res == bits1[0][diff] {
            return 0;
        } else {
            return 1;
        }
    }
}
```
