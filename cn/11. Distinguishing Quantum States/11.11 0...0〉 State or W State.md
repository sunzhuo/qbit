# |0...0〉 状态还是 W 状态？

**输入：** 
$N$ 量子位（存储在长度为 $N$ 的数组中）保证处于 $\ket{0...0}$ 状态或 W 状态 - 所有 $N$ 基本状态的相等叠加，其中恰好有一个 $\ket{1}$。 （例如，对于 $N = 3$，W 状态为 $\frac1{\sqrt3} (\ket{100} + \ket{010} + \ket{001})$）。

**输出：**
* 0 如果量子位处于 $\ket{0...0}$ 状态，
* 1 如果他们处于 W 状态。

操作结束时量子位的状态并不重要。

## 解决方案注释

我们可以看到，W 状态中的每个基础状态总是恰好有一个处于 $\ket{1}$ 状态的量子位，而在 $\ket{0...0}$ 状态下，所有量子位都处于 $\ket{0}$ 状态。

我们可以用它来得出解决方案：我们将计算在 `One` 状态下测量的量子位的数量；如果这个数字等于 1，我们就处于 W 状态，如果它等于 0，我们就知道它是 $\ket{0...0}$ 状态。

> 请注意使用可变变量 `countOnes` 来存储在 `One` 状态下测量的量子位数量，并使用三元运算符 `condition ? trueValue | falseValue` 来表示返回值。

来源：SolutionA.qs

```qsharp
namespace Kata {
    operation AllZerosOrWState(qs : Qubit[]) : Int {
        mutable countOnes = 0;

        for q in qs {
            if M(q) == One {
                set countOnes += 1;
            }
        }

        return countOnes == 0 ? 0 | 1;
    }
}
```

`MeasureInteger()` 也可以用于此任务以使解决方案更短。

来源：SolutionB.qs

```qsharp
namespace Kata {
    operation AllZerosOrWState(qs : Qubit[]) : Int {
        return MeasureInteger(qs) == 0 ? 0 | 1;
    }
}
```
