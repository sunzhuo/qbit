# 区分两个位串数组给出的两种状态

**输入：**

* $N$ 量子位（存储在长度为 $N$ 的数组中）保证处于给定位串数组描述的两种叠加状态之一。
* 两个位串数组，表示为 `Bool[][]`。
  这些数组以与上一个任务中相同的方式描述叠加状态，即它们分别具有维度 $M_1 \times N$ 和 $M_2 \times N$，$N$ 是量子位的数量。

对位串的唯一限制是**两个数组中的所有位串都是不同的**。

**输出：**
* 0 如果量子位处于第一个数组描述的叠加状态，
* 1 如果它们处于第二个数组描述的叠加状态。

> 例如，对于位串`[[false, true, false], [false, false, true]]`和`[[true, true, true], [false, true, true]]`返回0对应于状态 
$\frac{1}{\sqrt2}\big(\ket{010} + \ket{001}\big)$， 
返回 1 到状态 
$\frac{1}{\sqrt2}\big(\ket{111} + \ket{011}\big)$。

**您可以根据需要使用任意多个测量值**。操作结束时量子位的状态并不重要。

## 解决方案注释

因为所有位串都保证是不同的，并且我们可以执行的测量数量不受限制，所以我们可以使用比上一个任务更简单的解决方案。

当我们测量某个叠加态的所有量子位时，它会折叠为构成叠加态的基向量之一。我们可以准确地做到这一点，并将结果状态与给定的位字符串进行比较，以查看它属于哪个数组。

我们使用内置的库原语。首先，我们根据标准计算基础测量量子寄存器，即使用 `MeasureInteger()` 运算将 `PauliZ` 的本征基转换为整数。

接下来，我们使用 `BoolArrayAsInt()` 函数将每个输入位字符串转换为整数。在将位转换为整数时，这两个函数都使用小端编码。

现在我们有了两个整数，我们可以轻松地将测量结果与第一个数组中的每个位串进行比较，以检查它们是否属于该数组；如果他们这样做了，我们就知道我们被赋予了第一个状态，否则就是第二个状态。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Convert.*;

    operation SuperpositionMeasurement(qs : Qubit[], bits1 : Bool[][], bits2 : Bool[][]) : Int {
        let measuredState = MeasureInteger(qs);
        for s in bits1 {
            if BoolArrayAsInt(s) == measuredState {
                return 0;
            }
        }

        return 1;
    }
}
```
