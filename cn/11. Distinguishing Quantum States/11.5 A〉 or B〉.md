# |A〉 或 |B〉？

**输入：**

1. 角度 $\alpha$，以弧度为单位，表示为 `Double`。
2. 保证处于 $\ket{A}$ 或 $\ket{B}$ 状态的量子位，其中
$$\ket{A} = \cos \alpha \ket{0} + \sin \alpha \ket{1}$$ 
$$\ket{B} = - \sin \alpha \ket{0} + \cos \alpha \ket{1}$$

**输出：** 如果量子位处于 $\ket{A}$ 状态，则为 `true`；如果处于 $\ket{B}$ 状态，则为 `false`。操作结束时量子位的状态并不重要。

## 解决方案注释

我们采用与上一个任务类似的方法：找出一种从基本状态准备输入状态的方法，并在测量量子位之前应用该准备的伴随。

为了创建输入状态 $\ket{A}$ 和 $\ket{B}$，将具有 $\theta = 2\alpha$ 的 $R_y$ 门应用于基础状态 $\ket{0}$ 和 $\ket{1}$。提醒一下，

$$R_y(\theta) = \begin{bmatrix} \cos\frac{\theta}{2} & -\sin\frac{\theta}{2} \\ \sin\frac{\theta}{2} & \cos\frac{\theta}{2} \end{bmatrix}$$

我们可以通过将 $R_y$ 门以 $-2\alpha$ 作为旋转角度参数应用到输入量子位，将输入状态返回到基础状态。

The measurement in Pauli $Z$ basis gives two possibilities: 
1. The qubit is measured as $\ket{1}$, the input state was $\ket{B}$, we return `false`.
2. The qubit is measured as $\ket{0}$, the input state was $\ket{A}$, we return `true`.

来源：Solution.qs

```qsharp
namespace Kata {
    operation IsQubitA(alpha : Double, q : Qubit) : Bool {
        Ry(-2.0 * alpha, q);
        return M(q) == Zero;
    }
}
```
