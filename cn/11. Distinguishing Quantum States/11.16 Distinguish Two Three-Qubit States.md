# 区分两个三量子位状态

**输入：** 三个量子位（存储在长度为 3 的数组中），保证处于两个正交状态之一。

**输出：**
* 0 如果他们在该州 
  $\ket{S_0} = \frac{1}{\sqrt{3}} \big(\ket{100} + \omega \ket{010} + \omega^2 \ket{001} \big)$，
* 1 如果他们在该州 
  $\ket{S_1} = \frac{1}{\sqrt{3}} \big(\ket{100} + \omega^2 \ket{010} + \omega \ket{001} \big)$。

这里是$\omega = e^{2i \pi/3}$。

操作结束时量子位的状态并不重要。

## 解决方案注释

让我们找到一个将状态 $\ket{S_0}$ 转换为基础状态 $\ket{000}$ 的酉变换。为此，我们首先应用酉运算将第一个状态映射到 W 状态 
$\frac{1}{\sqrt{3}} \big(\ket{100} + \ket{010} + \ket{001}\big)$。

之后，我们将使用一个方便的旋转门 $R_1$，它将相对相位应用于 $\ket{1}$ 状态，并且不会更改 $\ket{0}$ 状态。
矩阵形式：

$$R_1(\theta) = \begin{bmatrix} 1 & 0 \\ 0 & e^{i\theta} \end{bmatrix} $$

这可以通过张量积来完成 
$I \otimes R_1(-\frac{2\pi}{3}) \otimes R_1(-\frac{4\pi}{3})$，其中

* $I$ 是应用于量子位 0 的恒等门，
* $R_1(-\frac{2\pi}{3}) = \begin{bmatrix} 1 & 0 \\ 0 & \omega^{-1} \end{bmatrix}$，应用于量子位1，
* $R_1(-\frac{4\pi}{3}) = \begin{bmatrix} 1 & 0 \\ 0 & \omega^{-2} \end{bmatrix}$，应用于量子位 2。

> 请注意，将此操作应用于状态 $\ket{S_1}$ 会将其转换为 $\frac{1}{\sqrt{3}} \big (\ket{100} + \omega \ket{010} + \omega^2 \ket{001} \big)$。

现在我们可以使用 W 状态的状态准备例程的伴随（来自准备量子状态 kata 的任务“任意数量的量子位上的 W 状态”），它将把 W 状态映射到状态 $\ket{000}$，将第二个状态映射到其他状态 $\ket{S'_1}$。

我们不需要进行数学计算来找出 $\ket{S'_1}$ 在这两个转换之后 $\ket{S_1}$ 将最终达到的确切状态。请记住，我们的变换是单一的，即它们保留向量的内积。由于状态 $\ket{S_0}$ 和 $\ket{S_1}$ 是正交的，因此在应用酉变换时，它们的内积 $\braket{S_0|S_1} = 0$ 被保留，并且变换后的状态将保持正交。

状态 $\ket{S'_1}$ 保证与状态 $\ket{000}$ 正交，即 $\ket{S_1}$ 映射到不包括基础状态 $\ket{000}$ 的叠加。为了区分状态 $\ket{000}$ 和 $\ket{S'_1}$，我们测量所有量子位；如果所有测量结果均为 0，则状态为 $\ket{000}$，我们返回 0，否则返回 1。

```qsharp
namespace Kata {
    import Std.Convert.*;
    import Std.Math.*;
    import Std.Measurement.*;

    operation ThreeQubitMeasurement(qs : Qubit[]) : Int {
        R1(-2.0 * PI() / 3.0, qs[1]);
        R1(-4.0 * PI() / 3.0, qs[2]);

        Adjoint WState_Arbitrary(qs);

        return MeasureInteger(qs) == 0 ? 0 | 1;
    }

    operation WState_Arbitrary(qs : Qubit[]) : Unit is Adj + Ctl {
        let N = Length(qs);

        if N == 1 {
            X(qs[0]);
        } else {
            let theta = ArcSin(1.0 / Sqrt(IntAsDouble(N)));
            Ry(2.0 * theta, qs[0]);

            X(qs[0]);
            Controlled WState_Arbitrary(qs[0..0], qs[1..N - 1]);
            X(qs[0]);
        }
    }
}
```
