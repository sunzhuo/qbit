# 区分四种贝尔状态

**输入：** 两个量子位（存储在长度为 2 的数组中），保证处于四种贝尔状态之一。

**输出：**
* 0 如果它们处于状态 $\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} \big(\ket{00} + \ket{11}\big)$，
* 1 如果它们处于状态 $\ket{\Phi^{-}} = \frac{1}{\sqrt{2}} \big(\ket{00} - \ket{11}\big)$，
* 2 如果它们处于状态 $\ket{\Psi^{+}} = \frac{1}{\sqrt{2}} \big(\ket{01} + \ket{10}\big)$，
* 3 如果它们处于状态 $\ket{\Psi^{-}} = \frac{1}{\sqrt{2}} \big(\ket{01} - \ket{10}\big)$。
  
操作结束时量子位的状态并不重要。

## 解决方案注释

如果量子位纠缠在贝尔态之一中，则不能简单地测量单个量子位来区分状态：如果这样做，前两个状态都会给你 00 或 11，最后两个状态为 01 或 10。我们需要想出一种方法，在测量之前将原始状态转换为易于区分的状态。

首先，让我们看一下从 $\ket{00}$ 基础状态开始的贝尔状态的准备过程。

> 关于准备所有贝尔态的更详细讨论可以在准备量子态 kata 的任务“所有贝尔态”中找到。

酉变换 $\text{CNOT}\cdot(H \otimes I)$（对应于将 $H$ 门应用于第一个量子位，然后应用 $\text{CNOT}$ 门，将第一个量子位作为控制，将第二个量子位作为目标）将计算基础的 4 个基向量转换为 4 个贝尔态。

$$\text{CNOT}\cdot(H \otimes I) = \frac{1}{\sqrt2} \begin{bmatrix} 1 & 0 & 1 & 0 \\ 0 & 1 & 0 & 1 \\ 0 & 1 & 0 & -1 \\ \underset{\ket{\Phi^{+}}}{\underbrace{1}} & \underset{\ket{\Psi^{+}}}{\underbrace{0}} & \underset{\ket{\Phi^{-}}}{\underbrace{-1}} & \underset{\ket{\Psi^{-}}}{\underbrace{0}} \end{bmatrix}$$

要将贝尔状态变换回基本状态，您可以应用此变换的伴随，这将撤消其效果。在这种情况下，使用的两个门都是自伴随的，因此伴随变换将需要以相反的顺序应用相同的门（首先是 $\text{CNOT}$，然后是 $H$）。

之后，原始状态将变换如下：

这些与我们在“区分四个基本状态”任务中看到的相同的四个两量子位基本状态，尽管与该任务相比顺序不同，因此将测量结果映射到返回值会略有不同。

```qsharp
namespace Kata {
    operation BellState(qs : Qubit[]) : Int {
        CNOT(qs[0], qs[1]);
        H(qs[0]);

        let m1 = M(qs[0]) == Zero ? 0 | 1;
        let m2 = M(qs[1]) == Zero ? 0 | 1;

        return m2 * 2 + m1;
    }
}
```
