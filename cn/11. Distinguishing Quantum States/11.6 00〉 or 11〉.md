#|00〉还是|11〉？

**输入：** 两个量子位（存储在长度为 2 的数组中），保证处于 $\ket{00}$ 或 $\ket{11}$ 状态。

**输出：** 如果量子位处于 $\ket{00}$ 状态，则为 0；如果量子位处于 $\ket{11}$ 状态，则为 1。操作结束时量子位的状态并不重要。

## 解决方案注释

输入数组中的两个量子位都处于相同的状态：对于 $\ket{00}$，每个单独的量子位都处于状态 $\ket{0}$，对于 $\ket{11}$，每个单独的量子位都处于状态 $\ket{1}$。因此，如果我们测量一个量子位，我们就会知道另一个量子位的状态。

换句话说，如果第一个量子位的测量值为 `One`，我们知道输入数组中的量子位处于状态 $\ket{11}$，如果它的测量值为 `Zero`，我们知道它们处于状态 $\ket{00}$。

> `condition ? truevalue | falsevalue` 是 Q# 的三元运算符：如果 `condition` 为 true，则返回 `trueValue`，否则返回 `falseValue`。

来源：Solution.qs

```qsharp
namespace Kata {
    operation ZeroZeroOrOneOne(qs : Qubit[]) : Int {
        return M(qs[0]) == One ? 1 | 0;
    }
}
```
