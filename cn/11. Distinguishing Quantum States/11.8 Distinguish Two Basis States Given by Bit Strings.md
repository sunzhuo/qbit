# 区分由位串给出的两个基本状态

**输入：**

1. $N$ 量子位（存储在长度为 $N$ 的数组中）保证处于给定位串描述的两个基本状态之一。
2. 两个位串表示为 `Bool[]`。

**输出：**

* 0 如果量子位处于第一个位串描述的基础状态，
* 1 如果它们处于第二位串描述的基本状态。

位值`false`和`true`对应于$\ket{0}$和$\ket{1}$状态。可以保证两个位串与量子位数组具有相同的长度，并且位串至少有一位不同。

**您可以仅使用一种测量。** 操作结束时量子位的状态并不重要。

> 示例：对于位串 `[false, true, false]` 和 `[false, false, true]`，返回 0 对应状态 $\ket{010}$，返回 1 对应状态 $\ket{001}$。

## 解决方案注释

为了解决这个任务，我们将使用两个步骤。 Like many other programming languages, Q# allows you to write functions to make code more readable and reusable.

第一步是找到位串 `bit1` 和 `bit2` 之间不同的第一个位。为此，我们定义了一个函数 `FindFirstDiff()`，它循环遍历两个 `Bool[]` 并返回位字符串不同的第一个索引。

第二步是实现主要操作：一旦找到第一个不同的位，我们就测量相应位置的量子位，看看它是否处于状态$\ket{0}$或$\ket{1}$。如果处于状态 $\ket{0}$，`res` 则采用值 `false`，如果处于状态 $\ket{1}$，则采用值 `true`。

`res == bits1[firstDiff]` 将测量结果与不同位置的 `bits1` 的位进行比较。这有效地检查量子位是否处于由第一位串或第二位串描述的基础状态。两种可能的结果是：

1. 量子位处于第一个位串描述的状态；那么 `res` 将等于 `bits1[firstDiff]` 并且该方法将返回 `0`。
2. 量子位处于第二位串描述的状态；那么 `res` 将不等于 `bits1[firstDiff]`（我们知道它必须等于 `bits2[firstDiff]`，而 `bits2[firstDiff]` 不等于 `bits1[firstDiff]`），并且该方法将返回 `1`。

来源：Solution.qs

```qsharp
namespace Kata {
    function FindFirstDiff(bits1 : Bool[], bits2 : Bool[]) : Int {
        for i in 0 .. Length(bits1) - 1 {
            if bits1[i] != bits2[i] {
                return i;
            }
        }
        return -1;
    }

    operation TwoBitstringsMeasurement(qs : Qubit[], bits1 : Bool[], bits2 : Bool[]) : Int {
        let firstDiff = FindFirstDiff(bits1, bits2);
        let res = M(qs[firstDiff]) == One;

        return res == bits1[firstDiff] ? 0 | 1;
    }
}
```
