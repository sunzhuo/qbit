# 区分四种基本状态

**输入：** 两个量子位（存储在长度为 2 的数组中），保证处于四种基本状态之一（$\ket{00}$、$\ket{01}$、$\ket{10}$ 或 $\ket{11}$）。

**输出：**

* 0 如果量子位处于 $\ket{00}$ 状态，
* 1 如果它们处于 $\ket{01}$ 状态，
* 2 如果它们处于 $\ket{10}$ 状态，
* 3 如果它们处于 $\ket{11}$ 状态。

在此任务和后续任务中，任务描述中的量子位状态顺序与数组中的量子位顺序匹配（即，$\ket{10}$ 状态对应于状态 $\ket{1}$ 中的 `qs[0]` 和状态 $\ket{0}$ 中的 `qs[1]`。

操作结束时量子位的状态并不重要。

## 解决方案注释

与之前的任务不同，这次测量第一个量子位不会为我们提供有关第二个量子位的任何信息，因此我们需要测量两个量子位。

首先，我们测量输入数组中的两个量子位并将结果存储在 `m1` 和 `m2` 中。我们可以这样解码这些结果：

* m1 为 $\ket{0}$，m2 为 $\ket{0}$：我们返回 $0\cdot2+0=0$
* m1 为 $\ket{0}$，m2 为 $\ket{1}$：我们返回 $0\cdot2+1=1$
* m1 为 $\ket{1}$，m2 为 $\ket{0}$：我们返回 $1\cdot2+0=2$
* m1 为 $\ket{1}$，m2 为 $\ket{1}$：我们返回 $1\cdot2+1=3$

换句话说，我们将测量结果视为大端表示法的返回值的二进制表示法，最高有效位存储在 `m1` 中，最低有效位存储在 `m2` 中。

来源：Solution.qs

```qsharp
namespace Kata {
    operation BasisStateMeasurement(qs : Qubit[]) : Int {
        let m1 = M(qs[0]) == Zero ? 0 | 1;
        let m2 = M(qs[1]) == Zero ? 0 | 1;
        return m1 * 2 + m2;
    }
}
```
