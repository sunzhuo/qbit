# |0〉 或 |+〉？

**输入：** 保证处于 $\ket{0}$ 或 $\ket{+}$ 状态的量子位。
    
**输出：** 如果量子位处于 $\ket{0}$ 状态，则为 `true`；如果处于 $\ket{+}$ 状态，则为 `false`。操作结束时量子位的状态并不重要。

在此任务中，您的解决方案将被多次调用，每次都会以相同的概率选择其中一个状态。您必须获得至少 80% 的总体准确率。

> 此任务是量子假设检验或最小误差状态辨别的示例。

## 解决方案注释

让 ${\ket{E_a}, \ket{E_b}}$ 是具有两个结果 $a$ 和 $b$ 的测量，我们将其与答案进行识别，即结果“a”表示我们回答“状态为 $\ket{0}$”，结果“b”表示我们回答“状态为 $\ket{+}$”。然后我们定义

* $P(a|0)$ = 假设状态为 $\ket{0}$ 时观察到第一个结果的概率
* $P(b|0)$ = 在状态为 $\ket{0}$ 的情况下观察到第二个结果的概率
* $P(a|+)$ = 假设状态为 $\ket{+}$ 时观察到第一个结果的概率
* $P(b|+)$ = 在状态为 $\ket{+}$ 的情况下观察到第二个结果的概率

任务是最大化单次实验正确的概率，这与最小化单次实验错误的概率相同。

由于该任务承诺输入 $\ket{0}$ 和 $\ket{+}$（即 $P(+) = P(0) = \frac{1}{2}$）的均匀先验分布，我们得到以下给出正确答案的概率表达式：

$$P_{correct} = P(0) P(a|0) + P(+) P(b|+) = \frac{1}{2} (P(a|0) + P(b|+))$$

我们可以将我们的测量表示为以下形式的冯诺依曼测量：

$$\ket{E_a} = R_y(2\alpha) \begin{bmatrix} 1 \\ 0 \end{bmatrix} = \begin{bmatrix} \cos \alpha \\ \sin \alpha \end{bmatrix}$$
$$\ket{E_b} = R_y(2\alpha) \begin{bmatrix} 0 \\ 1 \end{bmatrix} = \begin{bmatrix} - \sin \alpha \\ \cos \alpha \end{bmatrix}$$

使用这种表示，我们可以将概率表示如下：

$$P(a|0) = |\braket{E_a|0}|^2 = \cos^2 \alpha$$
    
$$P(b|+) = |\braket{E_b|+}|^2 = \frac{1}{2} - \cos \alpha \sin \alpha$$
    
$$P_{correct} = \frac{1}{2} (\cos^2 \alpha + \frac{1}{2} - \cos \alpha \sin \alpha)$$
    
最大化$\alpha$，我们得到最大值$P_{success} = \frac{1}{2} (1 + \frac{1}{\sqrt{2}}) = 0.8535...$，这是$\alpha = -\pi/8$获得的。
    
这意味着 $\ket{E_a}$ 和 $\ket{E_b}$ 分别是 $\ket{0}$ 和 $\ket{1}$ 旋转 $-\pi/8$ 的结果。如果我们将整个系统旋转$-\alpha = \pi/8$，我们将得到$\ket{E_a}=\ket{0}$和$\ket{E_b}=\ket{1}$，并且计算基础上的测量将为我们提供正确结果的概率为85％。
    
> 在 Q# 中，可以通过应用具有角度参数 $2\theta$ 的 $Ry$ 门来将输入状态旋转某个角度 $\theta$。

```qsharp
namespace Kata {
    import Std.Math.*;

    operation IsQubitZeroOrPlus(q : Qubit) : Bool {
        Ry(0.25 * PI(), q);
        return M(q) == Zero;
    }
}
```
