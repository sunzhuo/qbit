# 使用 DumpMachine 了解单个量子位的状态

**输入：** 处于未知状态的量子位 $\ket{\psi} = \alpha\ket{0} + \beta\ket{1}$。幅度 $\alpha$ 和 $\beta$ 将为实数且非负。

**输出：** 两个数字的元组 $(\alpha', \beta')$ - 您对幅度 $\alpha$ 和 $\beta$ 的估计。
绝对误差 $|\alpha - \alpha'|$ 和 $|\beta - \beta'|$ 应小于或等于 0.001。

请注意，状态参数保证相同
如果您多次运行该代码。您的操作将被称为
每次运行一次。

<details>
  <summary><b>需要提示吗？</b></summary>
  在物理量子系统上，无法从单个观测中获取 $\alpha$ 和 $\beta$ 的值。由于该程序在模拟器上运行，因此您可以使用 <code>DumpMachine</code> 检查量子位并记下其状态。此外，问题陈述保证每次调用的状态都是相同的。因此，您可以更新代码以返回您记录下的幅度。然后再次运行代码。
</details>

## 解决方案注释

通过调用 `DumpMachine` 了解幅度 $\alpha$ 和 $\beta$ 并返回此函数输出中显示的值。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Diagnostics.*;

    operation LearnSingleQubitState(q : Qubit) : (Double, Double) {
        DumpMachine(); // Only used to learn the amplitudes.
        return (0.9689, 0.2474);
    }
}
```
