# 矩阵的共轭

**输入：** 无。

**目标：** 返回以下矩阵的共轭：

$$\begin{bmatrix} 1 + 5i & 2 \\ 3 - 6i & 4i \end{bmatrix}$$

> 在此任务和接下来的几个任务中，矩阵表示为 Q# `Complex[][]` 类型、复数的二维数组。如果您需要复习如何使用此数据类型，请查看复杂算术类型。

## 解决方案注释

根据矩阵共轭的定义，您只需要用原始矩阵的元素填充转置矩阵，记住翻转数字的虚部符号。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function Conjugate() : Complex[][] {
        return [
            [Complex(1., -5.), Complex(2., 0.)],
            [Complex(3., 6.), Complex(0., -4.)]
        ];
    }
}
```
