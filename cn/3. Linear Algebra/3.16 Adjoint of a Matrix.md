# 矩阵的伴随

**输入：** 无。

**目标：** 返回以下矩阵的伴随：

$$\begin{bmatrix} 1 + 5i & 2 \\ 3 - 6i & 4i \end{bmatrix}$$

## 解决方案注释

要获得伴随矩阵，您可以按任意顺序对输入矩阵执行转置和共轭运算。
最简单的方法是从上一个练习中找到的矩阵的共轭开始，然后转置它。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function MatrixAdjoint() : Complex[][] {
        return [
            [Complex(1., -5.), Complex(3., 6.)],
            [Complex(2., 0.), Complex(0., -4.)]
        ];
    }
}
```
