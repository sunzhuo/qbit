# 概述

本套向您介绍线性代数的基础知识。线性代数是数学的一个分支，致力于研究矩阵和向量的性质，矩阵和向量广泛用于量子计算中来表示量子态及其运算。
此套路并没有涵盖该主题的全部内容，但它可以帮助您开始了解量子计算中使用的线性代数的主要概念。

**此型涵盖以下主题：**

* 矩阵和向量
* 基本矩阵运算
* 复值矩阵的运算和性质
* 内向量积和外向量积
* 张量积

**开始练习这个型时你应该知道什么：**

* 复数的基础知识。如果您需要回顾本主题，可以在解决此主题之前先查看复杂算术型。

*在此 kata 中，练习的重点是纸笔工作，而不是编写程序，因为在开始编写量子计算问题的代码之前，您更有可能先针对量子计算问题进行线性代数计算，而不是在 Q# 中进行。每项任务都要求您填写可以手动完成的特定示例的数学计算结果。 Q# 代码用于检查计算结果，而不是执行计算结果。*
