# 归一化向量

**输入：** 无。

**目标：** 返回归一化向量 $V$，定义为 $\frac{V}{||V||}$，其中

$$V = \begin{bmatrix} -6 \\ 8i \end{bmatrix}$$

> 向量是 $n \times 1$ 矩阵，因此在本练习中，结果表示为二维复数数组，其中两行每行仅包含一个元素。向量通常表示为一维数组。

## 解决方案注释

首先计算向量的范数

$$||V|| = \sqrt{\langle V , V \rangle} = \sqrt{-6 \cdot (-6) + (-8i) \cdot 8i} = \sqrt{6^2 + 8^2} = 10$$

然后将原始向量的每个元素除以其范数即可得到归一化向量：
 
$$\frac{V}{||V||} = \begin{bmatrix} -0.6 \\ 0.8i \end{bmatrix}$$

```qsharp
namespace Kata {
    import Std.Math.*;

    function NormalizedVector() : Complex[][] {
        return [
            [Complex(-0.6, 0.)],
            [Complex(0., 0.8)]
        ];
    }
}
```
