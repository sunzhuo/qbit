# 转置矩阵

**输入：** 无。

**目标：** 返回以下矩阵的转置：

$$\begin{bmatrix} 1 & 2 \\ 3 & 4 \end{bmatrix}$$

## 解决方案注释

根据矩阵转置的定义，您只需要用原始矩阵的元素按正确的顺序填充转置矩阵即可：主对角线上的元素保留在原来的位置，主对角线外的元素被交换。

来源：Solution.qs

```qsharp
namespace Kata {
    function Transpose() : Double[][] {
        return [[1., 3.], 
                [2., 4.]];
    }
}
```
