# 从量子位数组中读取颜色

在此任务中，您需要读取存储在量子位数组中的 $V$ 顶点图的着色。
这是 Grover 搜索算法的最后一步：执行测量并将结果解释为问题的潜在解决方案。

**输入**：

1. 着色[[​​QBIT_PH_0]]每种颜色的位数。
  2. 保证处于基础状态的 $V \cdot nBits$ 量子位数组。

**输出**：

表示此基本状态的 $V$ $nBits$ 位整数数组。
数组的第 $i$ 个整数以大端格式存储在索引为 $i \cdot nBits$、$i \cdot nBits + 1$、...、$(i + 1) \cdot nBits - 1$ 的量子位中。

例如，输入 $nBits = 2$ 和基础状态 $\ket{000110}$ 对应于两位颜色 $00, 01, 10$，或者十进制的 $0, 1, 2$。

该操作不应改变量子位的状态。

## 解决方案注释

本练习的解决方案由两部分组成：

1. 从长度为 `nBits` 的量子位数组中读取一个整数。
   由于可以保证阵列中的量子位处于基本状态，因此只需测量它们即可为您提供必要的信息，并使量子位的状态保持不变。 
   您可以使用库操作 `MeasureEachZ` 测量数组中的每个量子位，而无需随后将其重置为 $\ket{0}$ 状态。 
   然后，您可以使用函数 `ResultArrayAsInt` 将测量结果数组转换为整数。
   请注意，此函数执行从小端二进制编码的转换，并且练习要求颜色以大端表示，因此您需要使用函数 `Reversed` 反转测量结果数组。
2. 将给定的量子位数组拆分为长度为 `nBits` 的块，并从每个块中读取一个整数。
   一个方便的库函数 `Chunks` 正是这样做的，将数组分割成给定长度的块。
   最后，您可以使用操作 `ForEach` 将整数读出操作应用于每个块。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Arrays.*;
    import Std.Convert.*;

    operation ReadColoring(nBits : Int, qs : Qubit[]) : Int[] {
        let colorPartitions = Chunks(nBits, qs);
        let measureColor = qs => ResultArrayAsInt(Reversed(MeasureEachZ(qs)));
        return ForEach(measureColor, colorPartitions);
    }
}
```
