# 弱着色有效吗？ （经典）

**输入：**

1. 图 $V$ ($V \leq 6$) 中的顶点数。
2. $E$ 整数元组数组，表示图的边 ($E \leq 12$)。
每个元组给出边的起始和结束顶点的索引。
顶点编号为 $0$ 到 $V - 1$。
3. $V$ 整数数组，表示图的顶点着色。 
数组的第 $i$ 元素是顶点编号 $i$ 的颜色。

例如，考虑带有 $V = 3$ 和 `edges = [(0, 1), (1, 2), (0, 2)]` 的三角图 `0 -- 1 -- 2 -- 0`。 
尽管连接的顶点 $0$ 和 $2$ 被分配了相同的颜色，但着色 `[0, 1, 0]` 对其来说是有效的弱着色，因为它们每个都还连接到不同颜色的顶点 $1$。

**输出：**
如果给定的弱着色有效（即每个顶点没有邻居，或者至少有一个不同颜色的邻居），则为 true，否则为 false。

## 解决方案注释

当每个顶点都是弱着色时，即没有邻居或者至少有一个不同颜色的邻居时，弱着色是有效的。这意味着该解决方案需要迭代所有顶点并检查每个顶点的此条件，分别跟踪通过边连接到该顶点的顶点数量以及不同颜色的连接顶点的存在性。

来源：Solution.qs

```qsharp
namespace Kata {
    function IsWeakColoringValid(V : Int, edges: (Int, Int)[], colors: Int[]) : Bool {
        for vertex in 0 .. V - 1 {
            mutable neighborCount = 0;
            mutable hasDifferentNeighbor = false;

            for (start, end) in edges {
                if start == vertex or end == vertex {
                    set neighborCount += 1;
                    if colors[start] != colors[end] {
                        set hasDifferentNeighbor = true;
                    }
                }
            }

            if neighborCount > 0 and not hasDifferentNeighbor {
                return false;
            }
        }
        return true;
    }
}
```
