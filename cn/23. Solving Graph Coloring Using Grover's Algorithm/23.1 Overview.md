# 概述

使用 Grover 搜索算法解决经典问题的关键部分是为该问题实现量子预言。
在实践中，针对特定问题实施量子预言可能非常具有挑战性。

这个 kata 将引导您完成针对几种图形着色问题的量子预言 - 
寻找图的顶点或边的标签分配（通常称为“颜色”）的问题 
以一种满足给定约束集的方式。
它还鼓励您尝试使用这些预言来解决 Grover 搜索的图形着色问题。

**此型涵盖以下主题：**

- 针对多个版本的图形着色问题实现标记预言
- 使用Grover的搜索算法解决图着色问题

**开始练习这个型时你应该知道什么：**

- 基本量子概念
- 受控门
- 预言机，特别是标记预言机
- 格罗弗的搜索算法
