# 弱着色有效吗？ （量子）

**输入：**

1. 图 $V$ ($V \leq 6$) 中的顶点数。
2. $E$ 整数元组数组，表示图的边 ($E \leq 12$)。
每个元组给出边的起始和结束顶点的索引。
顶点编号为 $0$ 到 $V - 1$。
3. 处于任意状态 $\ket{x}$ 的 $2V$ 量子位数组，表示使用四种颜色 ($nBits = 2$) 的顶点指定着色，其格式与练习“从量子位数组读取着色”中的格式相同。
4. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。

**目标：**
实现一个量子预言机，检查给定的弱着色对于给定的图是否有效，
也就是说，图的每个顶点是孤立的还是连接到至少一个不同颜色的顶点。
    
让输入寄存器中的量子位保持其开始时的状态。
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

要实现这个预言，您需要检查每个顶点的颜色是否正确，并将各个检查的结果结合起来。您需要分配一组量子位，每个顶点一个，如果相应的顶点颜色较弱，则该数组将被翻转。
使用上一个任务中定义的 `Oracle_WeakColoring_OneVertex` 操作可以轻松完成此操作。然后，您需要使用受控 $X$ 门检查数组中的所有量子位是否处于状态 $\ket{1...1}$；如果是，则着色有效。

像往常一样，请记住在评估最终结果后取消计算对辅助量子位的更改，以便在发布之前保持它们干净。

```qsharp
namespace Kata {
    import Std.Arrays.*;

    operation Oracle_WeakColoring(
        V : Int,
        edges : (Int, Int)[],
        x : Qubit[],
        y : Qubit
    ) : Unit is Adj + Ctl {
        use vertexQubits = Qubit[V];
        within {
            for v in 0..V - 1 {
                Oracle_WeakColoring_OneVertex(V, edges, x, vertexQubits[v], v);
            }
        } apply {
            Controlled X(vertexQubits, y);
        }
    }

    // You might find these helper operations from earlier tasks useful.
    operation Oracle_WeakColoring_OneVertex(
        V : Int,
        edges : (Int, Int)[],
        x : Qubit[],
        y : Qubit,
        vertex : Int
    ) : Unit is Adj + Ctl {
        let neighborEdges = Filtered((a, b) -> a == vertex or b == vertex, edges);
        let nNeighbors = Length(neighborEdges);
        use sameColorChecks = Qubit[nNeighbors];
        within {
            for ((a, b), checkQubit) in Zipped(neighborEdges, sameColorChecks) {
                Oracle_ColorEquality(
                    x[a * 2 .. a * 2 + 1],
                    x[b * 2 .. b * 2 + 1],
                    checkQubit
                );
            }
        } apply {
            X(y);
            if nNeighbors > 0 {
                Controlled X(sameColorChecks, y);
            }
        }
    }

    operation Oracle_ColorEquality(x0 : Qubit[], x1 : Qubit[], y : Qubit) : Unit is Adj + Ctl {
        within {
            for i in 0..Length(x0) - 1 {
                CNOT(x0[i], x1[i]);
            }
        } apply {
            ApplyControlledOnInt(0, X, x1, y);
        }
    }
}
```
