# 顶点着色有效吗？ （古典）

**输入：**

1. 图 $V$ ($V \leq 6$) 中的顶点数。
2. $E$ 整数元组数组，表示图的边 ($E \leq 12$)。
每个元组给出边的起始和结束顶点的索引。
顶点编号为 $0$ 到 $V - 1$。
3. $V$ 整数数组，表示图的顶点着色。 
数组的第 $i$ 元素是顶点编号 $i$ 的颜色。

例如，图`0 -- 1 -- 2`具有$V = 3$和`edges = [(0, 1), (1, 2)]`。 
它的一些有效颜色是 `[0, 1, 0]` 和 `[-1, 5, 18]`。
请注意，在本练习中，与后面的一些练习不同，着色不限于从 $0$ 到 $3$ 的数字。

**输出：**
如果给定的顶点着色有效（即，由边连接的两个顶点都不具有相同的颜色），则为 true，否则为 false。

## 解决方案注释

当每条边连接的节点具有不同的颜色时，图形着色是有效的。这意味着我们必须检查每条边，看看节点是否具有相同的颜色，如果是这种情况，则返回图形着色无效。如果每条边都通过了测试，我们可以有把握地说图形着色是有效的。

由于顶点 $n$ 的颜色是 `colors` 数组的第 $n$ 元素，因此我们只需循环遍历每条边（一对顶点索引），并比较它们的颜色。

来源：Solution.qs

```qsharp
namespace Kata {
    function IsVertexColoringValid(V : Int, edges: (Int, Int)[], colors: Int[]) : Bool {
        for (v0, v1) in edges {
            if colors[v0] == colors[v1] {
                return false;
            }
        }
        return true;
    }
}
```
