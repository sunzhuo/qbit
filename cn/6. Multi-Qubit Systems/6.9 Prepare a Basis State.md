# 准备一个基础状态

**输入：** 处于基本状态 $\ket{00} = \begin{bmatrix} 1 \\ 0 \\ 0 \\ 0 \end{bmatrix}$ 的两个量子位系统。

**目标：** 将系统转变为基础状态 $\ket{11} = \begin{bmatrix} 0 \\ 0 \\ 0 \\ 1 \end{bmatrix}$。

## 解决方案注释

﻿起始状态可以表示如下：
$$ \begin{bmatrix} 1 \\ 0 \\ 0 \\ 0 \end{bmatrix} = \ket{0} \otimes \ket{0} $$

目标状态可以表示如下：
$$ \begin{bmatrix} 0 \\ 0 \\ 0 \\ 1 \end{bmatrix} = \ket{1} \otimes \ket{1} $$

将 $X$ 门应用于 $\ket{0}$ 状态的量子位会将量子位状态转换为 $\ket{1}$ 状态。因此，如果您在第一个量子位和第二个量子位上应用 $X$ 门，您将获得所需的状态。

来源：Solution.qs

```qsharp
namespace Kata {
    operation PrepareBasisState(qs : Qubit[]) : Unit is Adj + Ctl {
        X(qs[0]);
        X(qs[1]);
    }
}
```
