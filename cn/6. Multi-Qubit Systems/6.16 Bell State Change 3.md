# 贝尔状态变化 3

**输入：** 贝尔态 $\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} \big(\ket{00} + \ket{11}\big)$ 下的两个纠缠量子位。

**目标：** 在不添加全局阶段的情况下将两个量子位状态更改为 $\ket{\Psi^{-}} = \frac{1}{\sqrt{2}} \big(\ket{01} - \ket{10}\big)$。

## 解决方案注释

您在单量子位门套路中看到，泡利 Z 门使单量子位叠加的 $\ket{0}$ 分量的符号保持不变，但翻转了叠加的 $\ket{1}$ 分量的符号。您还刚刚在“Bell State Change 2”中看到了如何将我们的输入状态更改为状态 $\frac{1}{\sqrt{2}} \big(\ket{01} + \ket{10}\big)$，这几乎是您的目标状态（暂时忽略阶段变化）。因此，这两个门的组合似乎正是您所需要的。剩下的问题是以什么顺序应用它们以及应用到哪个量子位。

首先，哪个量子位？回顾任务“Bell state change 2”，很明显您需要使用量子位 `qs[0]`，就像您在那里所做的那样。

其次，您应该按什么顺序应用门？请记住，泡利 Z 门会翻转叠加的 $\ket{1}$ 分量的相位，并单独保留 $\ket{0}$ 分量。
我们首先尝试将 $X$ 门应用到 `qs[0]`。查看我们的“半途回答”状态 $\frac{1}{\sqrt{2}} \big(\ket{01} + \ket{10}\big)$，您可以看到，如果将 $Z$ 门应用到 `qs[0]`，它将单独保留 $\ket{0_{A}}$，但将 $\ket{1_{A}}$ 的相位翻转为 $-\ket{1_{A}}$，从而翻转 $\ket{0_{A}}$ 的相位我们贝尔状态的 $\ket{11}$ 组成部分。

来源：Solution.qs

```qsharp
namespace Kata {
    operation BellStateChange3(qs : Qubit[]) : Unit is Adj + Ctl {
        X(qs[0]);
        Z(qs[0]);
    }
}
```
