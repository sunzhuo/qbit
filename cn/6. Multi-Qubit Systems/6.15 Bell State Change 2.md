# 响铃状态变化 2

**输入：** 贝尔态 $\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} \big(\ket{00} + \ket{11}\big)$ 下的两个纠缠量子位。

**目标：** 将两个量子位状态更改为 $\ket{\Psi^{+}} = \frac{1}{\sqrt{2}} \big(\ket{01} + \ket{10}\big)$。

## 解决方案注释

您已经在单量子位门型中看到，Pauli X 门将 $\ket{0}$ 翻转到 $\ket{1}$，反之亦然，并且由于您似乎需要一些状态翻转，也许这个门可能有用。 （当然，请记住，$X$ 门在单个量子位上运行）。

让我们逐项比较起始状态 $\frac{1}{\sqrt{2}} \big(\ket{0_A0_B} + \ket{1_A1_B}\big)$ 和目标状态 $\frac{1}{\sqrt{2}} \big(\ket{1_A0_B} + \ket{0_A1_B}\big)$，看看需要如何转换它才能达到目标。

使用“贝尔状态更改 1”中的术语，您现在可以通过比较术语看到 $\ket{0_{A}}$ 已翻转到 $\ket{1_A}$ 以获得第一项，而 $\ket{1_{A}}$ 已翻转到 $\ket{0_A}$ 以获得第二项。这允许您说要使用的正确门是 Pauli X，应用于 `qs[0]`。

来源：Solution.qs

```qsharp
namespace Kata {
    operation BellStateChange2 (qs : Qubit[]) : Unit is Adj + Ctl {
        X(qs[0]);
    }
}
```
