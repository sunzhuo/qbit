# 使用 DumpMachine 学习基态振幅

**输入：** 2 个处于未知状态的量子位 $\ket{\psi} = \sum_{k = 0}^{3} x_k \ket{k}$。幅度 $x_k$ 将为实数且非负。

**输出：** 两个数字的元组 $(x_1', x_2')$ - 分别对状态 $\ket{01}$ 和 $\ket{10}$ 的幅度的估计。
绝对误差 $|x_1 - x_1'|$ 和 $|x_2 - x_2'|$ 应小于或等于 0.001。

请注意，状态参数保证相同
如果您多次运行该代码。您的操作将被称为
每次运行一次。

<details>
  <summary><b>需要提示吗？</b></summary>
  在物理量子系统上，无法从单个观察中获得这些值。由于该程序在模拟器上运行，因此您可以使用 <code>DumpMachine</code> 检查量子位并记下它们的状态。此外，问题陈述保证每次调用时的状态都是相同的。因此，您可以更新代码以返回您记录下的幅度，然后再次运行代码。
</details>

## 解决方案注释

您可以通过调用`DumpMachine`了解基态的幅度。
之后，您可以将这些值硬编码到您的解决方案中。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Diagnostics.*;

    operation LearnBasisStateAmplitudes(qs : Qubit[]) : (Double, Double) {
        DumpMachine(); // Only used to learn the amplitudes.
        return (0.3390, 0.3821);
    }
}
```
