# 响铃状态变化 1

**输入：** 贝尔态 $\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} \big(\ket{00} + \ket{11}\big)$ 下的两个纠缠量子位。

**目标：** 将两个量子位状态更改为 $\ket{\Phi^{-}} = \frac{1}{\sqrt{2}} \big(\ket{00} - \ket{11}\big)$。

## 解决方案注释

您认识到目标是另一个贝尔州。事实上，它是贝尔四个州之一。

您已经在单量子位门 kata 中看到，Pauli Z 门将改变单个量子位的 $\ket{1}$ 基础状态的状态，因此该门似乎是您想要实现的目标的良好候选者。该门保持叠加的 $\ket{0}$ 基础状态的符号不变，但翻转叠加的 $\ket{1}$ 基础状态的符号。

不要忘记 $Z$ 门仅作用于一个量子位，而这里有两个。
我们还要记住贝尔态是如何由各个量子位组成的。

如果两个量子位是 A 和 B，其中 A 是 `qs[0]`，B 是 `qs[1]`，则可以写成
$\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} \big(\ket{0_{A}0_{B}} + \ket{1_{A}1_{B}}\big)$。
如果将 $Z$ 门应用于量子位 A，它将翻转基本状态 $\ket{1_A}$ 的相位。由于此阶段在某种意义上分布在纠缠态上，$\ket{1_A}$ 基础状态是叠加的后半部分的一部分，因此此应用程序具有翻转整个基础状态 $\ket{1_A1_B}$ 符号的效果，如运行下面的解决方案所示。

如果将 $Z$ 应用于量子位 B，则可以完成完全相同的计算，因此这是另一种可能的解决方案。

来源：Solution.qs

```qsharp
namespace Kata {
    operation BellStateChange1 (qs : Qubit[]) : Unit is Adj + Ctl {
        Z(qs[0]);
    }
}
```
