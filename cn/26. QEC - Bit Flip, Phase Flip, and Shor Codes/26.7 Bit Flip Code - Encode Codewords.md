# 位翻转代码：编码码字

**输入**：处于状态 $\ket{\psi} \otimes \ket{00}$ 的三个量子位，其中 $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$ 是第一个量子位的状态，即 `qs[0]`。

**目标**：在这些量子位上准备状态 $\alpha \ket{000} + \beta \ket{111}$。

## 解决方案注释

您可以使用 $CNOT$ 门并以第一个量子位作为控制来完成此操作。这将使 $\alpha \ket{000}$ 项保持不变，并将 $\beta \ket{100}$ 翻转为 $\beta \ket{111}$。

来源：Solution.qs

```qsharp
namespace Kata {
    operation BitflipEncode (qs : Qubit[]) : Unit is Adj + Ctl {
        CNOT(qs[0], qs[1]);
        CNOT(qs[0], qs[2]);
    }
}
```
