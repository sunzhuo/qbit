# 相位翻转代码：检测 Z 错误

**输入**：处于状态 $\ket{\psi_L} = \alpha \ket{+++} + \beta \ket{---}$ 的三个量子位
或者处于状态 $(Z \otimes I \otimes I)\ket{\psi_L}$、$(I \otimes Z \otimes I)\ket{\psi_L}$ 或 $(I \otimes I \otimes Z)\ket{\psi_L}$ 之一（即，处于相位翻转码的有效码字中或在量子位之一上发生 $Z$ 错误的码字中）。

**目标**：确定是否发生了 $Z$ 错误，如果发生，则在哪个量子位上发生。 
返回值应该是发生错误的量子位的索引，如果没有发生错误，则返回 $-1$。
应用操作后量子位的状态不应改变。

## 解决方案注释

要识别发生错误的量子位，您可以使用与位翻转代码的错误检测相同的逻辑。您需要对任意两对不同的量子位进行两次奇偶校验测量（这次是在 $XX$ 基础上）并分析它们的结果。
下面的代码使用量子位对 $0, 1$ 和 $1, 2$ 上的 $XX$ 奇偶校验测量来实现此逻辑。

来源：Solution.qs

```qsharp
namespace Kata {
    operation PhaseflipDetectError (qs : Qubit[]) : Int {
        let m1 = Measure([PauliX, PauliX], qs[0 .. 1]);
        let m2 = Measure([PauliX, PauliX], qs[1 .. 2]);
        
        if m1 == One and m2 == Zero {
            return 0;
        } elif m1 == One and m2 == One {
            return 1;
        } elif m1 == Zero and m2 == One {
            return 2;
        } else {
            return -1;
        }
    }
}
```
