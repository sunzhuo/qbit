# 短代码：对代码字进行编码

**输入**：处于状态 $\ket{\psi} \otimes \ket{0 \ldots 0}$ 的九个量子位，其中 $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$ 是第一个量子位的状态，即 `qs[0]`。

**目标**：在这些量子位上准备一个状态 $\alpha \ket{0_L} + \beta \ket{1_L}$，其中
$$\ket{0_L} = \frac1{2\sqrt2} (\ket{000} + \ket{111}) \otimes (\ket{000} + \ket{111}) \otimes (\ket{000} + \ket{111})$$
和
$$\ket{1_L} = \frac1{2\sqrt2} (\ket{000} - \ket{111}) \otimes (\ket{000} - \ket{111}) \otimes (\ket{000} - \ket{111})$$

## 解决方案注释

您可以分两步实现此编码。

首先，您对索引为 $0, 3, 6$ 的量子位使用相位翻转编码，将 $\alpha \ket{000} + \beta \ket{100}$ 转换为 $\alpha \ket{+++} + \beta \ket{---}$。
此后，系统的状态为

$$\alpha \ket{+00} \otimes \ket{+00} \otimes \ket{+00} + \beta \ket{-00} \otimes \ket{-00} \otimes \ket{-00}$$

然后，您对量子位 $0 \ldots 2, 3 \ldots 5, 6 \ldots 8$ 的每个三元组使用位翻转编码，将每个 $\ket{+00}$ 转换为 $\frac1{\sqrt2} (\ket{000} + \ket{111})$，将每个 $\ket{-00}$ 转换为 $\frac1{\sqrt2} (\ket{000} - \ket{111})$。此后，九量子位系统将完全处于您正在寻找的状态。

来源：Solution.qs

```qsharp
namespace Kata {
    operation ShorEncode (qs : Qubit[]) : Unit is Adj + Ctl {
        BitflipEncode(qs[0 .. 3 .. 8]);
        ApplyToEachCA(H, qs[0 .. 3 .. 8]);
        for i in 0 .. 2 {
            BitflipEncode(qs[3 * i .. 3 * i + 2]);
        }
    }

    operation BitflipEncode (qs : Qubit[]) : Unit is Adj + Ctl {
        CNOT(qs[0], qs[1]);
        CNOT(qs[0], qs[2]);
    }
}
```
