# 以XX为基准进行测量

**输入**：存储在数组中的两个量子位，保证处于状态 $\ket{++}$ 和 $\ket{--}$ 的叠加或状态 $\ket{+-}$ 和 $\ket{-+}$ 的叠加。

**输出**：如果量子位处于第一次叠加，则为 0；如果处于第二次叠加，则为 1。  
操作结束时量子位的状态应与开始状态相同。

## 解决方案注释

这次，您需要对两个量子位进行 $X \otimes X$ 测量。

状态 $\alpha \ket{++} + \beta \ket{--}$ 是具有特征值 $+1$ 的 $X \otimes X$ 算子的本征状态，状态 $\alpha \ket{+-} + \beta \ket{-+}$ 是具有特征值 $-1$ 的本征状态。
因此，您可以使用此联合测量来识别给定的叠加状态，同时保留初始叠加状态。

来源：Solution.qs

```qsharp
namespace Kata {
    operation XXMeasurement(qs : Qubit[]) : Int {
        return Measure([PauliX, PauliX], qs) == Zero ? 0 | 1;
    }
}
```
