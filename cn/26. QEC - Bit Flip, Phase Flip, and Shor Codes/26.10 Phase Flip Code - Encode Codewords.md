# 相位翻转代码：编码码字

**输入**：处于状态 $\ket{\psi} \otimes \ket{00}$ 的三个量子位，其中 $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$ 是第一个量子位的状态，即 `qs[0]`。

**目标**：在这些量子位上准备状态 $\alpha \ket{+++} + \beta \ket{---}$。

## 解决方案注释

您可以分两步实现此编码：

1. 使用位翻转编码将$\alpha \ket{000} + \beta \ket{100}$转换为$\alpha \ket{000} + \beta \ket{111}$。
2. 对每个量子位应用 Hadamard 门，将每个 $\ket{0}$ 状态转换为 $\ket{+}$，并将每个 $\ket{1}$ 状态转换为 $\ket{-}$。

来源：Solution.qs

```qsharp
namespace Kata {
    operation PhaseflipEncode (qs : Qubit[]) : Unit is Adj + Ctl {
        CNOT(qs[0], qs[1]);
        CNOT(qs[0], qs[2]);
        ApplyToEachCA(H, qs);
    }
}
```
