# 以 ZZ 为基础的测量

**输入**：存储在数组中的两个量子位，保证处于状态 $\ket{00}$ 和 $\ket{11}$ 的叠加或状态 $\ket{01}$ 和 $\ket{10}$ 的叠加。

**输出**：如果量子位处于第一次叠加，则为 0；如果处于第二次叠加，则为 1。  
操作结束时量子位的状态应与开始状态相同。

## 解决方案注释

您需要测量状态的“奇偶性”，而不是将其一直折叠到基础状态。这意味着您需要对两个量子位进行 $Z \otimes Z$ 测量。

使用 $Z \otimes Z$ 运算符的联合测量可以被认为是将测量状态投影到 $Z \otimes Z$ 的两个特征空间之一，其中 $+1$ 和 $-1$ 作为相应的特征值。如果测量状态投影到特征值为 $+1$ 的空间，则测量返回 `Zero`；如果投影到特征值为 $-1$ 的空间，测量结果返回 `One`。

您可以看到状态 $\alpha \ket{00} + \beta \ket{11}$ 是 $Z \otimes Z$ 运算符的特征值 $+1$ 的特征值，状态 $\alpha \ket{01} + \beta \ket{10}$ 是特征值 $-1$ 的特征值。
因此，您可以使用此联合测量来识别给定的叠加状态，同时保留初始叠加状态。

来源：Solution.qs

```qsharp
namespace Kata {
    operation ZZMeasurement(qs : Qubit[]) : Int {
        return Measure([PauliZ, PauliZ], qs) == Zero ? 0 | 1;
    }
}
```
