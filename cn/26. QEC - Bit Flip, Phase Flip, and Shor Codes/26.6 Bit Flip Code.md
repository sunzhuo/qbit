# 位翻转代码

我们可以将经典重复码的思想重新用于量子纠错码吗？

最简单的方法是尝试将量子态 $\ket{\psi}$ 编码为其自身的多个副本：
$\ket{\psi} \rightarrow \ket{\psi} \otimes \ket{\psi} \otimes \ket{\psi}$。
不幸的是，不可克隆定理以及测量后无法准确重建状态使得这种方法变得不可能。

但是，您可以采取稍微不同的方法：使用酉变换对重复码中的*基础状态* $\ket{0}$ 和 $\ket{1}$ 进行编码，并根据其线性推导出此变换对叠加状态的影响：

$$\ket{0} \rightarrow \ket{000}, \ket{1} \rightarrow \ket{111}$$

$$\alpha \ket{0} + \beta \ket{1} \rightarrow \alpha \ket{000} + \beta \ket{111}$$

这种编码称为**位翻转码**，状态$\ket{000}$、$\ket{111}$及其线性组合在此代码中称为**码字**。位翻转代码使我们能够检测并纠正去极化通道中的量子位上可能发生的一些错误，尽管不是全部错误。

让我们看看如果其中一个量子位发生 $X$ 错误会发生什么，以及如何使用两个奇偶校验测量（以 $ZZ$ 为基础的测量）来检测它。

您可以看到，这两个奇偶校验测量为我们提供了不同的结果对，具体取决于 $X$ 错误是否发生以及在哪个量子位上发生。这意味着您可以使用它们来检测错误，然后通过将 $X$ 门应用于受错误影响的量子位来纠正错误。

但是，如果这些量子位中的任何一个发生 $Z$ 错误，您将无法检测到它：它将把状态 $\alpha \ket{000} + \beta \ket{111}$ 转换为状态 $\alpha \ket{000} - \beta \ket{111}$，这是此代码中的有效代码字 - 它是量子状态 $\alpha \ket{0} - \beta \ket{1}$ 的编码。您需要想出一种不同的方法来检测 $Z$ 错误。
