# 短代码：检测 X、Y 和 Z 错误

**输入**：九个量子位，要么处于状态 $\ket{\psi_L} = \alpha \ket{0_L} + \beta \ket{0_L}$（Shor 码的有效码字），要么处于其中一个量子位上发生 $X$、$Y$ 或 $Z$ 错误的码字状态。

**目标**：确定是否发生错误，如果发生错误，则确定错误类型和错误发生在哪个量子位上。 
返回值是两个元素的元组，描述检测到的错误如下：

- 返回的第一个元素是 `Int` - 发生错误的量子位的索引，如果没有发生错误，则为 $-1$。
- 返回的第二个元素是 `Pauli`，指示错误类型（`PauliX`、`PauliY` 或 `PauliZ`）。
如果没有发生错误，则返回的第二个元素可以是任何值，但不会进行验证。
- 如果出现单个 $Z$ 错误，则无法唯一识别发生该错误的量子位。 
在这种情况下，返回值应该是发生错误的量子位三元组的索引（对于量子位 $0 \ldots 2$ 为 $0$，对于量子位 $3 \ldots 5$ 为 $1$，对于量子位 $6 \ldots 8$ 为 $2$。

## 解决方案注释

对于此代码，您将分别检测 $X$ 和 $Z$ 错误，类似于位翻转代码和相位翻转代码的操作方式，然后将结果合并到最终答案中。 $Y$ 错误将由同时发生的 $X$ 和 $Z$ 错误发出信号，因为问题表明最多发生一个错误。

1. 要检测 $X$ 错误，请回想一下，在 Shor 码中，每个量子位三元组 $0 \ldots 2, 3 \ldots 5, 6 \ldots 8$ 都是位翻转码中 $\ket{+}$ 状态的编码。这意味着您可以通过分别将位翻转错误检测逻辑应用于每个量子位三元组来检测 $X$ 错误。

2. 要检测 $Z$ 错误，回想一下，在 $X$ 基础上进行 6 量子位联合测量可以检测测量中前三个量子位和后三个量子位的相对相位的奇偶性，即区分

    $$\frac12 \left( (\ket{000} + \ket{111}) \otimes (\ket{000} + \ket{111}) + (\ket{000} - \ket{111}) \otimes (\ket{000} - \ket{111}) \right)$$

从

    $$\frac12 \left( (\ket{000} + \ket{111}) \otimes (\ket{000} - \ket{111}) + (\ket{000} - \ket{111}) \otimes (\ket{000} + \ket{111}) \right)$$

这意味着您可以通过对量子位三元组 $0 \ldots 2$ 和 $3 \ldots 5$ 以及 $3 \ldots 5$ 和 $6 \ldots 8$ 进行两次 6 量子位联合测量来检测 $Z$ 错误，并以与位翻转和相位翻转代码相同的方式解释它们的结果。

3. 要合并结果，请检查出现了哪些结果：

- 如果未检测到 $X$ 和 $Z$ 错误，则返回“无错误”；
- 如果两者都被检测到，则返回 $Y$ 错误并使用检测到 $X$ 错误的量子位的索引（请记住，您只能将 $Z$ 错误跟踪到它发生的三元组，而不是确切的量子位）；
- 如果仅检测到一个错误，则返回该错误。

```qsharp
namespace Kata {
    operation ShorDetectError (qs : Qubit[]) : (Int, Pauli) {
        // Detect X error first
        mutable x_ind = -1;
        for i in 0 .. 2 {
            let err_ind = BitflipDetectError(qs[3 * i .. 3 * i + 2]);
            if err_ind > -1 {
                set x_ind = 3 * i + err_ind;
            }
        }

        // Detect Z error 
        mutable z_ind = -1;
        let m1 = Measure([PauliX, size = 6], qs[0 .. 5]);
        let m2 = Measure([PauliX, size = 6], qs[3 .. 8]);

        if m1 == Zero and m2 == Zero {
            set z_ind = -1;
        } elif m1 == One and m2 == Zero {
            set z_ind = 0;
        } elif m1 == One and m2 == One {
            set z_ind = 1;
        } elif m1 == Zero and m2 == One {
            set z_ind = 2;
        }

        // Combine both errors into return value
        if x_ind == -1 and z_ind == -1 {
            return (-1, PauliI);
        }
        if x_ind > -1 and z_ind > -1 {
            return (x_ind, PauliY);
        }
        if x_ind > -1 {
            return (x_ind, PauliX);
        }
        return (z_ind, PauliZ);
    }

    // You might find this helper operation from an earlier task useful.
    operation BitflipDetectError (qs : Qubit[]) : Int {
        let m1 = Measure([PauliZ, PauliZ], qs[0 .. 1]);
        let m2 = Measure([PauliZ, PauliZ], qs[1 .. 2]);
        
        if m1 == One and m2 == Zero {
            return 0;
        } elif m1 == One and m2 == One {
            return 1;
        } elif m1 == Zero and m2 == One {
            return 2;
        } else {
            return -1;
        }
    }
}
```
