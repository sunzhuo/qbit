# 位翻转代码：检测 X 错误

**输入**：处于状态 $\ket{\psi_L} = \alpha \ket{000} + \beta \ket{111}$ 的三个量子位
或者处于 $(X \otimes I \otimes I)\ket{\psi_L}$、$(I \otimes X \otimes I)\ket{\psi_L}$ 或 $(I \otimes I \otimes X)\ket{\psi_L}$ 状态之一（即，位翻转代码的有效代码字或在其中一个量子位上发生 $X$ 错误的代码字）。

**目标**：确定是否发生 $X$ 错误，如果发生，则在哪个量子位上发生。 
返回值应该是发生错误的量子位的索引，如果没有发生错误，则返回 $-1$。
应用操作后量子位的状态不应改变。

## 解决方案注释

要识别发生错误的量子位，您需要在 $ZZ$ 基础上对任意两对不同的量子位进行两次奇偶校验测量并分析其结果：

- 如果两个奇偶校验测量结果都为 $0$，则没有发生错误。
- 如果两个奇偶校验测量结果均产生 $1$，则错误发生在测量的量子位对之间共享的量子位上。
- 如果奇偶校验测量之一产生 $1$，另一个产生 $0$，则错误发生在仅属于产生 $1$ 的测量所涉及的一对量子位的一部分的量子位上，而不是另一对量子位上。

下面的代码使用量子位对 $0, 1$ 和 $1, 2$ 上的 $ZZ$ 奇偶校验测量来实现此逻辑。

来源：Solution.qs

```qsharp
namespace Kata {
    operation BitflipDetectError (qs : Qubit[]) : Int {
        let m1 = Measure([PauliZ, PauliZ], qs[0 .. 1]);
        let m2 = Measure([PauliZ, PauliZ], qs[1 .. 2]);
        
        if m1 == One and m2 == Zero {
            return 0;
        } elif m1 == One and m2 == One {
            return 1;
        } elif m1 == Zero and m2 == One {
            return 2;
        } else {
            return -1;
        }
    }
}
```
