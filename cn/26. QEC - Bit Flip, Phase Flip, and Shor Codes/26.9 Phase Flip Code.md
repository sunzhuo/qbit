# 相位翻转代码

什么样的代码可以检测并纠正 $Z$ 错误？您检测到 $X$ 错误，因为在 $\{\ket{0}, \ket{1}\}$ 基础上该错误更改了基础状态。同样，您可以使用 $\{\ket{+}, \ket{-}\}$ 基础检测 $Z$ 错误，其中 $Z$ 门将 $\ket{+}$ 转换为 $\ket{-}$，反之亦然，充当基础更改操作。

基于这个想法，您可以构造使用以下编码的**相位翻转代码**：

$$\ket{0} \rightarrow \ket{+++}, \ket{1} \rightarrow \ket{---}$$

$$\alpha \ket{0} + \beta \ket{1} \rightarrow \alpha \ket{+++} + \beta \ket{---}$$

让我们看看如果其中一个量子位发生 $Z$ 错误会发生什么，以及如何使用两个奇偶校验测量来检测它。
这次，您在 $X$ 基础上进行奇偶校验测量，以区分 $\ket{++}$ 和 $\ket{--}$（奇偶校验 $0$）与 $\ket{+-}$ 和 $\ket{-+}$（奇偶校验 $1$）的情况。

您可以看到，这两个奇偶校验测量给出了不同的结果对，具体取决于 $Z$ 错误是否发生以及在哪个量子位上发生。这意味着您可以使用它们来检测错误，然后通过将 $Z$ 门应用于受错误影响的量子位来纠正错误。

但是，如果这些量子位中的任何一个发生 $X$ 错误，您将无法检测到它：它将把状态 $\alpha \ket{+++} + \beta \ket{---}$ 转换为状态 $\alpha \ket{+++} - \beta \ket{---}$，这是此代码中的有效代码字 - 它是量子状态 $\alpha \ket{0} - \beta \ket{1}$ 的编码。您需要想出一种不同的方法来检测同一编码中的 **$X$ 和 $Z$ 错误。
