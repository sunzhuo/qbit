# 区分四种基本状态

**输入：** 两个量子位（存储在长度为 2 的数组中），保证处于四种基本状态之一（$\ket{00}$、$\ket{01}$、$\ket{10}$ 或 $\ket{11}$）。

**输出：**

* 0 如果量子位处于 $\ket{00}$ 状态，
* 1 if they were in the $\ket{01}$ state, 
* 2 if they were in the $\ket{10}$ state, 
* 3 if they were in the $\ket{11}$ state.

在此任务和后续任务中，任务描述中的量子位状态顺序与数组中的量子位顺序匹配（即，$\ket{10}$ 状态对应于状态 $\ket{1}$ 中的 `qs[0]` 和状态 $\ket{0}$ 中的 `qs[1]`。

操作结束时量子位的状态并不重要。

## 解决方案注释

首先，测量输入数组中的每个量子位，将测量结果转换为整数
并将它们存储在变量 `m1` 和 `m2` 中。您可以像这样解码这些结果：

- `m1` 为 $\ket{0}$ 且 `m2` 为 $\ket{0}$：返回值为 $0\cdot2+0 = 0$
- `m1` 为 $\ket{0}$ 且 `m2` 为 $\ket{1}$：返回值为 $0\cdot2+1 = 1$
- `m1` 为 $\ket{1}$ 且 `m2` 为 $\ket{0}$：返回值为 $1\cdot2+0 = 2$
- `m1` 为 $\ket{1}$ 且 `m2` 为 $\ket{1}$：返回值为 $1\cdot2+1 = 3$

换句话说，您将测量结果视为大端计数法返回值的二进制计数法。

来源：Solution.qs

```qsharp
namespace Kata {
    operation BasisStateMeasurement(qs : Qubit[]) : Int {
        // Measurement on the first qubit gives the most significant bit of the answer, 
        // on the second - the least significant bit.
        let m1 = M(qs[0]) == Zero ? 0 | 1;
        let m2 = M(qs[1]) == Zero ? 0 | 1;
        return m1 * 2 + m2;
    }
}
```

您可以将此代码推广为使用多个库操作和函数从任意长度的量子位数组中读出大端表示法的整数：

- `MeasureEachZ` 在计算基础上测量数组中的每个量子位，并返回 `Result` 数据类型的数组。
- `Reversed` 反转给定的数组。
- `ResultArrayAsInt` 将 `Result` 给出的位数组转换为整数，假设采用小端表示法（这就是为什么在转换数组之前必须反转数组）。

来源：SolutionAlt.qs

```qsharp
namespace Kata {
    import Std.Arrays.*;
    import Std.Convert.*;

    operation BasisStateMeasurement(qs : Qubit[]) : Int {
        return ResultArrayAsInt(Reversed(MeasureEachZ(qs)));
    }
}
```
