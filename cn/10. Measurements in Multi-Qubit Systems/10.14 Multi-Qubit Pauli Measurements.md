# 多量子位泡利测量

联合测量是计算基础上测量的概括。
泡利测量也可以推广到更多数量的量子位。多量子位泡利测量对应于运算符 $M_1 \otimes \dotsc \otimes M_n$，每个 $M_i$ 来自门集 $\{X,Y,Z,I\}$。如果至少其中一个运算符不是单位矩阵，则测量可能会产生两种结果：对应于特征值 $+1$ 的 `Zero` 的 `Result` 和对应于特征值 $-1$ 的 `One` 的 `Result`。相应的投影算子是到相应特征空间的投影。

例如，对应于 $X\otimes Z$ 运算符的泡利/联合测量可以表征如下：

## 🔎 分析

### 不同基础上的奇偶校验测量

考虑一个处于状态 $\alpha \ket{00} + \beta \ket{01} + \beta \ket{10} + \alpha \ket{11}$ 的系统。

如果进行 $XX$ 泡利测量中的测量，可能的结果及其相关概率是什么？

<details>
<summary><b>解决方案</b></summary>
识别联合测量的结果及其概率的第一步是识别与泡利算子的特征值 $\pm1$ 相对应的特征向量。请注意，自 $X\ket{\pm}= \pm\ket{\pm}$ 以来，您已经

$$XX \ket{++} = \ket{++}$$
$$XX \ket{--} = \ket{--}$$
$$XX \ket{+-} = -\ket{+-}$$
$$XX \ket{-+} = -\ket{-+}$$

因此，$XX$ 运算符测量 Hadamard 或 $\ket{\pm}$ 基中的奇偶校验。也就是说，它区分具有偶数个$+$的基本状态和具有奇数个$+$的基本状态。

与`Zero`的结果对应的投影仪由$P_{+1} = \ket{++}\bra{++} + \ket{--}\bra{--}$给出，而与`One`的结果对应的投影仪由$P_{-1} = \ket{+-}\bra{+-} + \ket{-+}\bra{-+}$给出。然后，请注意，$P_{+1}$ 消除了具有奇奇偶校验的状态，同时使具有偶奇偶校验的状态不受影响。也就是说，对于任何常数值
$$P_{+1} ( \gamma \ket{++} + \delta \ket{--} ) = ( \gamma \ket{++} + \delta \ket{--} )$$
$$P_{+1} ( \mu \ket{-+} + \nu \ket{+-} ) = 0$$

类似地，$P_{-1}$ 消除具有偶奇偶校验的状态，同时使具有奇奇偶校验的状态不受影响。

现在您用哈达玛基表达给定的状态。请注意，可以使用以下关系从计算基础转换为 Hadamard 基础：
$$\ket{0} = \frac{1}{\sqrt{2}} \left( \ket{+} + \ket{-} \right)$$
$$\ket{1} = \frac{1}{\sqrt{2}} \left( \ket{+} - \ket{-} \right)$$

使用这些，您可以获得
$$ \alpha \ket{00} + \beta \ket{01} + \beta \ket{10} + \alpha \ket{11} = (\alpha + \beta) \ket{++} + (\alpha - \beta) \ket{--}$$
因此，该状态在哈达玛基础上具有偶宇称。由此可见，$XX$ 泡利测量将以概率 1 产生结果 `Zero`，测量后状态保持不变。
</details>
