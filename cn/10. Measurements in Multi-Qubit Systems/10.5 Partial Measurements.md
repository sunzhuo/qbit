# 部分测量

对于具有 $n>1$ 量子位的系统，可以逐个测量 $m<n$ 量子位。测量结果的数量为 $2^m$ 而不是 $2^n$。可以使用测量的投影形式来找到每个结果的概率和量子位的测量后状态。

首先，回顾一下单量子位系统测量中引入的投影算子的概念。测量结果由正交投影算子建模 - 满足以下条件的矩阵
$$P^2 = P^\dagger = P$$
考虑处于状态 $\ket{\psi}$ 的 $n$-量子位系统，其中第一个 $m<n$ 量子位是在正交基 $\{ \ket{b_0} , \ket{b_1}, \dotsc, \ket{b_{2^m-1}}\}$ 上测量的，对应于正在测量的 $m$ 量子位。然后，将与每个 $\ket{b_i}$ 状态相对应的 $2^m$ 投影仪定义为

$$P_i = \ket{b_i} \bra{ b_i} \otimes \mathbb{1}_{n-m} $$

其中 $\mathbb{1}_{n-m}$ 是剩余 $(n-m)$ 量子位上的恒等运算符。

> 符号$\otimes$表示两个矩阵的张量积或克罗内克积。它与通常的矩阵乘法不同。
> 在当前上下文中，$\ket{b_i} \bra{ b_i} \otimes \mathbb{1}_{n-m}$ 只是意味着
>
> - 运算符 $\ket{b_i} \bra{ b_i}$ 仅作用于正在测量的 $m$ 量子位。
> - $P_i$ 对剩余量子位的影响是 $\mathbb{1}_{n-m} $，即恒等运算符。

与单量子位系统测量的情况类似，部分测量概率和结果的规则可以总结如下：

- 测量完成后，随机选择其中一台投影仪。选择投影仪 $P_i$ 的概率是 $\big|P_i\ket{\psi}\big|^2$。
- 如果选择投影仪 $P_i$，则测量结果为 $b_i$，测量后系统的状态由下式给出

$$
\frac{P_i \ket{\psi}}{\big|P_i \ket{\psi}\big|}.
$$

例如，考虑处于状态 $\ket \psi = \frac{1}{\sqrt{2}}\ket{01} - \frac{1}{\sqrt 2}\ket{10}$ 的两个量子位系统。考虑计算基础（即 $\{\ket 0 , \ket 1 \}$ 基础）中第一个量子位的测量。然后，您有两个代表此测量的投影仪：
$$P_0 = \ket 0\bra 0 \otimes \mathbb{1}$$
$$P_1 = \ket 1 \bra 1 \otimes \mathbb{1}$$

$P_0$ 对 $\ket \psi$ 的操作是

$$P_0 \ket \psi = \left(\ket 0\bra 0 \otimes \mathbb{1}\right) \frac{1}{\sqrt 2}\big(\ket{01} - \ket{10}\big) =$$

$$= \frac{1}{\sqrt 2} \big( \ket 0 \braket{0|0} \otimes \mathbb{1} \ket{1} - \ket 0 \braket{0|1} \otimes \mathbb{1} \ket 0 \big) = \frac{1}{\sqrt 2} \ket{01}$$

同样，您获得
$$P_1 \ket\psi = -\frac{1}{\sqrt 2} \ket{10}$$

显然，在这种情况下您有 $\big|P_0 \ket \psi\big| = \big|P_1 \ket \psi\big| = \frac{1}{2}$。因此，测量$0$和$1$的概率均为$0.5$，系统的测量后状态分别为$\ket{01}$和$\ket{10}$。

> 与单量子位系统测量的情况类似，上述形式的适用性需要对多量子位系统的状态 $\ket \psi$ 进行归一化。这是为了确保单个结果的所有概率加起来为 1 所必需的。

## 🔎 分析

### Hardy 态的部分测量概率

考虑状态为 $\ket \psi = \frac{1}{\sqrt{12}} \big(3\ket{00} + \ket{01} + \ket{10} + \ket{11}\big)$ 的 2 量子位系统。

如果在计算基础上仅测量第一个量子位，则结果的概率以及系统的测量后状态是多少？

<details>
<summary><b>解决方案</b></summary>
第一个量子位上 $0$ 的测量结果对应于投影算子 $P_0 = \ket{0}\bra{ 0} \otimes \mathbb{1}$。将其应用到状态 $\ket \psi$ 可以得到
$$\big|P_0 \ket{\psi}\big|^2 = \big|\frac{1}{\sqrt{12}} \left(3\ket {00} + \ket{01}\right) \big|^2 = \frac{5}{6}$$
和
$$\frac{P_0 \ket{\psi}}{\big|P_0 \ket{\psi}\big|} = \frac{1}{\sqrt{10}} \left( 3\ket{00} + \ket{01}\right)$$

类似地，$P_1 = \ket{1} \bra{ 1 } \otimes \mathbb{1}$是与第一个量子位上的$1$测量结果相对应的投影仪。将 $P_1$ 应用于 $\ket{\psi}$ 得到 $\big|P_1 \ket{\psi}\big|^2 = \frac{1}{6}$ 和

$$\frac{P_1 \ket{\psi}}{\big|P_1 \ket{\psi}\big|} = \frac{1}{\sqrt{2}} \left(\ket{10} + \ket{11}\right)$$
