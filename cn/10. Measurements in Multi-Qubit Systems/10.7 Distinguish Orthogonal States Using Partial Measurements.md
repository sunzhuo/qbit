# 使用部分测量来区分正交态

**输入**：保证处于 $\ket{++-}$ 状态或 $\ket{---}$ 状态的 3 量子位系统。

**输出**： 
- 如果量子位处于 $\ket{++-}$ 状态，则为 0，
- 如果量子位处于 $\ket{---}$ 状态，则为 1。

操作结束时量子位的状态应与初始状态相同。

## 解决方案注释

由于第一个量子位的状态在这些状态中不同（分别为 $\ket +$ 和 $\ket -$），因此仅测量泡利 X 基中的第一个量子位就足以区分这两个状态。此外，该测量不会改变系统的状态，因为无论输入状态如何，投影后状态都与原始状态相同。

来源：Solution.qs

```qsharp
namespace Kata {
    operation IsPlusPlusMinus(qs : Qubit[]) : Int {
        return Measure([PauliX], [qs[0]]) == Zero ? 0 | 1;
    }
}
```
