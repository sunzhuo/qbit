# 使用部分测量进行状态选择

**输入**： 
1. 状态下的2量子位系统
$$
\ket{\psi} = \frac{1}{\sqrt{2}} \ket{0} \otimes ( a \ket{0} + b\ket{1}) + \frac{1}{\sqrt{2}} \ket{1} \otimes (b\ket{0} + a \ket{1}),
$$
其中满足 $|a|^2 + |b|^2 = 1$ 的常量 $a$ 和 $b$ 未知。
2. 整数 $ind$，可以是 $0$ 或 $1$。

**目标**： 
- 如果 $ind$ 等于 0，则将第二个量子位的状态转换为 $a\ket{0} + b\ket{1}$
- 如果 $ind$ 等于 1，则将第二个量子位的状态转换为 $b\ket{0} + a\ket{1}$。

最后第一个量子位的状态并不重要（它必须不与第二个量子位纠缠）。

## 解决方案注释

请注意，如果您测量计算基础中的第一个量子位，则 $0$ 的结果会将第二个量子位折叠到状态 $a\ket 0 + b \ket 1$，而 $1$ 的结果会将第二个量子位折叠到状态 $b\ket 0 + a \ket 1$。

因此，如果 $ind=0$ 且您测量 $0$，或者如果 $ind=1$ 且我们测量 $1$，则测量后第二个量子位将处于所需状态。另一方面，如果 $ind=1$ 且您测量 $0$，或者如果 $ind=0$ 且您测量 $1$，则测量后第二个量子位的状态不是您想要的，但您可以使用泡利 X 门对其进行调整。

来源：Solution.qs

```qsharp
namespace Kata {
    operation StateSelection(qs : Qubit[], ind : Int) : Unit {
        // It is convenient to convert measurement outcome to an integer
        let outcome = M(qs[0]) == Zero ? 0 | 1;
        if outcome != ind {
            X(qs[1]);
        }
    }
}
```
