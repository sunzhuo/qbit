# 全面测量：多量子位基础的测量

考虑一个由 $n\geq1$ 量子位组成的系统。这种系统的波函​​数属于维度为$2^n$的向量空间。因此，向量空间由正交基跨越，例如由向量$\ket{0\dotsc0}, \dotsc, \ket{1\dotsc 1}$组成的计算基。出于一般性考虑，考虑任意正交基，用 $\{ \ket{b_0}, \ket{b_1}, \dotsc, \ket{b_{2^n-1}} \}$ 表示。

然后，多量子位系统的状态$\ket{\psi}$可以表示为$2^n$基向量$\ket{b_i}$的线性组合。也就是说，存在复数 $c_0,c_1,\dotsc, c_{2^n-1}$ 使得

$$
\ket{\psi} = \sum_{i=0}^{2^n-1} c_i\ket{b_i} \equiv \begin{pmatrix} c_0 \\ c_1 \\ \vdots \\ c_{2^n-1} \end{pmatrix}
$$

按照通常的约定，我们选择要归一化的波函数，即$|c_0|^2 + \dotsc + |c_{2^n-1}|^2 =1$。那么，基于$\{ \ket{b_0}, \ket{b_1}, \dotsc, \ket{b_{2^n-1}} \}$的量子测量满足以下规则：

- 测量结果 $b_i$ 出现的概率为 $|c_i|^2$。
- 每当测量结果为 $b_i$ 时，波函数就会崩溃到状态 $\ket{b_i}$。即，系统的测量后状态等于$\ket{b_i}$。

> 与单量子位系统中的测量类似，需要对原始波函数进行归一化假设，以确保所有结果概率的总和为 1。

## 🔎 分析

### 多量子位测量结果概率 I

您将获得一个处于以下状态的双量子位系统：
$$\ket \psi =  \frac{2}{3}\ket {00} + \frac{1}{3} \ket {01} + \frac{2}{3}\ket {11}$$

如果在计算基础上同时测量所有量子位，结果概率是多少？

<details>
<summary><b>解决方案</b></summary>

自 $\left(\frac{2}{3}\right)^2 + \left(\frac{1}{3}\right)^2 + \left(\frac{2}{3}\right)^2 = 1$ 起，波函数 $\ket{\psi}$ 已归一化。因此，测量每个计算基础状态的概率只是相应系数绝对值的平方。也就是说，测量 $00$、$01$ 和 $11$ 的概率分别为 $\frac{4}{9}$、$\frac{1}{9}$ 和 $\frac{4}{9}$，测量不属于叠加部分的基础状态 $10$ 的概率为$0$：

### 多量子位测量结果概率 II

您将获得一个与上一个练习处于相同状态的两个量子位系统：
$$\ket \psi = \frac{2}{3}\ket {00} + \frac{1}{3} \ket {01} + \frac{2}{3}\ket {11}$$

如果所有量子位在泡利 X 基（即 $\{ \ket{++}, \ket{+-}, \ket{-+}, \ket{--}\}$ 基）下同时测量，结果概率是多少？

<details>
<summary><b>解决方案</b></summary>

使用表达式 $\ket{0} = \frac{1}{\sqrt{2}} \big( \ket{+} + \ket{-} \big)$ 和 $\ket{1} = \frac{1}{\sqrt{2}} \big( \ket{+} - \ket{-} \big)$，首先在 Pauli X 基中表达 $\ket{\psi}$。这给你
$$\ket \psi =  \frac{2}{3}\ket {00} + \frac{1}{3} \ket {01} + \frac{2}{3}\ket {11} =$$

$$= \frac{2}{3} \big[ \frac{1}{\sqrt{2}}\big(\ket{+} + \ket{-}\big) \otimes \frac{1}{\sqrt{2}} \big(\ket{+} + \ket{-}\big) \big] + \frac{1}{3} \big[ \frac{1}{\sqrt{2}}\big(\ket{+} + \ket{-}\big) \otimes \frac{1}{\sqrt{2}} \big(\ket{+} - \ket{-}\big) \big] + \frac{2}{3} \big[ \frac{1}{\sqrt{2}}\big(\ket{+} - \ket{-}\big) \otimes \frac{1}{\sqrt{2}} \big(\ket{+} - \ket{-}\big) \big] =$$

$$= \frac{1}{3} \big[ \big(\ket{+} + \ket{-}\big) \otimes \big(\ket{+} + \ket{-}\big) \big] + \frac{1}{6} \big[ \big(\ket{+} + \ket{-}\big) \otimes \big(\ket{+} - \ket{-}\big) \big] + \frac{1}{3} \big[ \big(\ket{+} - \ket{-}\big) \otimes \big(\ket{+} - \ket{-}\big) \big] =$$

$$= \frac{1}{3} \big[ \ket{++} + \ket{+-} + \ket{-+} + \ket{--} \big] + \frac{1}{6} \big[ \ket{++} - \ket{+-} + \ket{-+} - \ket{--} \big] + \frac{1}{3} \big[ \ket{++} - \ket{+-} - \ket{-+} + \ket{--} \big] =$$

$$= (\frac{1}{3} + \frac{1}{6} + \frac{1}{3})\ket{++} + (\frac{1}{3} - \frac{1}{6} - \frac{1}{3})\ket{+-} + (\frac{1}{3} + \frac{1}{6} - \frac{1}{3})\ket{-+} + (\frac{1}{3} - \frac{1}{6} + \frac{1}{3})\ket{--} =$$

$$= \frac{5}{6}\ket{++} - \frac{1}{6}\ket{+-} + \frac{1}{6}\ket{-+} + \frac{1}{2}\ket{--}$$

此后，测量四个基向量中每一个的概率由其叠加时幅度绝对值的平方给出：

1. 首先准备状态 $\ket \psi$。
   为此，您可以将 $\ket \psi$ 表示如下：
   $$\frac 2 3 \ket{00} + \big( \frac 1 {\sqrt 5} \ket{0} + \frac 2 {\sqrt 5} \ket{1} \big) \frac {\sqrt 5} 3 \ket{1}$$

namespace Kata {
    import Std.Diagnostics.*;
    import Std.Math.*;

    @EntryPoint()
    operation CalculateProbabilities() : Unit {
        use qs = Qubit[2];

        Ry(2. * ArcCos(2. / 3.), qs[1]);
        Controlled Ry([qs[1]], (2. * ArcCos(1. / Sqrt(5.)), qs[0]));

        Message("The initial state of the system");
        Message("(includes measurement outcome probabilities in computational basis):");
        DumpMachine();

        // Change the basis
        H(qs[0]);
        H(qs[1]);

        Message("Final state of the two-qubit system:");
        Message("(includes measurement outcome probabilities in Pauli X basis):");
        DumpMachine();

        // Return the qubit array into state |00❭.
        ResetAll(qs);
    }
}

## 依次测量系统中的每个量子位

如前面部分所述，理论上可以在正交基础上同时测量 $n$-量子位系统中的所有量子位。量子位的测量后状态恰好是 $2^n$ 可能的基础状态之一。

实际上，这是通过依次测量所有量子位来实现的。例如，如果想要在计算基础上测量双量子位系统，可以通过首先测量计算基础中的第一个量子位以获得$0$或$1$，然后测量计算基础中的第二个量子位来实现。这可能导致四种可能结果之一：$00, 01, 10, 11$。

这可以推广到其他基础的测量，例如 2 量子位 Pauli X 基础 $\ket{++}, \ket{+-}, \ket{-+}, \ket{--}$，以及更多量子位的基础。

> 请注意，所有量子位的顺序测量只能在正交基 $\{ \ket{b_i}\}$ 中完成，这样每个 $\ket{b_i}$ 都是一个 **张量积状态**。也就是说，每个 $\ket{b_i}$ 必须采用 $\ket{v_0} \otimes \ket{v_1} \dotsc \otimes \ket{v_{n-1}}$ 形式，每个 $\ket{v_j}$ 都是单量子位基础状态。
例如，对于两个量子位 Pauli X 基 $\ket{++}, \ket{+-}, \ket{-+}, \ket{--}$，每个基状态是状态 $\ket{+}$ 和 $\ket{-}$ 的张量积，它们形成单量子位基状态。
>
> 在包含非张量积状态的正交基中进行测量（例如贝尔基），实施起来比较困难，并且除了逐个测量所有量子位之外，还需要适当的单位旋转。
> 本型中不讨论此类测量。
>
如果测量仅限于张量积状态，那么同时测量所有量子位与依次测量所有量子位之间的区别对于理想的量子计算机来说并不重要：就结果和测量概率而言，两种方法是相同的。此外，只要测量了所有量子位，测量它们的顺序也是无关紧要的。对于具有不完美量子位的真实量子计算机来说，这些因素可能很重要，但本套中的讨论仅限于理想系统。
