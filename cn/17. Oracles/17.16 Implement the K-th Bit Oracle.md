# 实现第K位预言机

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
  2. 整数 $k$ 使得 $0 \leq k < N$。

**目标：**

如果 $x$ 的第 $k$ 位为 $1$，则翻转输入状态 $\ket{x}$ 的符号。  
*在不使用辅助量子位的情况下实现此预言。*

**示例：**

* 如果查询寄存器处于状态$\ket{010}$和$k=0$，则不执行任何操作。
* 如果查询寄存器处于状态$\ket{010}$和$k=1$，则翻转基本状态的符号。

<br/>
<details>
  <summary><b>在实现此预言之前，请回答以下问题：您是否正在实现标记预言机或阶段预言机？</b></summary>
    这是一个相位预言，因为您要根据函数 $f(x)$ 的值更改输入状态 $\ket{x}$ 的相位。
</details>

## 解决方案注释

由于该预言的效果仅取决于第 $k$ 个量子位的值，因此您可以忽略其余量子位并仅关注 `x[k]`。
如果量子位处于 $\ket{1}$ 状态，则需要翻转其相位，否则保持不变 - 这正是 $Z$ 门的效果。

来源：Solution.qs

```qsharp
namespace Kata {
    operation KthBit_Oracle(x : Qubit[], k : Int) : Unit is Adj + Ctl {
        Z(x[k]);
    }
}
```
