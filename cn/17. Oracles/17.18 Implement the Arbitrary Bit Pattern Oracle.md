# 实现任意位模式 Oracle

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
  2. 处于任意状态 $\ket{y}$ 的量子位（目标量子位）。
  3. 长度为$N$`pattern`的布尔数组，表示基础状态； `true`和`false`元素分别对应于$\ket{1}$和$\ket{0}$。

**目标：**

如果输入寄存器与基本状态匹配，则翻转 $\ket{y}$ 的状态
由 `pattern` 表示。

**示例：**

* 如果查询寄存器处于状态$\ket{010}$和`pattern = [false, true, false]`，则翻转状态$\ket{y}$。
* 如果查询寄存器处于状态$\ket{1001}$和`pattern = [false, true, true, false]`，则不执行任何操作。

## 解决方案注释

该预言机的效果与库操作 `ApplyControlledOnBitString` 的操作完全匹配 
如果控制量子位处于某种状态，则对目标量子位应用酉（在本例中为 $X$）。

来源：Solution.qs

```qsharp
namespace Kata {
    operation ArbitraryBitPattern_Oracle(x : Qubit[], y : Qubit, pattern : Bool[])
    : Unit  is Adj + Ctl {
        ApplyControlledOnBitString(pattern, X, x, y);
    }
}
```
