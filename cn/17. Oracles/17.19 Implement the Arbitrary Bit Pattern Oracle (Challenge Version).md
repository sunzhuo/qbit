# 实现任意位模式预言机（挑战版）

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
  2、长度为$N$`pattern`的布尔数组，表示基础状态； `true`和`false`元素分别对应于$\ket{1}$和$\ket{0}$。

**目标：**
 
如果输入寄存器与基本状态匹配，则翻转输入状态 $\ket{x}$ 的符号
由 `pattern` 表示。  
*在不使用辅助量子位的情况下实现此预言*

**示例：**

* 如果查询寄存器处于状态$\ket{010}$和`pattern = [false, true, false]`，则翻转输入寄存器的符号。
 * 如果查询寄存器处于状态$\ket{1001}$和`pattern = [false, true, true, false]`，则不执行任何操作。

## 解决方案注释

最简单的解决方案是变换给定状态，使与给定模式匹配的基本状态变为 $\ket{1...1}$，
应用受控 $Z$ 门来翻转该基本状态的相位，然后取消计算以确保仅基本状态的相对相位发生变化，而不是基本状态本身发生变化。

您可以使用 `within ... apply` 构造来自动取消计算。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Arrays.*;

    operation ArbitraryBitPattern_Oracle_Challenge(
        x : Qubit[],
        pattern : Bool[]
    ) : Unit is Adj + Ctl {
        within {
            for i in IndexRange(x) {
                if not pattern[i] {
                    X(x[i]);
                }
            }
        } apply {
            Controlled Z(Most(x), Tail(x));
        }
    }
}
```
