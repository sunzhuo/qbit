# 实现一个经典的预言机

**输入：** 
  长度为 3 的位向量表示为 `Bool[]` - 以小端表示法表示的数字的二进制表示形式。

**输出：**
  如果输入数组表示数字 $7$，则返回 `true`，否则返回 `false`。

**示例：**

* 如果输入数组为`[true, true, true]`，则返回`true`。
* 如果输入数组为`[true, true, false]`，则返回`false`。

## 解决方案注释

$7$ 的二进制表示法是 $111_2$。这意味着当且仅当输入的每一位都是 `true` 时，您的解决方案才应返回 `true`。

请注意，在这个问题中，解决方案并不依赖于是否使用位串转换为整数 
大端或小端表示法。一般来说，这是您需要考虑的一个重要区别。

来源：Solution.qs

```qsharp
namespace Kata {
    function IsSeven(x : Bool[]) : Bool {
        return x[0] and x[1] and x[2];
    }
}
```
