# 重建消息并对其进行测量

使用从 Alice 收到的两个经典位将 Bob 的量子位转换为所需的状态，并按照她准备消息的相同基础对其进行测量。

**输入：** 
1. Bob 属于纠缠量子位对 `qBob`。
2. 从 Alice 接收的经典位元组，采用发送消息练习中使用的格式。
3. 最初准备消息量子位的基础 - `PauliX`、`PauliY` 或 `PauliZ`。

**输出：** 
结果值，指示准备消息量子位的基础状态，`One` 或 `Zero`。操作结束时量子位 `qBob` 的状态并不重要。

## 解决方案注释

使用早期练习的解决方案可以轻松完成该练习。
- 首先，使用先前练习中的 `ReconstructMessage` 操作重建量子位 `qBob` 的状态。这会将 `qBob` 状态带入给定基的基状态之一。
- 现在，我们从“单量子位系统中的测量”中得知，默认测量操作 `M` 在泡利 $Z$ 基础上执行测量。要获得 `qBob` 量子位测量的正确结果，您需要使用操作 `Measure` 在给定基础上执行测量。

来源：Solution.qs

```qsharp
namespace Kata {
    operation ReconstructAndMeasureMessage(qBob : Qubit, (b1 : Bool, b2 : Bool), basis : Pauli) : Result {
        ReconstructMessage(qBob, (b1, b2));
        return Measure([basis], [qBob]);
    }

    operation ReconstructMessage(qBob : Qubit, (b1 : Bool, b2 : Bool)) : Unit {
        if b1 {
            Z(qBob);
        }
        if b2 {
            X(qBob);
        }
    }
}
```
