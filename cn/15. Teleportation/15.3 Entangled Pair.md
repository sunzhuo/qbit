# 纠缠对

**输入：** 
两个量子位 `qAlice` 和 `qBob`，每个都处于 $\ket{0}$ 状态。

**目标：**
在这些量子位上准备贝尔状态 $\ket{\Phi^{+}} = \frac{1}{\sqrt{2}}(\ket{00} + \ket{11})$。

## 解决方案注释

值得记住的是我们在“多量子位门”和“准备量子态”套路中已经看到的状态。

解决方案可以分为两步：
- 将 Hadamard 门应用于 Alice 的量子位： 
$\big(\frac{\ket{0} + \ket{1}}{\sqrt{2}}\big) \otimes \ket{0}$
- 应用 $CNOT$ 门，以 Alice 的量子位为控制，以 Bob 的量子位为目标。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Entangle (qAlice : Qubit, qBob : Qubit) : Unit is Adj {
        H(qAlice);
        CNOT(qAlice, qBob);
    }
}
```
