# 准备消息并发送

给定表示为泡利基的消息和指示该基的基状态的结果值，准备该状态下的消息量子位。 
结果值 `Zero` 和 `One` 对应于 `PauliZ` 基础的准备状态 $\ket{0}$ 或 $\ket{1}$，$\ket{+}$ 或 $\ket{-}$ 用于 `PauliX` 基础，以及 $\ket{i}$或 $\ket{-i}$ 为 `PauliY` 基础。然后将此 `qMessage` 量子位与 Alice 的量子位纠缠，并提取两个经典位发送给 Bob。

**输入：** 
1. Alice 属于纠缠量子位对 `qAlice` 的一部分。
2. 应准备消息量子位的基础 - `PauliX`、`PauliY` 或 `PauliZ`。
3. 指示量子位状态的结果值。

**输出：** 
Alice 将通过经典通道将两个经典位作为 `Bool` 值的元组发送给 Bob。与之前的任务相同，元组中的第一位应保存消息量子位的测量结果，第二位应保存 Alice 量子位的测量结果。将测量结果 `One` 表示为 `true`，将 `Zero` 表示为 `false`。操作结束时量子位 `qAlice` 的状态并不重要。

## 解决方案注释

该练习的目的是准备 `qMessage` 作为所需泡利基的基态之一。

- 根据 `state` 的值初始化 $\ket{0}$ 或 $\ket{1}$ 中的 `qMessage` 量子位。这些是泡利 $Z$ 门的本征态。
- 使用 $H$ 门，可以根据 Pauli $Z$ 的本征态准备 Pauli $X$ 门的本征态：
  $$\ket{+} = H\ket{0}, \ket{-} = H\ket{1}$$
- 使用 $S$ 门，可以根据 Pauli $X$ 的本征态准备 Pauli $Y$ 门的本征态：
  $$\ket{i} = S\ket{+}, \ket{-i} = S\ket{-}$$

最后，使用前面练习中实现的 `SendMessage` 操作来发送消息。

来源：Solution.qs

```qsharp
namespace Kata {
    operation PrepareAndSendMessage(qAlice : Qubit, basis : Pauli, state : Result) : (Bool, Bool) {
        use qMessage = Qubit();
        if state == One {
            X(qMessage);
        }
        if basis != PauliZ {
            H(qMessage);
        }
        if basis == PauliY {
            S(qMessage);
        }
        let classicalBits = SendMessage(qAlice, qMessage);
        Reset(qMessage);
        return classicalBits;
    }

    operation SendMessage(qAlice: Qubit, qMessage: Qubit) : (Bool, Bool) {
        CNOT(qMessage, qAlice);
        H(qMessage);
        return (M(qMessage) == One, M(qAlice) == One);
    }
}
```
