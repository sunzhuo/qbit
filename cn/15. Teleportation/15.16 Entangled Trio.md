# 纠缠三人组

**输入：** 
三个量子位 `qAlice`、`qBob` 和 `qCharlie`，每个都处于 $\ket{0}$ 状态。

**目标：** 
在这些量子位上创建纠缠态 $\ket{\Psi^{3}} = \frac{1}{2}(\ket{000}+\ket{011}+\ket{101}+\ket{110})$。

## 解决方案注释

我们可以将所需的状态表示如下：

$$\frac{1}{2}(\ket{000}+\ket{011}+\ket{101}+\ket{110}) = \frac1{\sqrt2} \left( \ket{0} \otimes \frac1{\sqrt2}(\ket{00} + \ket{11}) + \ket{1} \otimes \frac1{\sqrt2}(\ket{01} + \ket{10}) \right)$$
 
换句话说，根据量子位`qAlice`的状态，其他两个量子位在贝尔状态之一中准备。

- 从$\ket{000}$开始，第一步是在贝尔状态$\frac{1}{\sqrt{2}}(\ket{00} + \ket{11})$下准备`qBob`和`qCharlie`。这可以使用 Hadamard 和 $CNOT$ 操作来完成，类似于 `Entangle` 练习。这给出的结果状态为 $\frac{1}{\sqrt{2}}(\ket{000} + \ket{011})$。
- 此后，将 Hadamard 门应用于 `qAlice` 量子位，我们得到的状态为 $\frac{1}{2}(\ket{000} + \ket{011} + \ket{100} + \ket{111})$。
- 最后，我们需要调整 `qAlice` 处于 $\ket{1}$ 状态的基础状态，将其他两个量子位上的贝尔状态翻转为不同的状态。我们可以使用 $CNOT$ 门来完成此操作，以 `qAlice` 作为控制，以 `qBob` 或 `qCharlie` 作为目标。

来源：Solution.qs

```qsharp
namespace Kata {
    operation EntangleThreeQubits(qAlice : Qubit, qBob : Qubit, qCharlie : Qubit) : Unit is Adj {
        H(qBob);
        CNOT(qBob, qCharlie);
        H(qAlice);
        CNOT(qAlice, qCharlie);
    }
}
```
