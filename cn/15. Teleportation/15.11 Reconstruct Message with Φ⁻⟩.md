# 用 |Φ⁻⟩ 重建消息

**目标：** 
将 Bob 的量子位 `qBob` 转换为消息量子位最初的状态。假设 `qAlice` 和 `qBob` 最初纠缠在 $\ket{\Phi ^{-}} = \frac{1}{\sqrt{2}}(\ket{00}-\ket{11})$ 状态。

## 解决方案注释

Bob 这边的协议实际上可以表示为两个步骤的序列：

1. 将他与 Alice 共享的 Bell 对转换回 $\ket{\Phi ^+}$。这可以通过将 $Z$ 门应用于 Bob 的量子位来完成。
 2. 应用标准传送协议。

我们需要的最后一组校正是 $Z$ 门，然后是标准传送校正。这总计了以下更正：
- 对于 00，仅需要 $Z$ 校正。
- 对于 01，无需更改。
- 对于 10，需要 $Z$ 和 $X$ 更正。
- 对于 11，仅需要 $X$ 更正。

来源：Solution.qs

```qsharp
namespace Kata {
    operation ReconstructMessage_PhiMinus(qBob : Qubit, (b1 : Bool, b2 : Bool)) : Unit {
        if not b1 {
            Z(qBob);
        }
        if b2 {
            X(qBob);
        }
    }
}
```
