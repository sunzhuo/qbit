# 重建消息（鲍勃的任务）

使用从 Alice 收到的两个经典位将 Bob 的量子位转换为所需的状态。

**输入：** 
1. Bob 属于纠缠量子位对 `qBob`。
2. 从 Alice 处收到的经典位元组，采用之前练习中使用的格式。

**目标：** 
将 Bob 的量子位 `qBob` 转换为消息量子位最初的状态。

## 解决方案注释

鲍勃的量子位现在包含有关传送状态振幅的信息，但需要根据接收到的经典消息进行校正，以使他的量子位精确匹配传送状态。正如我们在上一个任务的解决方案中看到的，有四种可能的测量结果被编码为元组 (b1, b2)：

- 对于 (0,0)，Bob 的量子位最终处于状态 $\alpha \ket{0} + \beta \ket{1}$，因此不需要更改。
- 对于 (0,1)，Bob 的量子位最终处于状态 $\alpha \ket{1} + \beta \ket{0}$，因此我们需要应用 $X$ 校正。
- 对于 (1,0)，Bob 的量子位最终处于状态 $\alpha \ket{0} - \beta \ket{1}$，因此我们需要应用 $Z$ 校正。
- 对于 (1,1)，Bob 的量子位最终处于状态 $\alpha \ket{1} - \beta \ket{0}$，因此我们需要应用 $Z$ 和 $X$ 校正。

来源：Solution.qs

```qsharp
namespace Kata {
    operation ReconstructMessage(qBob : Qubit, (b1 : Bool, b2 : Bool)) : Unit {
        if b1 {
            Z(qBob);
        }
        if b2 {
            X(qBob);
        }
    }
}
```
