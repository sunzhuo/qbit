# 发送消息（Alice 的任务）

将消息量子位与 Alice 的量子位纠缠，并提取两个经典位发送给 Bob。

**输入：** 
1. Alice 属于纠缠量子位对 `qAlice` 的一部分。
2. 消息量子位`qMessage`。

**输出：** 
Alice 将通过经典通道将两个经典位作为布尔值元组发送给 Bob。元组中的第一位应保存消息量子位的测量结果，第二位应保存爱丽丝量子位的测量结果。将测量结果 `One` 表示为 `true`，将 `Zero` 表示为 `false`。操作结束时量子位的状态并不重要。

## 解决方案注释

该协议的这一步相当于在贝尔基础上执行测量。让我们看看 Alice 执行的步骤如何影响协议中涉及的三个量子位的联合状态。

最初，消息量子位处于状态 $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$，Alice 和 Bob 的量子位处于 Bell 状态 $\ket{\Phi^{+}} = \frac1{\sqrt2}(\ket{00} + \ket{11})$。我们将通过按以下顺序写出量子位的状态来跟踪状态的演变：消息量子位、爱丽丝的量子位、鲍勃的量子位。

$$\ket{\psi} \ket{\Phi^{+}} = (\alpha \ket{0} + \beta \ket{1}) \frac1{\sqrt2}(\ket{00} + \ket{11}) = $$
$$= \alpha \ket{0} \frac1{\sqrt2}(\ket{00} + \ket{11}) + \beta \ket{1} \frac1{\sqrt2}(\ket{00} + \ket{11})$$

接下来，我们应用 $CNOT$ 门，以 `qMessage` 作为控制，以 `qAlice` 作为目标。状态变成

$$\alpha \ket{0} \frac1{\sqrt2}(\ket{00} + \ket{11}) + \beta \ket{1} \frac1{\sqrt2}(\ket{\textbf{1}0} + \ket{\textbf{0}1})$$

之后，我们在 `qMessage` 上应用 Hadamard 门：

$$\alpha \mathbf{\frac1{\sqrt2}(\ket{0} + \ket{1})} \frac1{\sqrt2}(\ket{00} + \ket{11}) + \beta \mathbf{\frac1{\sqrt2}(\ket{0} - \ket{1})} \frac1{\sqrt2}(\ket{10} + \ket{01})$$

最后，我们测量两个量子位并将测量结果作为布尔值元组返回。
为了更容易地看到此步骤对状态的影响，让我们根据前两个量子位的状态重新组合术语：

$$\frac12 \left( \mathbf{\ket{00}} (\alpha \ket{0} + \beta \ket{1}) \right) +
  \frac12 \left( \mathbf{\ket{01}} (\alpha \ket{1} + \beta \ket{0}) \right) +$$
$$+ \frac12 \left( \mathbf{\ket{10}} (\alpha \ket{0} - \beta \ket{1}) \right) +
    \frac12 \left( \mathbf{\ket{11}} (\alpha \ket{1} - \beta \ket{0}) \right)$$

现在，当爱丽丝测量前两个量子位时，鲍勃的量子位最终处于一种振幅取决于原始消息量子位振幅的状态。

来源：Solution.qs

```qsharp
namespace Kata {
    operation SendMessage(qAlice : Qubit, qMessage : Qubit) : (Bool, Bool) {
        CNOT(qMessage, qAlice);
        H(qMessage);
        return (M(qMessage) == One, M(qAlice) == One);
    }
}
```
