# 无测量传送

在本练习中，您将把延迟测量原理应用于隐形传态电路。

**输入：** 
1. 两个量子位 `qAlice` 和 `qBob` 处于 $\ket{\Phi^{+}}$ 状态。
2. 处于状态 $\ket{\psi}$ 的消息量子位 `qMessage` 将被传送。

**目标：** 
不使用测量将 Bob 的量子位 `qBob` 转换为状态 $\ket{\psi}$。在操作结束时，量子位 `qAlice` 和 `qMessage` 不应与 `qBob` 纠缠在一起。

## 解决方案注释

让我们看看如何在不使用测量的情况下执行隐形传态。

我们开始与之前练习中的 `SendMessage` 操作类似，先应用 $CNOT$ 门，然后应用哈达玛门。此后，三量子位系统的状态将与常规隐形传态协议中的状态相同：

$$\frac12 \left( \ket{00} (\alpha \ket{0} + \beta \ket{1}) + \ket{01} (\alpha \ket{1} + \beta \ket{0}) + \ket{10} (\alpha \ket{0} - \beta \ket{1}) + \ket{11} (\alpha \ket{1} - \beta \ket{0}) \right)$$

现在，在标准协议中，我们将测量前两个量子位，以将最后一个量子位折叠成涉及系数 $\alpha$ 和 $\beta$ 的叠加之一。相反，让我们看看如何将 Bob 量子位的状态调整为前两个量子位的每个可能值的 $\alpha \ket{0} + \beta \ket{1}$。

为此，我们需要将每个测量后的经典条件门替换为受控门：

- 基于 `qMessage` 的测量结果应用的 $Z$ 门成为受控 $Z$，以 `qMessage` 作为控制，`qBob` 作为目标。
- 基于 `qAlice` 的测量结果应用的 $X$ 门成为受控 $X$，以 `qAlice` 作为控制，`qBob` 作为目标。

您可以检查这两个门是否将 Bob 的量子位转换为所需的状态，并使前两个量子位处于所有基础状态的相等叠加状态。

来源：Solution.qs

```qsharp
namespace Kata {
    operation MeasurementFreeTeleport(qAlice : Qubit, qBob : Qubit, qMessage : Qubit) : Unit {
        CNOT(qMessage, qAlice);
        H(qMessage);
        
        Controlled Z([qMessage], qBob);
        Controlled X([qAlice], qBob);
    }
}
```
