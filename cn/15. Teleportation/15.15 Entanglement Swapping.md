# 纠缠交换

Alice 和 Bob 彼此独立，各自持有状态为 $\ket{\Phi^{+}} = \frac{1}{\sqrt{2}}(\ket{00} + \ket{11})$ 的纠缠量子位对。他们每人将自己的一部分交给查理。

查理现在可以将爱丽丝的量子位的状态传送到鲍勃剩余的量子位上，从而传送纠缠。就像在“标准”隐形传输中一样，鲍勃仍然需要将重建步骤（基于查理的测量结果）应用于他拥有的另一个量子位。

在此过程之后，状态 $\ket{\Phi^{+}} = \frac{1}{\sqrt{2}}(\ket{00} + \ket{11})$ 现在跨越 Alice 和 Bob 的其他量子位 - 他们没有发送给 Charlie 的量子位。这些量子位现在处于最大程度的纠缠状态，尽管它们一开始从未相互作用！

**输出：** 
两个操作的元组：

1. 第一个操作是协议中 Charlie 的部分。它将采用两个量子位作为输入（Alice 和 Bob 发送给 Charlie 的量子位），并生成一条编码为整数的消息，Charlie 将将该消息发送给 Bob。

2. 第二个操作是协议中Bob 的部分。它将把 Bob 保留的量子位和 Charlie 的编码整数作为输入，并使用该消息来调整 Bob 的量子位的状态，以便 Alice 和 Bob 的量子位最终处于共享状态 $\ket{\Phi^{+}}$。

> 您将需要实现两个单独的辅助操作，一个用于协议的每个部分，然后返回它们，而不是在顶级操作的主体中实现解决方案。

## 解决方案注释

该练习演示了在 Alice 和 Bob 之间共享铃对而无需任何直接通信的能力。这需要爱丽丝和鲍勃都信任的第三方查理。

其实现可以分为以下几个操作：

- Alice 和 Bob 各自准备一个 Bell 状态，两个量子位处于 $\ket{\Phi^{+}} = \frac{1}{\sqrt{2}}(\ket{00} + \ket{11})$ 状态。
- Charlie 现在从 Alice `qAlice1` 接收一个量子位，从 Bob `qBob1` 接收另一个量子位。
- 现在，查理将爱丽丝的量子位 `qAlice1` 视为消息量子位，并将其状态传送给鲍勃。使用之前练习中实现的 `SendMessage` 操作，Charlie 获得两个 `Bool` 值并将它们编码为一个 `Int` - 经典消息。根据 Charlie 的测量结果，其余两个量子位 `qAlice2` 和 `qBob2` 可以处于四种贝尔状态之一。
- 最后，使用之前练习中实现的 `ReconstructMessage` 操作，Bob 可以对 `qBob2` 量子位进行校正，将 `qAlice2` 和 `qBob2` 量子位对的状态转换为 $\ket{\Phi^{+}}$。

在本练习中，您只需实现最后两个步骤。

```qsharp
namespace Kata {
    import Std.Convert.*;

    operation EntanglementSwapping() : ((Qubit, Qubit) => Int, (Qubit, Int) => Unit) {
        return (SendMessageCharlie, ReconstructMessageBob);
    }

    operation SendMessageCharlie(qAlice1 : Qubit, qBob1 : Qubit) : Int {
        let (c1, c2) = SendMessage(qAlice1, qBob1);
        return BoolArrayAsInt([c1, c2]);
    }

    operation ReconstructMessageBob(qBob2 : Qubit, resultCharlie : Int) : Unit {
        let classicalBits = IntAsBoolArray(resultCharlie, 2);
        ReconstructMessage(qBob2, (classicalBits[0], classicalBits[1]));
    }

    operation SendMessage(qAlice : Qubit, qMessage : Qubit) : (Bool, Bool) {
        CNOT(qMessage, qAlice);
        H(qMessage);
        return (M(qMessage) == One, M(qAlice) == One);
    }

    operation ReconstructMessage(qBob : Qubit, (b1 : Bool, b2 : Bool)) : Unit {
        if b1 {
            Z(qBob);
        }
        if b2 {
            X(qBob);
        }
    }
}
```
