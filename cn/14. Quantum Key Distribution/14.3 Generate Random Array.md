# 生成随机数组

**输入：** 整数 $N$。

**输出**：长度为 $N$ 的 `Bool` 数组，其中每个元素以 $50\%$ 概率随机选择为 `true` 或 `false`。

> Alice 和 Bob 都将使用它来选择要发送的位序列或碱基序列 
> 在编码/测量位时使用。

## 解决方案注释

您需要执行三个步骤来生成随机值数组：

1. 创建一个大小为 $N$ 的可变数组 - 它将需要一个默认值，可以是 `false`。
2. 对于从 $0$ 到 $N-1$ 的每个索引，随机选择两个值 `true` 或 `false` 之一，并将该值分配给该索引处的数组元素。您可以通过分配一个量子位、在 $\ket{+}$ 状态下准备它并测量它来选择随机位。但是，您不需要这些位具有量子起源，因此您可以改用 Q# 库操作 `DrawRandomInt`。
3.最后返回生成的数组。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Random.*;

    operation RandomArray(N : Int) : Bool[] {
        mutable array = [false, size = N];
        for i in 0..N - 1 {
            set array w/= i <- DrawRandomInt(0, 1) == 0;
        }
        return array;
    }
}
```
