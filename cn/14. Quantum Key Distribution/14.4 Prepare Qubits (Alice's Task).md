# 准备量子位（爱丽丝的任务）

**输入：**

1. `qs`：处于 $\ket{0}$ 状态的 $N$ 量子位数组，
2. `bases`：长度为$N$的`Bool`数组； 
    `bases[i]` 表示应用于准备量子位 `i` 的基础：
    * `false`：使用基础$\ket{0}$ / $\ket{1}$（计算），
    * `true`：使用基础 $\ket{+}$ / $\ket{-}$ (Hadamard)。
3. `bits`：长度为$N$的`Bool`数组；
    `bits[i]` 表示第 i 个量子位中要编码的位：`false` = 0，`true` = 1。

**目标：** 准备处于所描述状态的量子位。

## 解决方案注释

Alice 在发送给 Bob 之前总共可以准备 4 种状态，每种状态都对应于位和基数 bool 数组的唯一组合。

1. 状态$\ket{0}$对应于`bases[i]`处于计算基础中（即`false`）并且`bits[i]`等于0（即`false`）。
2. 状态$\ket{1}$对应于`bases[i]`处于计算基础中（即`false`）并且`bits[i]`等于1（即`true`）。
3. 状态$\ket{+}$对应于Hadamard基中的`bases[i]`（即`true`）并且`bits[i]`等于0（即`false`）。
4. 状态$\ket{-}$对应于Hadamard基中的`bases[i]`（即`true`）并且`bits[i]`等于1（即`true`）。

因此，如果 `bits[i]` 设置为 `true`，则需要将 $X$ 门应用于第 i 个量子位，然后如果 `bases[i]` 设置为 `true`，则需要将 $H$ 门应用于第 i 个量子位。

来源：Solution.qs

```qsharp
namespace Kata {
    operation PrepareQubits(qs : Qubit[], bases : Bool[], bits : Bool[]) : Unit {
        for i in 0 .. Length(qs) - 1 {
            if bits[i] {
                X(qs[i]);
            }
            if bases[i] {
                H(qs[i]);
            }
        }
    }
}
```
