# 测量量子位（鲍勃的任务）

**输入：**

1. `qs`：$N$量子位数组；  
   每个量子位处于以下状态之一：$\ket{0}$、$\ket{1}$、$\ket{+}$、$\ket{-}$。 
2. `bases`：长度为$N$的`Bool`数组； 
   `bases[i]` 表示用于测量量子位 `i` 的基础：
    * `false`：使用基础$\ket{0}$ / $\ket{1}$（计算），
    * `true`：使用基础 $\ket{+}$ / $\ket{-}$ (Hadamard)。

**目标：** 测量相应基中的每个量子位，并将结果数组返回为布尔值，将测量结果 `Zero` 编码为 `false`，将 `One` 编码为 `true`。 
操作结束时量子位的状态并不重要。

## 解决方案注释

如果 `bases[i]` 数组元素是 `true`，则意味着您正在为此量子位选择 Hadamard 基，因此需要应用 $H$ 门。否则，您可以选择计算基础，并且在测量量子位之前不需要应用 $H$ 门。

现在，输出预计是一个布尔数组，因此您需要测量每个量子位并将该测量结果转换为 `Bool`。 
1. 要在一次操作调用中测量每个量子位，您可以使用 Q# 库操作 `MeasureEachZ`。
2. 要将这些测量结果转换为布尔数组，可以使用函数 `ResultArrayAsBoolArray`，该函数将 `Result` 类型的数组作为输入，并返回所需的 `Bool` 类型的数组。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Convert.*;

    operation MeasureQubits(qs : Qubit[], bases : Bool[]) : Bool[] {
        for i in 0..Length(qs) - 1 {
            if bases[i] {
                H(qs[i]);
            }
        }
        return ResultArrayAsBoolArray(MeasureEachZ(qs));
    }
}
```
