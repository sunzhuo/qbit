# 所有具有相位翻转的两量子位基本向量

**输入：** 处于 $\ket{00}$ 状态的两个量子位（存储在长度为 2 的数组中）。

**目标：** 将量子位的状态更改为 $\frac{1}{2} \big(\ket{00} + \ket{01} + \ket{10} - \ket{11}\big)$。

## 解决方案注释

在这里，您从上一个任务 $\frac{1}{2} \big(\ket{00} + \ket{01} + \ket{10} + \ket{11}\big)$ 的结束状态开始。查看所需状态，$\ket{11}$ 状态的相位被翻转（$+$ 更改为 $-$）。

一个量子位上的常规相位翻转可以使用 $Z$ 门来完成：
$$\begin{bmatrix} 1 & 0 \\ 0 & -1 \end{bmatrix}$$
该门将仅在 $\ket{1}$ 状态下执行相位翻转：

$$Z(\alpha\ket{0} + \beta\ket{1}) = \alpha\ket{0} - \beta\ket{1}$$

在这种情况下，您只想翻转 $\ket{11}$ 状态的相位，而不是 $\ket{01}$ 状态的相位。为了实现这一点，您可以使用受控的 $Z$ 门；这将确保仅当控制位处于 $\ket{1}$ 状态时才应用 $Z$ 门，并且 $\ket{01}$ 状态不会改变。

> 在 Q# 中，您可以通过在门之前使用 `Controlled` 关键字来应用受控门。受控门将采用两个参数；第一个参数是控制量子位数组（您可以将多个量子位用作控制），第二个参数是传递到原始门的参数元组（在这种情况下，如果控制位为 $\ket{1}$，则它只是要应用门的量子位）。

来源：Solution.qs

```qsharp
namespace Kata {
    operation AllBasisVectorsWithPhaseFlip_TwoQubits (qs : Qubit[]) : Unit is Adj + Ctl {
        H(qs[0]);
        H(qs[1]);
        Controlled Z([qs[0]], qs[1]);
    }
}
```
