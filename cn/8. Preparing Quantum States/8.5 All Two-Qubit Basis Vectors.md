# 所有二量子位基础向量

**输入：** 处于 $\ket{00}$ 状态的两个量子位（存储在长度为 2 的数组中）。

**目标：** 将量子位的状态更改为 $\ket{+} \otimes \ket{+} = \frac{1}{2} \big(\ket{00} + \ket{01} + \ket{10} + \ket{11}\big)$。

## 解决方案注释

您知道 Hadamard 门将基本状态 $\ket{0}$ 映射到 $\frac{1}{\sqrt2}(\ket{0} + \ket{1})$，因此这是解决此问题的逻辑起点。

接下来，您会看到最终状态有一个 $\frac{1}{2}$ 项，暗示您可能正在应用涉及 $\frac{1}{\sqrt{2}}$ 项的两个操作。

现在，如何获得 $\ket{00} + \ket{01} + \ket{10} + \ket{11}$ 表达式？让我们看看表达式 $\ket{0} + \ket{1}$ 本身相乘会是什么样子：

$$\big(\ket{0} + \ket{1}\big) \otimes \big(\ket{0} + \ket{1}\big) = \ket{0}\ket{0} + \ket{0}\ket{1} + \ket{1}\ket{0} + \ket{1}\ket{1}$$

因此，将哈达玛门单独应用于每个量子位将提供所需的最终结果：

$$H\ket{0} \otimes H\ket{0} = \frac{1}{\sqrt2} \big(\ket{0} + \ket{1}\big) \otimes \frac{1}{\sqrt2}\big(\ket{0} + \ket{1}\big)
= \frac{1}{2} (\ket{00} + \ket{01} + \ket{10} + \ket{11})$$

Q# 数组与其他语言中的数组类似：您可以将数组 `qs` 的第 $i$ 个元素作为 `qs[i]` 访问（索引从 0 开始）。

来源：Solution.qs

```qsharp
namespace Kata {
    operation AllBasisVectors_TwoQubits (qs : Qubit[]) : Unit is Adj + Ctl {
        H(qs[0]);
        H(qs[1]);
    }
}
```
