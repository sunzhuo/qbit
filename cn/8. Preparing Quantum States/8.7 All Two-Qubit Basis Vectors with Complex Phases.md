# 所有具有复相的两量子位基本向量

**输入：** 处于 $\ket{00}$ 状态的两个量子位（存储在长度为 2 的数组中）。

**目标：** 将量子位的状态更改为 $\frac{1}{2} \big(\ket{00} + i\ket{01} - \ket{10} - i\ket{11}\big)$。

## 解决方案注释

您将从期望的最终结果开始解决问题。让我们看看是否可以从 $\big(\ket{00} + i\ket{01} - \ket{10} - i\ket{11}\big)$ 中分解任何表达式：

$$
\ket{00} + i\ket{01} - \ket{10} - i\ket{11}
= \ket{00} + \big(\ket{0} - \ket{1}\big) i\ket{1} - \ket{10} =
$$
$$
= \big(\ket{0} - \ket{1}\big) \ket{0} + \big(\ket{0} - \ket{1}\big) i\ket{1}
= \big(\ket{0} - \ket{1}\big) \otimes \big(\ket{0} + i\ket{1}\big)
$$

事实上，您能够将状态分解为两项的张量积，这意味着状态是可分离的。

这看起来很有希望。现在让我们尝试从另一端，即从$\ket{00}$的起始状态来解决问题。 
正如您在上一个任务中所看到的，对每个 $\ket{0}$ 应用 Hadamard 运算可以让您更接近分解的表达式：

$$
H\ket{0} \otimes H\ket{0} = \frac{1}{\sqrt2} \big(\ket{0} + \ket{1}\big) \otimes \frac{1}{\sqrt2} \big(\ket{0} + \ket{1}\big)
=\frac{1}{2} \big(\ket{0} + \ket{1}\big) \otimes \big(\ket{0} + \ket{1}\big) 
$$

如果比较这两个方程（同时忽略第二个方程中的 $\frac{1}{2}$ 项），您最终会得到需要对各个量子位执行以下转换：

$$
\ket{0} + \ket{1} \overset{???}\rightarrow \ket{0} - \ket{1}
$$

$$
\ket{0} + \ket{1} \overset{???}\rightarrow \ket{0} + i\ket{1}
$$

接下来，让我们看一下基本的门，特别是 Pauli Z 门：

$$Z = \begin{bmatrix} 1 & 0 \\ 0 & -1 \end{bmatrix}$$

如果将其应用于状态 $\frac{1}{\sqrt2} \big(\ket{0} + \ket{1}\big)$，它将保持基本状态 $\ket{0}$ 不变，并将 $\ket{1}$ 映射到 $-\ket{1}$。因此，

$$Z\frac{1}{\sqrt2} \big(\ket{0} + \ket{1}\big) = \frac{1}{\sqrt2} \big(\ket{0} - \ket{1}\big)$$

所以 $Z$ 门是如何进行第一次转换问题的答案。

寻找另一个门来解决第二个转换，您会找到 $S$ 门：

$$S = \begin{bmatrix} 1 & 0 \\ 0 & i \end{bmatrix}$$ 

如果将其应用于状态 $\frac{1}{\sqrt2} \big(\ket{0} + \ket{1}\big)$，它将保持基本状态 $\ket{0}$ 不变，并将 $\ket{1}$ 映射到 $i\ket{1}$。因此，

$$S\frac{1}{\sqrt2} \big(\ket{0} + \ket{1}\big) = \frac{1}{\sqrt2} \big(\ket{0} + i\ket{1}\big)$$

因此，$S$ 门现在回答了如何进行第二次转换的问题。

总结一下，你需要准备的状态可以表示如下：
$$ZH\ket{0} \otimes SH\ket{0}$$

请记住，在 Q# 中，与数学符号相比，必须以相反的顺序应用门 - 首​​先应用最接近 ket 符号的门。

```qsharp
namespace Kata {
    operation AllBasisVectorsWithComplexPhases_TwoQubits (qs : Qubit[]) : Unit is Adj + Ctl {
        // Qubit 0 is converted into |+⟩ and then z-rotated into |-⟩.
        H(qs[0]);
        Z(qs[0]);

        // Qubit 1 is converted into |+⟩ and then z-rotated into |i⟩.
        H(qs[1]);
        S(qs[1]);
    }
}
```
