# 哈代州

**输入：** 处于 $\ket{00}$ 状态的两个量子位。

**目标：** 将量子位的状态更改为 $\frac{1}{\sqrt{12}} \big(3\ket{00} + \ket{01} + \ket{10} + \ket{11} \big)$。

## 解决方案注释

您从状态 $\ket{\psi_0}=\ket{00}$ 开始。

**步骤 1：** 首先将第一个量子位置于状态 $\alpha\ket{0} + \beta\ket{1}$，其中 $\alpha$ 和 $\beta$ 分别是以 0 和 1 开头的所有基本状态的相对权重的平方根。

在 Hardy 状态的情况下，两个基本状态以 0 开头：$\frac{1}{\sqrt{12}} \big (3\ket{00} + \ket{01}\big )$，两个基本状态以 1 开头：$\frac{1}{\sqrt{12}} \big (\ket{10} + \ket{11}\big )$。
0 和 1 的相对权重只是它们幅度的平方和：
分别为$\alpha^2 = \frac{9}{12} + \frac{1}{12} = \frac{10}{12}$和$\beta^2 = \frac{1}{12} + \frac{1}{12} = \frac{2}{12}$。
因此，您需要使用 $R_y$ 门将第一个量子位置于状态 $\sqrt{\frac{10}{12}}\ket{0} + \sqrt{\frac{2}{12}}\ket{1}$ 中：

$$\ket{00} \overset{R_{y_1}}\rightarrow \big (\sqrt{\frac{10}{12}}\ket{0} + \sqrt{\frac{2}{12}}\ket{1} \big ) \otimes \ket{0} =: \ket{\psi_1}$$

这里是$R_{y_1} := R_y(2\arccos \sqrt{\frac{10}{12}}) \otimes I$。

**步骤 2：** 通过将第二个量子位置于正确的状态，应用受控 $R_y$ 门（以第一个量子位作为控制）来完成准备工作。

为了使前两项正确，您需要转换这些术语

$$\big (\sqrt{\frac{10}{12}}\ket{0} \big) \otimes \ket{0} \overset{R_{y_2}}\rightarrow \frac{1}{\sqrt{12}} \big (3\ket{00} + \ket{01} \big) \text{  (2.a) }$$
和
$$\big (\sqrt{\frac{2}{12}}\ket{1} \big) \otimes \ket{0} \overset{R_{y_3}}\rightarrow \frac{1}{\sqrt{12}} \big (\ket{10}+\ket{11} \big) \text{  (2.b) }$$

**步骤 2.a：** 转型
$$\big (\sqrt{\frac{10}{12}}\ket{0} \big) \otimes \ket{0} \overset{R_{y_2}}\rightarrow \frac{1}{\sqrt{12}} \big (3\ket{00} + \ket{01} \big)$$

等价于以下变换（要看到这一点，只需将两边乘以 $\sqrt{\frac{12}{10}}$）：

$$\ket{0} \otimes \ket{0} \overset{R_{y_2}}\rightarrow \frac{1}{\sqrt{10}} \big (3\ket{00} + \ket{01} \big) = \ket{0} \otimes \frac{1}{\sqrt{10}} \big (3\ket{0} + \ket{1} \big)$$

仅当第一个量子位处于状态 $\ket{0}$ 时才应应用此旋转，也就是说，您需要零条件旋转。旋转角度可以由$\cos\frac{\theta_2}{2} = \frac{3}{\sqrt{10}}$和$\sin\frac{\theta_2}{2} = \frac{1}{\sqrt{10}}$确定。

**步骤2.b：** 同样，转换

$$\big (\sqrt{\frac{2}{12}}\ket{1} \big) \otimes \ket{0} \overset{R_{y_3}}\rightarrow \frac{1}{\sqrt{12}} \big (\ket{10}+\ket{11} \big)$$

相当于

$$\ket{1} \otimes \ket{0} \overset{R_{y_3}}\rightarrow \frac{1}{\sqrt{2}} \big (\ket{10}+\ket{11} \big)$$

并且可以使用受控旋转来完成，如果第一个量子位是 $\ket{1}$，即有条件旋转，则应用该旋转。旋转角度可以由$\cos\frac{\theta_3}{2} = \frac{1}{\sqrt{2}}$和$\sin\frac{\theta_3}{2} = \frac{1}{\sqrt{2}}$确定。

```qsharp
namespace Kata {
    import Std.Math.*;

    operation Hardy_State(qs : Qubit[]) : Unit is Adj {
        let theta = ArcCos(Sqrt(10.0 / 12.0));
        Ry(2.0 * theta, qs[0]);

        ApplyControlledOnInt(0, Ry, [qs[0]], (2.0 * ArcCos(3.0 / Sqrt(10.0)), qs[1]));
        ApplyControlledOnInt(1, Ry, [qs[0]], (2.0 * PI() / 4.0, qs[1]));
    }
}
```
