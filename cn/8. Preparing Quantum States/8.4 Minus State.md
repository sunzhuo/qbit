# 减去状态

**输入：** 处于 $\ket{0}$ 状态的量子位。

**目标：** 将量子位的状态更改为 $\ket{-} = \frac{1}{\sqrt{2}} \big(\ket{0} - \ket{1}\big)$。

## 解决方案注释

正如您在上一个任务中所看到的，Hadamard 门将基本状态 $\ket{0}$ 映射到 $\frac{1}{\sqrt2}\big(\ket{0} + \ket{1}\big)$，将 $\ket{1}$ 映射到 $\frac{1}{\sqrt2}\big(\ket{0} - \ket{1}\big)$。 
如果您的量子位已经处于 $\ket{1}$ 状态，您只需应用 Hadamard 门即可准备所需的 $\ket{-}$ 状态。 
幸运的是，您可以使用另一个操作将状态 $\ket{0}$ 更改为 $\ket{1}$，即 $X$ 门：

$$X = \begin{bmatrix} 0 & 1 \\ 1 & 0 \end{bmatrix}$$

该门变换 $\ket{0} \longmapsto \ket{1}$ 和 $\ket{1} \longmapsto \ket{0}$。
$X$ 是 `Std.Intrinsic` 命名空间中 Q# 中的另一个内置门。

因此，您的解决方案应该将 $X$ 门应用于您的量子位，然后应用哈达玛门。

来源：Solution.qs

```qsharp
namespace Kata {
    operation MinusState(q : Qubit) : Unit is Adj + Ctl {
        X(q);
        H(q);
    }
}
```
