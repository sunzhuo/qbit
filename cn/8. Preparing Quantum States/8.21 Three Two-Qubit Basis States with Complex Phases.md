# 具有复杂相位的三个二量子位基础状态

**输入：** 处于 $\ket{00}$ 状态的两个量子位（存储在长度为 2 的数组中）。

**目标：** 将状态更改为 $\frac{1}{\sqrt{3}} \big( \ket{00} + \omega \ket{01} + \omega^2 \ket{10} \big)$，其中 $\omega = e^{2\pi i/3}$。

## 解决方案注释

首先，您将使用上一个任务的解决方案准备 $\frac{1}{\sqrt{3}} \big( \ket{00} +  \ket{01} + \ket{10} \big)$ 状态。要达到最终状态，您需要将相对相位添加到 $\ket{01}$ 和 $\ket{10}$ 基本状态，而不更改 $\ket{00}$ 状态。

首先，您想要将 $\ket{01}$ 状态转换为 $\omega \ket{01} = e^{2\pi i/3} \ket{01}$ 状态，同时不更改其他状态。
使用 [$R_1$](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.intrinsic/r1) 门，您可以将量子位状态从 $\ket{1}$ 更改为 $e^{i\theta}\ket{1}$，而不更改 $\ket{0}$ 状态。
事实上，这是 $R_1$ 门对 $\ket{0}$ 和 $\ket{1}$ 状态的影响：

$$ R_1 \ket{0} = \begin{bmatrix} 1 & 0 \\ 0 & e^{i\theta} \end{bmatrix} \cdot \begin{bmatrix} 1 \\ 0 \end{bmatrix} = \begin{bmatrix} 1 \\ 0 \end{bmatrix} = \ket{0} $$

$$ R_1 \ket{1} = \begin{bmatrix} 1 & 0 \\ 0 & e^{i\theta} \end{bmatrix} \cdot \begin{bmatrix} 0 \\ 1 \end{bmatrix} = \begin{bmatrix} 0 \\ e^{i\theta} \end{bmatrix} = e^{i\theta}\ket{1} $$

当您将 $R_1$ 门应用于第二个量子位时，这只会影响 $\ket{01}$ 项，这正是您想要的。知道了这一点，您只需要 $\theta$ 的正确值，在本例中为 $\frac{2\pi}{3}$。

您可以使用相同的方法将 $\ket{10}$ 项更改为 $\omega^2 \ket{10}$。通过将 $R_1$ 门应用于第一个量子位，您只会更改 $\ket{10}$ 项。在这种情况下，右侧的 $\theta$ 将是 $\frac{4\pi}{3}$。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    operation ThreeStates_TwoQubits_Phases(qs : Qubit[]) : Unit {
        // First create (|00⟩ + |01⟩ + |10⟩) / sqrt(3) state
        ThreeStates_TwoQubits(qs);

        R1(4.0 * PI() / 3.0, qs[0]);
        R1(2.0 * PI() / 3.0, qs[1]);
    }

    operation ThreeStates_TwoQubits(qs : Qubit[]) : Unit is Adj {
        let theta = ArcSin(1.0 / Sqrt(3.0));
        Ry(2.0 * theta, qs[0]);
        ApplyControlledOnInt(0, H, [qs[0]], qs[1]);
    }
}
```
