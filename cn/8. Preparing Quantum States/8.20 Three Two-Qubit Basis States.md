# 三个二量子位基础状态

**输入：** 处于 $\ket{00}$ 状态的两个量子位。

**目标：** 将量子位的状态更改为 $\frac{1}{\sqrt{3}} \big(\ket{00} + \ket{01} + \ket{10}\big)$。

## 解决方案注释

有多种方法可以准备此状态；所描述的第一个解决方案将侧重于在不使用任意旋转门的情况下准备这种状态。

最初，您将通过对每个起始量子位应用 $H$ 门来准备两个起始量子位上所有基本状态的相等叠加：
$$\frac{1}{2} \big(\ket{00} + \ket{01} + \ket{10} + \ket{11}\big)$$

此状态是您要准备的状态和您要丢弃的 $\ket{11}$ 状态的叠加。 
您可以通过在系统上执行正确的测量来准确地做到这一点。为此，您需要分配一个额外的量子位（有时称为 *ancilla* 量子位）。有了这个额外的量子位，新状态就变成了：
$$\frac{1}{2} \big(\ket{00\textbf{0}} + \ket{01\textbf{0}} + \ket{10\textbf{0}} + \ket{11\textbf{0}}\big)$$

现在，您想要将前三个基本状态与最后一个基本状态分开，并将这种分离存储在额外的量子位中。 
例如，您可以保留您想要保留的基础状态的额外量子位 $\ket{0}$ 的状态，并将其切换到您想要丢弃的基础状态的 $\ket{1}$。 
可以使用 $CCNOT$ 门来实现此目的，其中前两个量子位用作控制量子位，额外的量子位作为目标。 
应用门时，如果两个控制量子位都处于 $\ket{11}$ 状态，则额外量子位的状态只会更改为 $\ket{1}$，这恰好标记了您要丢弃的状态：

$$CCNOT\frac{1}{2} \big(\ket{00\textbf{0}} + \ket{01\textbf{0}} + \ket{10\textbf{0}} + \ket{11\textbf{0}}\big) = 
\frac{1}{2}\big(\ket{00} + \ket{01} + \ket{10} \big) \otimes \ket{\textbf{0}} + \frac{1}{2}\ket{11} \otimes \ket{\textbf{1}} $$

最后，您只测量额外的量子位；这会导致系统部分崩溃到测量结果定义的状态：
* 如果结果为 $\ket{0}$，则系统崩溃为基本状态的线性组合状态，其中额外量子位处于状态 $\ket{0}$，也就是说，两个量子位最终处于目标状态 $\frac{1}{\sqrt3}\big(\ket{00} + \ket{01} + \ket{10}\big)$。 
* 如果结果是$\ket{1}$，系统崩溃到状态$\ket{11}$，所以你的目标没有实现。好处是，这只发生在 $25\%$ 的情况下，您只需将量子位重置为 $\ket{00}$ 状态，然后重试。

> Q# 有一个内置的 [重复直到成功 (RUS) 循环](https://learn.microsoft.com/azure/quantum/user-guide/language/expressions/conditionalloops#repeat-expression)，在这种情况下会派上用场。 
> * 您将在循环的 `repeat` 部分中描述主要操作（应用 $H$ 和 $CCNOT$ 门以及测量），循环指定其主体。  
> * `until` 部分指定将打破循环的条件。在这种情况下，测量结果需要为 `Zero` 才能表明您成功。  
> * 最后，如果不满足成功标准，`fixup` 部分允许您在重试之前清理循环体执行的结果。在这种情况下，您将前两个量子位重置回 $\ket{00}$ 状态。

这种技术有时称为“后选择”。

来源：SolutionA.qs

```qsharp
namespace Kata {
    operation ThreeStates_TwoQubits (qs : Qubit[]) : Unit {
        use anc = Qubit();
        mutable res = Zero;
        repeat {
            ApplyToEach(H, qs);
            Controlled X(qs, anc);
            set res = MResetZ(anc);
        } 
        until res == Zero
        fixup {
            ResetAll(qs);
        }
    }
}
```

或者，可以使用任意旋转（$R_y$ 门）来准备此状态。

首先，您将尝试找到目标状态的分解，以便更轻松地了解如何准备状态。  
知道 $\ket{+} = \frac{1}{\sqrt{2}}\big(\ket{0}+\ket{1}\big)$，您可以将状态表示如下：

$$ \frac{1}{\sqrt{3}} \big(\ket{00} + \ket{01} + \ket{10}\big) = \frac{\sqrt{2}}{\sqrt{3}}\ket{0} \otimes \ket{+} + \frac{1}{\sqrt{3}}\ket{1} \otimes \ket{0} $$ 

要准备此状态，您首先要准备状态 $ \frac{\sqrt{2}}{\sqrt{3}}\ket{0} + \frac{1}{\sqrt{3}}\ket{1} $ 中的第一个量子位，同时保持第二个量子位不变。 
为此，您可以使用旋转门 $R_y$（请参阅单量子位门 kata），它将执行以下转换：
$$ R_y\ket{0} = \cos\frac{\theta}{2}\ket{0} + \sin\frac{\theta}{2}\ket{1} $$
您需要找到满足两者的 $\theta$ 值：
$$\cos\frac{\theta}{2} = \frac{\sqrt{2}}{\sqrt{3}} \text{ and } \sin\frac{\theta}{2} = \frac{1}{\sqrt{3}}$$

求解 $\theta$ 的最后一个方程得到 $\frac{\theta}{2} = \arcsin\frac{1}{\sqrt{3}}$ 或 $\theta = 2 \arcsin\frac{1}{\sqrt{3}}$。

当您将其应用于第一个量子位时，您将获得所需的中间状态：
$$ R_y(2 \arcsin\frac{1}{\sqrt{3}})\ket{0} \otimes \ket{0} = 
\left(\frac{\sqrt{2}}{\sqrt{3}}\ket{0} + \frac{1}{\sqrt{3}}\ket{1} \right) \otimes \ket{0} = 
\frac{\sqrt{2}}{\sqrt{3}}\ket{0} \otimes \ket{0} + \frac{1}{\sqrt{3}}\ket{1} \otimes \ket{0}$$

现在，该状态的第二项已经符合您的最终目标，因此您需要调整第一项： 
仅当第一个量子位处于 $\ket{0}$ 状态时，才在第二个量子位上准备 $\ket{+}$ 状态。 
为此，您将条件 $H$ 门应用于第二个量子位，如果第一个量子位处于 $\ket{0}$ 状态，这将给出您所需的状态：
$$ \frac{\sqrt{2}}{\sqrt{3}}\ket{0} \otimes \ket{+} + \frac{1}{\sqrt{3}}\ket{1} \otimes \ket{0}$$

> 在 Q# 中，您可以使用 [`ApplyControlledOnInt` 操作](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.canon/applycontrolledonint) 应用具有任意控制的条件门。 
> 在这种情况下，如果控制量子位处于 $\ket{0}$ 状态，您希望应用 $H$ 门，因此您将使用 `ApplyControlledOnInt(0, H)` 门。

来源：SolutionB.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    operation ThreeStates_TwoQubits(qs : Qubit[]) : Unit {
        let theta = ArcSin(1.0 / Sqrt(3.0));
        Ry(2.0 * theta, qs[0]);
        ApplyControlledOnInt(0, H, [qs[0]], qs[1]);
    }
}
```
