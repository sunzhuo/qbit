# 两个量子比特数的幂的 W 状态

**输入：** $N = 2^k$ 量子位处于 $\ket{0 \dots 0}$ 状态。

**目标：** 将量子位的状态更改为 [W 状态](https://en.wikipedia.org/wiki/W_state) - $N$ 基础状态在汉明权重为 1 的 $N$ 量子位上的相等叠加。

例如，对于 $N = 4$，所需状态为 $\frac{1}{2}\big(\ket{1000} + \ket{0100} + \ket{0010} + \ket{0001}\big)$。

<details>
  <summary><b>需要提示吗？</b></summary>
  您可以使用 <code>Controlled</code> 修饰符来执行任意受控门。
</details>

## 解决方案注释

如果将问题分解为最简单的情况并从那里开始构建，问题就会变得更容易管理。

1. 问题的最小实例$N = 1$需要准备$\ket{W_1} = \ket{1}$；这可以使用 $X$ 门轻松完成。

2. 下一个实例 $N = 2$ 需要准备 $\ket{W_2} = \frac{1}{\sqrt2}\big(\ket{10} + \ket{01}\big)$。它与您之前看到的贝尔状态之一相匹配，但如果您以递归术语来考虑它，那么准备它会更有趣（并且对后续步骤更有用！）。让我们看看如何用 $\ket{W_1}$ 来表达 $\ket{W_2}$：

$$\ket{W_2} = \frac{1}{\sqrt2}\big(\ket{10} + \ket{01}\big) = \frac{1}{\sqrt2}\big(\ket{W_1} \otimes \ket{0} + \ket{0} \otimes \ket{W_1}\big)$$

此表示建议您一个解决方案：将起始状态 $\ket{00}$ “拆分”为两项，在第一个量子位上为第一项准备 $\ket{W_1}$，在第二个量子位上为第二项准备 $\ket{W_1}$。
为此，您可以再次使用在 $\ket{+}$ 状态下准备的辅助量子位，并根据辅助量子位的状态控制第一个或第二个量子位上 $\ket{W_1}$ 状态的准备：

$$\ket{0}_{aux} \ket{00}_{reg} \overset{H}{\longrightarrow} \frac{1}{\sqrt2}(\ket{0}_{aux} + \ket{1}_{aux}) \otimes \ket{00}_{reg} = \frac{1}{\sqrt2}(\ket{0}_{aux} \ket{00}_{reg} + \ket{1}_{aux} \ket{00}_{reg})\overset{CNOT_0}{\longrightarrow}$$

$${\longrightarrow}\frac{1}{\sqrt2}(\ket{0}_{aux} \ket{W_1}\ket{0}_{reg} + \ket{1}_{aux} \ket{00}_{reg})\overset{CNOT_1}{\longrightarrow}$$

$${\longrightarrow}\frac{1}{\sqrt2}(\ket{0}_{aux} \ket{W_1}\ket{0}_{reg} + \ket{1}_{aux} \ket{0}\ket{W_1}_{reg})$$

> 辅助量子位现在与其余量子位纠缠在一起，因此您不能简单地重置它而不影响您准备使用它的叠加。

将寄存器设置为所需状态的最后一步是取消计算术语 $\ket{1}_{aux} \ket{0}\ket{W_1}_{reg}$ 的辅助量子位（另一个术语已将其置于状态 $\ket{0}$）。

为此，您需要考虑状态 $\ket{0}\ket{W_1} = \ket{01}$ 的显式表达式。与前面的任务类似，您将通过使用受控的 $X$ 门来取消计算本项的辅助量子位，其中辅助量子位作为目标，$\ket{01}$ 状态下的主寄存器作为控制。这将确保该门仅适用于该术语，而不适用于任何其他术语。

最后一步可以简化为使用更少的量子位作为控制：您可以仅使用处于状态 $\ket{1}$ 的主寄存器的第二个量子位作为控制，因为您知道如果第二个量子位处于状态 $\ket{1}$，则第一个量子位必须处于状态 $\ket{0}$（您不需要将它们都用作控制模式）。

3. 如果您更进一步，到达 $N = 4$，您将看到相同的递归逻辑可以应用于越来越大的问题。的确，

$$\ket{W_4} = \frac{1}{2}\big(\ket{1000} + \ket{0100} + \ket{0010} + \ket{0001}\big) = $$
$$= \frac{1}{\sqrt2} \big(\frac{1}{\sqrt2}(\ket{10} + \ket{01}) \otimes \ket{00} + \ket{00} \otimes \frac{1}{\sqrt2}(\ket{10} + \ket{01}) \big) = $$
$$= \frac{1}{\sqrt2} \big(\ket{W_2} \otimes \ket{00} + \ket{00} \otimes \ket{W_2}\big)$$

对于这种情况，您可以使用相同的方法：在 $\ket{+}$ 状态下准备一个辅助量子位，并使用它来控制寄存器前半部分和后半部分上 $W_2$ 状态的准备。最后一步是使用两个受控 $X$ 门来取消计算辅助量子位的 $\ket{1}$ 状态，并以状态 $\ket{1}$ 中寄存器后半部分的每个量子位作为控制。

相同的递归方法可以推广到 2 的任意幂作为寄存器大小。

来源：SolutionA.qs

```qsharp
namespace Kata {
    operation WState_PowerOfTwo (qs : Qubit[]) : Unit is Adj + Ctl {
        let N = Length(qs);

        if N == 1 {
            X(qs[0]);
        } else {
            let K = N / 2;
            use anc = Qubit();
            H(anc);

            ApplyControlledOnInt(0, WState_PowerOfTwo, [anc], qs[0 .. K - 1]);
            ApplyControlledOnInt(1, WState_PowerOfTwo, [anc], qs[K .. N - 1]);

            for i in K .. N - 1 {
                CNOT(qs[i], anc);
            }
        }
    }
}
```

此递归实现需要分配 $\log_2 N$ 个额外量子位来控制准备工作（每级递归一个）。您可以修改您的方法，一次仅使用一个额外的量子位。

为此，我们请注意，要准备 $\ket{W_{N}}$，您需要为辅助量子位的两种状态的一半量子位准备 $\ket{W_{N-1}}$ 状态，区别仅在于您使用的是寄存器的哪一半。这意味着您可以在量子位的前半部分准备 $\ket{W_{N-1}}$ 状态，并使用叠加的辅助量子位来控制寄存器前半部分与后半部分的交换。辅助量子位的未计算以与第一种方法相同的方式发生。

来源：SolutionB.qs

```qsharp
namespace Kata {
    operation WState_PowerOfTwo (qs : Qubit[]) : Unit is Adj + Ctl {
        let N = Length(qs);

        if N == 1 {
            X(qs[0]);
        } else {
            let K = N / 2;
            WState_PowerOfTwo(qs[0 .. K - 1]);

            use anc = Qubit();
            H(anc);

            for i in 0 .. K - 1 {
                Controlled SWAP([anc], (qs[i], qs[i + K]));
            }
            for i in K .. N - 1 {
                CNOT(qs[i], anc);
            }
        }
    }
}
```
