# 所有贝尔州

**输入：**

1. 两个处于 $\ket{00}$ 状态的量子位（存储在长度为 2 的数组中）。
2. 整数索引。

**目标：** 根据索引值将量子位的状态更改为贝尔状态之一：

  <tr>
    <th style="text-align:center">索引</th>
    <th style="text-align:center">状态</th>
  </tr>
  <tr>
    <td style="text-align:center">0</td>
    <td style="text-align:center">$\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} \big (\ket{00} + \ket{11}\big)$</td>
  </tr>
  <tr>
    <td style="text-align:center">1</td>
    <td style="text-align:center">$\ket{\Phi^{-}} = \frac{1}{\sqrt{2}} \big (\ket{00} - \ket{11}\big)$</td>
  </tr>
  <tr>
    <td style="text-align:center">2</td>
    <td style="text-align:center">$\ket{\Psi^{+}} = \frac{1}{\sqrt{2}} \big (\ket{01} + \ket{10}\big)$</td>
  </tr>
  <tr>
    <td style="text-align:center">3</td>

## 解决方案注释

> [Bell states](https://en.wikipedia.org/wiki/Bell_state) 形成 4 维空间中的正交基，描述 2 量子位系统的状态。 
您可以检查每个状态的范数为 1，并且每对状态的内积为 0。

目标是根据作为输入给出的 `index` 值，将 $\ket{00}$ 基础状态转换为贝尔基础状态之一。

本练习提供了两个解决方案，其中一个基于上一个任务，第二个帮助您理解将计算基转换为贝尔基的酉变换。

#### 解决方案 1

让我们使用您在上一个任务中准备的第一个贝尔状态，并根据 `index` 的值对其进行转换。

$$\frac{1}{\sqrt2} \big(\ket{00} + \ket{11}\big)$$

您需要应用什么转换才能达到最终状态？

* 如果是 `index = 0`，则不执行任何操作 - 准备状态已经是 $\ket{\Phi^{+}}$。

* 如果是`index = 1`，则需要在$\ket{11}$项中添加$-1$的相对相位。请记住，$Z$ 门对于量子位来说正是这样做的：
  
  $$Z(H\ket{0}) \otimes \ket{0} = \frac{1}{\sqrt2} \big(\ket{0} - \ket{1}\big) \otimes \ket{0} = \frac{1}{\sqrt2} \big(\ket{00} - \ket{10}\big)$$
  
如果您现在像以前一样应用 $CNOT$，您将拥有：

  $$\frac{1}{\sqrt2} \big(\ket{00} - \ket{\overset{\curvearrowright}{10}}\big) \underset{\text{CNOT}}{\Longrightarrow} \frac{1}{\sqrt2} \big(\ket{00} - \ket{11}\big) = \ket{\Phi^{-}}$$

* 如果是 `index = 2`，您需要更改 $\ket{00}$ 和 $\ket{11}$ 项中的第二个量子位，这可以通过应用 $X$ 门来完成：
  
  $$H\ket{0} \otimes X\ket{0} = H\ket{0} \otimes \ket{1} = \frac{1}{\sqrt2} \big(\ket{0} + \ket{1}\big) \otimes \ket{1} = \frac{1}{\sqrt2} \big(\ket{01} + \ket{11}\big)$$
  
如果您现在像以前一样应用 $CNOT$，您将拥有：
  
  $$\frac{1}{\sqrt2} \big(\ket{01} + \ket{\overset{\curvearrowright}{11}}\big) \underset{\text{CNOT}}{\Longrightarrow} \frac{1}{\sqrt2} \big(\ket{01} + \ket{10}\big) = \ket{\Psi^{+}}$$

* 如果是 `index = 3`，您使用相同的逻辑来认识到您需要同时应用 $Z$ 和 $X$ 校正才能获得 $\ket{\Psi^{-}}$ 状态。

最终的步骤顺序如下：
1. 将 $H$ 门应用于第一个量子位。 
2. 如果 `index == 1` 或 `index == 3`，则将 $Z$ 门应用于第一个量子位。
3. 如果 `index == 2` 或 `index == 3`，则将 $X$ 门应用于第二个量子位。
4. 应用 $CNOT$ 门，将第一个量子位作为控制，将第二个量子位作为目标。

来源：SolutionA.qs

```qsharp
namespace Kata {   
    operation AllBellStates (qs : Qubit[], index : Int) : Unit is Adj + Ctl {
        H(qs[0]);
        
        if index == 1 {
            Z(qs[0]);
        }
        if index == 2 {
            X(qs[1]);
        }
        if index == 3 {
            Z(qs[0]);
            X(qs[1]);
        }
        
        CNOT(qs[0], qs[1]);
    }
}
```

#### 解决方案2

让我们仔细看看上一个任务中讨论的酉变换 $\text{CNOT}\cdot(H \otimes I)$。

$$\frac{1}{\sqrt2} \begin{bmatrix} 1 & 0 & 1 & 0 \\ 0 & 1 & 0 & 1 \\ 0 & 1 & 0 & -1 \\ \underset{\ket{\Phi^{+}}}{\underbrace{1}} & \underset{\ket{\Psi^{+}}}{\underbrace{0}} & \underset{\ket{\Phi^{-}}}{\underbrace{-1}} & \underset{\ket{\Psi^{-}}}{\underbrace{0}} \end{bmatrix}$$

请注意，酉矩阵中的每一列对应于贝尔状态之一。
这种酉变换将计算基础转换为贝尔基础，这正是任务要求您做的。

您会看到此转换将 $\ket{00}$ 转换为第一个 Bell 状态，将 $\ket{01}$ 转换为第二个 Bell 状态，等等。 
您只需确保在应用此转换之前将量子位设置为正确的状态，如果需要，请使用 $X$ 门将初始 $\ket{0}$ 状态更改为 $\ket{1}$。

在 Q# 中，您可以使用 <a href="https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.convert/intasboolarray">IntAsBoolArray</a> 函数将输入 `index` 转换为正确的位模式。

来源：SolutionB.qs

```qsharp
namespace Kata {
    import Std.Convert.*;

    operation AllBellStates(qs : Qubit[], index : Int) : Unit is Adj + Ctl {
        let bitmask = IntAsBoolArray(index, 2);

        if bitmask[0] {
            X(qs[0]);
        }

        if bitmask[1] {
            X(qs[1]);
        }

        H(qs[0]);
        CNOT(qs[0], qs[1]);
    }
}
```
