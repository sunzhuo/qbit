# 偶数或奇数基向量

**输入：**

1. $N$ ($N \ge 1$) 个量子位处于 $\ket{0 \dots 0}$ 状态。
2. 布尔值 `isEven`。

**目标：** 如果 `isEven` 为 `true`，则准备所有*偶*数的叠加；如果 `isEven` 为 `false`，则准备所有*奇*数的叠加。  
基本状态使用 [big-endian](https://en.wikipedia.org/wiki/Endianness) 二进制表示法对整数进行编码：状态 $\ket{01}$ 对应于整数 $1$，状态 $\ket{10 }$ 对应于整数 $2$。

> 例如，对于$N = 2$和`isEven = false`你需要准备叠加$\frac{1}{\sqrt{2}} \big (\ket{01} + \ket{11}\big )$，  
对于 $N = 2$ 和 `isEven = true` - 叠加 $\frac{1}{\sqrt{2}} \big (\ket{00} + \ket{10}\big )$。

## 解决方案注释

让我们看一些基本状态的示例来说明二进制编号系统。

$N = 2$ 量子位上的 4 个基础状态可以分为两列，其中左列表示形成 `isEven = true` 所需叠加状态的基础状态，右列表示形成 `isEven = false` 所需叠加状态的基础状态。

  <tr><th>偶</th><th>奇</th></tr>
  <tr><td> <b>0</b>0 </td><td><b>0</b>1 </td></tr>
  <tr><td> <b>1</b>0 </td><td><b>1</b>1 </td></tr>
</table>
 
如果对 $N = 3$ 量子位进行相同的基础状态分割，该模式将变得更加明显。

<table>
  <tr><th>偶</th><th>奇</th></tr>
  <tr><td><b>00</b>0 </td><td><b>00</b>1</td></tr>
  <tr><td> <b>01</b>0 </td><td><b>01</b>1</td></tr>
  <tr><td> <b>10</b>0 </td><td><b>10</b>1</td></tr>
  <tr><td> <b>11</b>0 </td><td><b>11</b>1</td></tr>
</table>

最左边的两个量子位经历 `isEven = true` 和 `isEven = false` 的所有可能的基本状态，最右边的量子位对于 `isEven = true` 保持在 $\ket{0}$ 状态，对于 `isEven = false` 保持在 $\ket{1}$ 状态。

对 $N = 4$ 量子位的快速健全性检查重新确认了该模式。

同样，最左边的三个量子位会经历两列中所有可能的基础状态，而最右边的量子位在每列中都保持相同的状态。

解决方案是将除最右边的量子位之外的所有量子位放入相等的叠加（类似于您在“所有基本向量的叠加”任务中所做的操作），并根据 `isEven` 标志将最右边的量子位设置为 $\ket{0}$ 或 $\ket{1}$，使用 $X$ 运算符转换 $\ket{0}$如果 `isEven = false` 则为 $\ket{1}$。
 

来源：Solution.qs

```qsharp
namespace Kata {
    operation EvenOddNumbersSuperposition(qs : Qubit[], isEven : Bool) : Unit is Adj + Ctl {
        let N = Length(qs);

        for i in 0 .. N-2 {
            H(qs[i]);
        }

        // for odd numbers, flip the last bit to 1
        if not isEven {
            X(qs[N-1]);
        }
    }
}
```
