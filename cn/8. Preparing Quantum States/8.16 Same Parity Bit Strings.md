# 相同的奇偶校验位串

**输入：**
1. $N$ ($N \ge 1$) 个量子位处于 $\ket{0 \dots 0}$ 状态。
2. `Int` `parity`。

**目标：** 将状态更改为所有具有的基础状态的相等叠加
* 其中有偶数个 1 if `parity = 0`，或者
* if `parity = 1` 其中有奇数个 1。

> 例如，对于 $N = 2$，如果 `parity = 0`，则所需状态为 $\frac{1}{\sqrt{2}}\big(\ket{00} + \ket{11}\big)$；如果 `parity = 1`，则所需状态为 $\frac{1}{\sqrt{2}}\big(\ket{01} + \ket{10}\big)$。

## 解决方案注释

有多种方法可以解决这个问题。第一个解决方案使用任务中暗示的递归方法。

我们将 $N$ 量子位上所需的状态表示为 `parity = 0` 的 $\ket{P_{N,0}}$ 和 `parity = 1` 的 $\ket{P_{N,1}}$。您可以按状态 $\ket{P_{N,0}}$ 中包含的 $2^{N-1}$ 基础状态按其第一位（$0$ 或 $1$）进行分组，并将状态写入如下：

$$\ket{P_{N,0}} = \frac{1}{\sqrt{2^{N-1}}} \sum_{k : k \text{ has parity }0} \ket{k}_N = $$
$$= \frac{1}{\sqrt{2^{N-1}}} \big( \ket{0} \otimes \sum_{k' : k' \text{ has parity }0} \ket{k'}_{N-1} + \ket{1} \otimes \sum_{k'' : k'' \text{ has parity }1} \ket{k''}_{N-1} \big) = $$
$$= \frac{1}{\sqrt{2}} \big( \ket{0} \otimes \ket{P_{N-1,0}} + \ket{1} \otimes \ket{P_{N-1,1}} \big)$$
您可以以类似的方式考虑 $\ket{P_{N,1}}$ 的表达式，并获得两种状态的统一表达式：
$$\ket{P_{N,p}} = \frac{1}{\sqrt{2}} \big( \ket{0} \otimes \ket{P_{N-1,p}} + \ket{1} \otimes \ket{P_{N-1,1-p}} \big)$$

现在您可以使用此表达式，使用您之前见过的递归方法来准备状态：

1. 将 $H$ 门应用于第一个量子位以准备状态 $\frac{1}{\sqrt{2}} ( \ket{0} + \ket{1} ) \otimes \ket{0}_{N-1}$。
2. 对最后一个 $N-1$ 量子位应用准备 $\ket{P_{N-1,p}}$ 过程的受控变体，并以 $\ket{0}$ 状态中的第一个量子位作为对照。
3. 对最后一个 $N-1$ 量子位应用准备 $\ket{P_{N-1,1-p}}$ 过程的受控变体，以 $\ket{1}$ 状态中的第一个量子位作为对照。

> Q# 库函数 [`ApplyControlledOnInt`](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.canon/applycontrolledonint) 让您可以轻松做到这一点。

4. 递归的基础是准备 $N = 1$ 的状态：
* 对于 `parity = 0`，有一个具有此奇偶校验的单量子位状态：$\ket{0}$（无需准备任何操作）。
* 对于 `parity = 1`，有一个具有此奇偶校验的单量子位状态：$\ket{1}$（应用 $X$ 门进行准备）。

来源：SolutionA.qs

```qsharp
namespace Kata {
    operation AllStatesWithParitySuperposition (qs : Qubit[], parity : Int) : Unit is Adj + Ctl {
        if Length(qs) == 1 {
            if parity == 1 {
                X(qs[0]);
            }
        } else {
            H(qs[0]);
            ApplyControlledOnInt(0, AllStatesWithParitySuperposition, qs[0 .. 0], (qs[1 ...], parity));
            ApplyControlledOnInt(1, AllStatesWithParitySuperposition, qs[0 .. 0], (qs[1 ...], 1 - parity));
        }
    }
}
```

第二种解决方案使用后选择。首先准备所有基础状态的相等叠加并分配一个额外的量子位。

这次，您使用额外的量子位来计算输入状态的奇偶校验：应用一系列 $CNOT$ 门，每个门以一个输入量子位作为控制，额外的量子位作为目标，将计算状态的奇偶校验。

现在，您测量额外的量子位：如果测量结果与奇偶校验匹配，那么您就完成了 - 输入量子位折叠为具有该奇偶校验的所有状态的相等叠加。如果测量结果相反，您可以重试整个过程。

如果您的测量结果与所需的奇偶校验不匹配，您可以避免重试状态准备：请注意，将 $X$ 门应用于任何一个量子位会将每个基本状态的奇偶校验更改为相反的奇偶校验，从而将您获得的状态转换为您需要的状态。

来源：SolutionB.qs

```qsharp
namespace Kata {
    operation AllStatesWithParitySuperposition (qs : Qubit[], parity : Int) : Unit {
        use aux = Qubit();
        ApplyToEach(H, qs);
        ApplyToEach(CNOT(_, aux), qs);
        let res = MResetZ(aux);
        if ((res == Zero ? 0 | 1) != parity) {
            X(qs[0]);
        }
    }
}
```

获得所需叠加的另一种方法可能是迭代地准备所有量子位的所有可能基础状态的混合，并保持每一步的奇偶性。

首先准备所有具有奇偶校验 $0$ 的基本状态的叠加。为了实现这一点，您需要循环遍历除第一个量子位之外的所有量子位，并以相等的叠加方式准备所有量子位。为了保持所涉及的基本状态的奇偶性，请使用第一个量子位，并使用 $CNOT$ 门有条件地翻转其状态，并以每个下一个量子位作为控制，以便每次链中存在 $\ket{1}$ 状态时，您都会回到该基本状态中偶数个 $\ket{1}$ 状态。

例如，在第一次循环迭代之后，您将获得状态 $\frac12(\ket{00} + \ket{11})$。第二次迭代后，你得到状态

$$CNOT_{2,0} \frac1{\sqrt2}(\ket{00} + \ket{11}) \otimes \frac1{\sqrt2}(\ket{0} + \ket{1}) = $$
$$= CNOT_{2,0} \frac12(\ket{000} + \ket{\textbf{0}0\textbf{1}} + \ket{110} + \ket{\textbf{1}1\textbf{1}}) = $$
$$= \frac12(\ket{000} + \ket{101} + \ket{110} + \ket{011})$$

循环结束后，您将获得所有可能的基本状态与偶数个 $\ket{1}$ 的叠加。
然后，如果 `parity` 等于 1 并且我们想要奇数个 $\ket{1}$，则只需再次翻转第一个量子位（或任何量子位）的状态即可。

例如，如果输入有 3 个量子位，则在 for 循环之后，您将得到 4 个基本状态的叠加：

$$\frac12(\ket{000} + \ket{101} + \ket{110} + \ket{011})$$

如果 `parity = 0`，则完成，拥有偶数个 $\ket{1}$。如果 `parity = 1`，则翻转第一个量子位的状态，得到所需的结果：

$$\frac12(\ket{100} + \ket{001} + \ket{010} + \ket{111})$$

来源：SolutionC.qs

```qsharp
namespace Kata {
    operation AllStatesWithParitySuperposition (qs : Qubit[], parity : Int) : Unit {
        for i in 1 .. Length(qs) - 1 {
            H(qs[i]);
            CNOT(qs[i], qs[0]);
        }

        if parity == 1 {
            X(qs[0]);
        }
    }
}
```
