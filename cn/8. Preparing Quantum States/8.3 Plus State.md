# 加上状态

**输入：** 处于 $\ket{0}$ 状态的量子位。

**目标：** 将量子位的状态更改为 $\ket{+} = \frac{1}{\sqrt{2}} \big(\ket{0} + \ket{1}\big)$。

<details>
  <summary><b>需要提示吗？</b></summary>
  Hadamard 门 $H$ 会将 $\ket{0}$ 状态转换为 $\ket{+}$ 状态。
</details>

## 解决方案注释

当您开始学习基本量子门时，首先描述的门之一将是哈达玛门：

$$H = \frac{1}{\sqrt2} \begin{bmatrix} 1 & 1 \\ 1 & -1 \end{bmatrix}$$

该门将 $\ket{0}$ 转换为 $\ket{+} = \frac{1}{\sqrt{2}} \big(\ket{0} + \ket{1}\big)$，将 $\ket{1}$ 转换为 $\ket{−} = \frac{1}{\sqrt{2}} \big(\ket{0} - \ket{1}\big)$。  这些转变中的第一个正是您正在寻找的！

Hadamard 门是 Q# 中的内置门之一，可在 `Std.Intrinsic` 命名空间中使用。
默认情况下，它会导入到任何 Q# 源文件中，因此您可以立即使用它。

来源：Solution.qs

```qsharp
namespace Kata {
    operation PlusState(q : Qubit) : Unit is Adj + Ctl {
        H(q);
    }
}
```
