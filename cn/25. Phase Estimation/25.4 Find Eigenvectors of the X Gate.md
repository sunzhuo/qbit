# 求 X 门的特征向量

**输入：** 无。

**输出：** 返回对应于不同特征值的 $X$ 门的两个特征向量的数组。

$$X = \begin{bmatrix} 0 & 1 \\ 1 & 0 \end{bmatrix}$$

特征向量不必标准化，也就是说，它们不必描述有效的量子态。
两个特征向量都必须非零。

> 在此任务中，特征向量表示为长度为 $2$ 的 Q# `Double` 类型的数组。您的返回应该是两个此类数组的数组。

## 解决方案注释

由于 $X$ 门是自伴的，因此您知道它的特征值只能是 $+1$ 和 $-1$。
现在，您需要找到与这些特征值相对应的特征向量。 
为此，您需要求解两个方程：
- $X \begin{bmatrix} v_0 \\ v_1 \end{bmatrix} = \begin{bmatrix} v_0 \\ v_1 \end{bmatrix}$，它为您提供$v_0 = v_1$。
- $X \begin{bmatrix} v_0 \\ v_1 \end{bmatrix} = -\begin{bmatrix} v_0 \\ v_1 \end{bmatrix}$，它为您提供$v_0 = -v_1$。

其中一个特征向量应由两个相等的元素组成，另一个特征向量应由两个绝对值相等但符号相反的元素组成。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function EigenvectorsX() : Double[][] {
        return [
            [1.0, 1.0],
            [1.0, -1.0]
        ];
    }
}
```
