# 一位相位估计

**输入：**

1. 保证具有特征值 $+1$ 或 $-1$ 的单量子位酉 $U$ 
（分别具有特征相 $0$ 或 $\frac12$）。统一体将定义其受控变体。
2. 单量子位状态 $\ket{\psi}$，作为从 $\ket{0}$ 状态准备的单一 $P$ 给出。换句话说，将酉 $P$ 应用到状态 $\ket{0}$ 的结果是 $\ket{\psi}$ 状态：
   $$P\ket{0} = \ket{\psi}$$

**输出：** 返回与特征状态 $\ket{\psi}$（$+1$ 或 $-1$）相对应的特征值。

<details>
  <summary><b>需要提示吗？</b></summary>
  您可以通过精确分配两个量子位并调用 <code>Controlled U<\code> 一次来实现此目的。
</details>

## 解决方案注释

如果将单一 $U$ 应用于状态 $\ket{\psi}$ 会发生什么？您最终会处于 $\ket{\psi}$ 或 $-\ket{\psi}$ 状态之一，具体取决于特征值是 $+1$ 还是 $-1$。您需要区分这两种情况，但它们在全局阶段上有所不同，因此您无法仅使用一个量子位来做到这一点。

但是，如果您使用了 $U$ 的受控变体并且可以分配多个量子位这一事实，则可以使用相位回扣技巧的变体来区分这些场景。

如果将受控 $U$ 门应用于两个量子位：控制量子位处于 $\ket{+}$ 状态，目标量子位处于状态 $\ket{\psi}$，您将得到以下状态：

$$CU \ket{+}\ket{\psi} = \frac1{\sqrt2}(CU \ket{0}\ket{\psi} + CU \ket{1}\ket{\psi}) = \frac1{\sqrt2}(\ket{0}\ket{\psi} + \ket{1}U\ket{\psi}) = $$
$$= \frac1{\sqrt2}(\ket{0}\ket{\psi} + \ket{1}\lambda\ket{\psi}) = 
\begin{cases}
\ket{+}\ket{\psi} \textrm{ 如果 } \lambda = 1 \\ 
\ket{-}\ket{\psi} \textrm{ 如果 } \lambda = -1
\end{cases}$$

您只需测量哈达玛基中的控制量子位即可确定其状态是$\ket{+}$还是$\ket{-}$，然后就可以知道特征值$\lambda$的值。

来源：Solution.qs

```qsharp
namespace Kata {
    operation OneBitPhaseEstimation(U : Qubit => Unit is Ctl, P : Qubit => Unit) : Int {
        use (control, eigenstate) = (Qubit(), Qubit());
        H(control);
        P(eigenstate);
        Controlled U([control], eigenstate);
        Reset(eigenstate);

        return MResetX(control) == Zero ? 1 | -1;
    }
}
```
