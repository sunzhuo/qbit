# 量子相位估计算法

量子相位估计算法是用于估计与给定酉特征向量对相对应的特征相位的最常用算法。
它依赖于量子傅立叶变换，并允许您估计本征相位 $\theta$ 的 $n$ 二进制数字。
让我们看看这个算法是如何工作的。

### 输入

1. 作用于 $m$ 量子位的单量子位单一 $U$。
2. $m$-量子位状态 $\ket{\psi}$ - $U$ 的特征向量。
3. 一个整数 $n$，指定本征相位估计所需的二进制精度。

### 起始状态

该算法作用于 $n+m$ 量子位，分为两个寄存器。
- 第一个 $n$ 量子位从状态 $\ket{0...0} = \ket{0}^{\otimes N}$ 开始。这些量子位用于获取特征值。
- 最后的 $m$ 量子位以状态 $\ket{\psi}$ 开始 - $U$ 的特征向量。

### 步骤 1. 将 Hadamard 变换应用于第一个寄存器的每个量子位

此步骤准备所有 $n$-qubit 基础状态的偶数叠加中的第一个量子位，或者等效地，从 $\ket{0}$ 到 $\ket{2^n-1}$ 的所有整数的偶数叠加中准备第一个量子位。

系统的状态最终将是

$$\frac1{\sqrt{2^n}}\sum_{k=0}^{2^n-1} \ket{k} \otimes \ket{\psi}$$

### 步骤 2. 应用受控的 $U$ 梯子

在此步骤中，算法应用一系列受控的 $U$ 门，其中第一个寄存器的不同量子位作为控制，第二个寄存器作为目标。

假设第一个寄存器以小端表示法存储整数：第一个量子位对应于最低有效位，最后一个量子位对应于最高有效位。然后，在此步骤中应用的门序列如下所示：

1. 以第一个量子位作为控制的受控 $U$ 门。
2. 以第二个量子位为控制的受控 $U^2$ 门。
3. 以第三个量子位为控制的受控 $U^4$ 门。
4. 依此类推，以受控 $U^{2^{n-1}}$ 门结束，并以量子位 $n$ 作为控制。

这一系列门的作用是什么？

让我们考虑这样的场景，其中第一个寄存器处于基本状态 $\ket{k} = \ket{k_0} \otimes \ket{k_1} \otimes ... \otimes \ket{k_{n-1}}$，其中 $k$ 是一个整数，其二进制表示为 $n$ 位：
$$k = 2^0k_0 + 2^1 k_1 + 2^2k_2 + ... + 2^{n-1}k_{n-1}$$

在这种情况下，该门序列将把单一的 $U^k$ 应用到第二个寄存器！的确，

1. 如果是 $k_0 = 1$，则以第一个量子位作为控制的受控 $U$ 门将应用 $U$ 门，如果 $k_0 = 0$，则不执行任何操作。
2. 如果是 $k_1 = 1$，则以第二个量子位作为控制的受控 $U^2$ 门将应用 $U^2$ 门，如果 $k_1 = 0$，则不执行任何操作。
3. 如果是 $k_2 = 1$，则以第三个量子位作为控制的受控 $U^4$ 门将应用 $U^4$ 门，如果 $k_2 = 0$，则不执行任何操作。
4.等等。

您可以看到，最终应用于第二个寄存器的 $U$ 的所有幂加起来正好等于第一个寄存器中写入的数字 $k$。

如果第一个寄存器处于状态叠加（如该算法的情况），您可以使用酉变换的线性来编写受控门对系统状态的影响：

$$\frac1{\sqrt{2^n}}\sum_{k=0}^{2^n-1} \ket{k} \otimes \ket{\psi}\rightarrow \frac1{\sqrt{2^n}}\sum_{k=0}^{2^n-1} \ket{k} \otimes U^k\ket{\psi}$$

现在，回想一下 $\ket{\psi}$ 是 $U$ 的特征向量：
$$U\ket{\psi} = e^{2 \pi i \theta} \ket{\psi}$$

这意味着
$$U^k\ket{\psi} = (e^{2 \pi i \theta})^k \ket{\psi} = e^{2 \pi i \theta \cdot k} \ket{\psi}$$

在此步骤之后，您可以使用它为系统状态编写以下表达式：

$$\frac1{\sqrt{2^n}}\sum_{k=0}^{2^n-1} e^{2 \pi i \theta \cdot k} \ket{k} \otimes \ket{\psi}$$

请注意，第一和第二寄存器最终不会相互纠缠。

### 步骤 3. 应用逆 QFT

现在，回想一下，作用于表示 $n$ 位整数 $j$ 的基本状态 $\ket{j}$ 的量子傅立叶变换执行以下变换：

$$\ket{j} \rightarrow \frac1{\sqrt{2^n}}\sum_{k=0}^{2^n-1}  e^{2\pi i jk/2^n} \ket{k} = \frac1{\sqrt{2^n}}\sum_{k=0}^{2^n-1}  e^{2\pi i (j/2^n) \cdot k} \ket{k}$$

如果将此表达式与上一步结束时第一个寄存器的状态进行比较，您会发现它们与 $\theta = j / {2^n}$ 完全匹配。

这意味着您可以使用傅里叶逆变换（傅里叶变换的伴随）将第一个寄存器的状态转换为整数的二进制表示法，该整数存储以二进制分数形式存储的 $\theta$ 的前 $n$ 位。
如果 $\theta = 0.\theta_1 \theta_2... \theta_n$，则将逆 QFT 应用于第一个寄存器将产生一个状态

$$\ket{j} = \ket{\theta_n} \otimes ... \otimes \ket{\theta_1}$$

（请记住，上一步假设第一个寄存器以小端表示法存储整数，因此您必须在此步骤中使用相同的表示法。）

### 步骤 4. 测量第一个寄存器

在最后一步中，测量第一个寄存器，将测量结果转换为整数，然后除以 $2^n$ 以获得特征相位 $\theta$。
