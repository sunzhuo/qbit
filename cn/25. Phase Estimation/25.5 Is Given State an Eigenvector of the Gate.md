# 给定状态是门的特征向量吗？

**输入：**

1. 单量子位酉 $U$。
2. 单量子位状态 $\ket{\psi}$，作为从 $\ket{0}$ 状态准备的单一 $P$ 给出。换句话说，将酉 $P$ 应用到状态 $\ket{0}$ 的结果是 $\ket{\psi}$ 状态：
   $$P\ket{0} = \ket{\psi}$$

**输出：** 如果给定状态是给定酉的本征态，则返回 true，否则返回 false。

<details>
  <summary><b>需要提示吗？</b></summary>

库操作 <code>CheckZero</code> 允许您检查给定量子位的状态是否为 $\ket{0}$。

</details>

## 解决方案注释

如果将该门应用于该状态（除了将其乘以全局相位之外）不会改变量子态，则该量子态是量子门的本征态。这意味着您的解决方案可能应该首先准备状态 $\ket{\psi}$ 并向其应用单一 $U$。如何检查之后的状态是否仍然是 $\ket{\psi}$（直到全局阶段）？

让我们考虑一下如果将 $P$ 的伴随应用于状态 $\ket{\psi}$ 会发生什么：

$$P^\dagger \ket{\psi} = P^\dagger P\ket{0} = I\ket{0} = \ket{0}$$

您可以使用它来完成解决方案：将 `Adjoint P` 应用于应用 $U$ 后获得的状态，并使用库操作 `CheckZero` 检查结果是否为 $\ket{0}$。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Diagnostics.CheckZero;
    operation IsEigenvector(U : Qubit => Unit, P : Qubit => Unit is Adj) : Bool {
        use q = Qubit();
        P(q);
        U(q);
        Adjoint P(q);
        let ret = CheckZero(q);
        Reset(q);
        return ret;
    }
}
```
