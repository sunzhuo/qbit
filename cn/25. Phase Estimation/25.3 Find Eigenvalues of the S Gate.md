# 求 S 门的特征值

**输入：** 无。

**输出：** 返回 $S$ 门的两个特征值的数组。

$$S = \begin{bmatrix} 1 & 0 \\ 0 & i \end{bmatrix}$$

按实部降序对特征值进行排序。

> 在此任务中，特征值表示为 Q# `Complex` 类型的复数。

## 解决方案注释

由于 $S$ 门是对角的，因此很容易意识到它的特征向量是基向量 $\ket{0}$ 和 $\ket{1}$。

要找到相应的特征值，需要求解两个方程：
- $S\ket{0} = \lambda \ket{0}$，它为您提供$\lambda = 1$。
- $S\ket{1} = \lambda \ket{1}$，它为您提供$\lambda = i$。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function EigenvaluesS() : Complex[] {
        return [
            Complex(1.0, 0.0),
            Complex(0.0, 1.0)
        ];
    }
}
```
