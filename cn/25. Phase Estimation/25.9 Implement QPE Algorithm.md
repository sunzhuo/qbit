# 实现QPE算法

**输入：**

1. 单量子位酉 $U$。统一体将定义其受控变体。
2. 单量子位状态 $\ket{\psi}$，作为从 $\ket{0}$ 状态准备的单一 $P$ 给出。换句话说，将酉 $P$ 应用到状态 $\ket{0}$ 的结果是 $\ket{\psi}$ 状态：
   $$P\ket{0} = \ket{\psi}$$
3. 正整数$n$。

**输出：** 返回对应于本征态 $\ket{\psi}$ 的本征相 $\theta$ 乘以 $2^n$。您可以保证该特征值最多具有 $n$ 位二进制精度，因此将其乘以 $2^n$ 会得到一个整数。

## 解决方案注释

要完成此练习，您需要遵循本课程中所述的 QPE 算法。

1. 分配两个量子位寄存器，$n$-量子位`phaseRegister`和单量子位`eigenstate`（为简单起见，这可以只是一个单独的量子位，而不是一个量子位数组）。
2. 准备算法的初始状态：使用酉 $P$ 的状态 $\ket{\psi}$ 中的 `eigenstate` 和使用 Hadamard 门的所有基本状态的偶数叠加状态中的 `phaseRegister`。
3. 使用两个 `for` 循环应用受控 $U$ 门。外部循环将迭代 `phaseRegister` 的量子位，从存储最低有效数字的第一个量子位到存储最高有效数字的最后一个量子位。内部循环将应用受控的 $U$ $2^k$ 次，其中 $k$ 是用作外部循环计数器的变量。
4. 应用伴随 QFT。在这里，您可以使用库操作 `ApplyQFT`，请记住它仅应用量子傅里叶变换的旋转部分。要使用此操作实现完整的转换，您需要在应用 `ApplyQFT` 后反转寄存器中量子位的顺序。由于这里需要使用伴随 QFT，因此需要在调用 `Adjoint ApplyQFT` 之前反转量子位的顺序。
5. 测量 `phaseRegister` 中的量子位并将结果转换为整数。 `MeasureInteger` 正是针对小尾数法的寄存器执行此操作。您还需要在释放用作本征态的量子位之前对其进行重置。

来源：Solution.qs

```qsharp
namespace Kata {
    operation PhaseEstimation(
        U : Qubit => Unit is Ctl, 
        P : Qubit => Unit,
        n : Int
    ) : Int {
        use (phaseRegister, eigenstate) = (Qubit[n], Qubit());
        P(eigenstate);
        ApplyToEach(H, phaseRegister);
        for k in 0 .. n - 1 {
            for _ in 1 .. 1 <<< k {
                Controlled U([phaseRegister[k]], eigenstate);
            }
        }
        SwapReverseRegister(phaseRegister);
        Adjoint ApplyQFT(phaseRegister);

        Reset(eigenstate);
        return MeasureInteger(phaseRegister);
    }
}
```
