# 量子比特交换

**输入：**

1. 处于任意状态 $\ket{\psi}$ 的 $N$ 个量子位，存储在长度为 $N$ 的数组中。
2. 整数 `index1` 和 `index2` 使得 $0 \le \text{index1} < \text{index2} \le N - 1$。

**目标：** 交换给定索引处的量子位的状态。

## 解决方案注释

$SWAP$ 门允许您交换任意两个量子位的状态。在本练习中，您需要将其应用于两个量子位，这两个量子位由量子位数组中的索引给出；您可以通过 `qs[index of a qubit]` 访问数组的各个量子位。

来源：Solution.qs

```qsharp
namespace Kata {
    operation QubitSwap (qs : Qubit[], index1 : Int, index2 : Int) : Unit is Adj + Ctl {
       SWAP(qs[index1], qs[index2]);
    }
}
```
