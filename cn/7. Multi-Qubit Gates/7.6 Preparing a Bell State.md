# 准备贝尔状态

**输入：** 状态为 $\ket{00}$ 的两个量子位，存储在长度为 2 的数组中。

**目标：** 将系统转变为 Bell 状态 $\ket{\Phi^+} = \frac{1}{\sqrt{2}}\big(\ket{00} + \ket{11}\big)$。

## 解决方案注释

您之前在多量子位系统 kata 中已经看到过这种状态，您在其中确定了这种状态是不可分离的，也就是说，不能仅使用单量子位门来准备它。为了准备它，您需要使用$CNOT$门。

让我们看看 $CNOT$ 门对教程中描述的可分离状态的影响：
$$CNOT_{1,2}\big(\alpha\ket{0} + \beta\ket{1}\big) \otimes \ket{0} = CNOT_{1,2}(\alpha\ket{00} + \beta\ket{10}) = \alpha\ket{00} + \beta\ket{11}$$

这个结果状态正是您需要准备的状态，$\alpha = \beta = \frac{1}{\sqrt{2}}$！

该解决方案需要两个步骤：
1. 准备状态 $\big(\frac{1}{\sqrt{2}}\ket{0} + \frac{1}{\sqrt{2}}\ket{1}\big) \otimes \ket{0}$。
您可以使用 Hadamard 门来执行此操作。
2. 应用 $CNOT$ 门，将第一个量子位作为对照，将第二个量子位作为目标。

来源：Solution.qs

```qsharp
namespace Kata {
    operation BellState (qs : Qubit[]) : Unit is Adj + Ctl {
        H(qs[0]);
        CNOT(qs[0], qs[1]);
    }
}
```
