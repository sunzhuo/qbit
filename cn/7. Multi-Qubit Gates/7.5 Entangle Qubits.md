# 纠缠量子位

**输入：** 两个未纠缠的量子位（存储在长度为 2 的数组中）。
第一个量子位将处于状态 $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$，第二个量子位将处于状态 $\ket{0}$
（这可以写成两个量子位状态$\big(\alpha \ket{0} + \beta \ket{1} \big) \otimes \ket{0} = \alpha \ket{00} + \beta \ket{10}$）。

**目标：** 将两个量子位状态更改为 $\alpha \ket{00} + \beta \ket{11}$。

## 解决方案注释

我们将状态 $\alpha \ket{0} + \beta \ket{1}$ 中的第一个量子位表示为 A，将状态 $\ket{0}$ 中的第二个量子位表示为 B。

将输入状态 $\alpha \ket{0_A0_B} + \beta \ket{1_A0_B}$ 与目标状态 $\alpha \ket{0_A0_B} + \beta \ket{1_A1_B}$ 进行比较。 
您希望将我们的输入量子位通过执行以下操作的一个或多个门（待定）。如果量子位 A 处于 $\ket{0}$ 状态，那么您希望单独保留量子位 B（叠加的第一项）。 
但是，如果 A 处于 $\ket{1}$ 状态，您希望将量子位 B 从 $\ket{0}$ 翻转到 $\ket{1}$ 状态。换句话说，B 的状态取决于 A 的状态。 
这正是 $CNOT$ 门的效果。根据**控制**量子位（本例中为 A）的状态，受控或**目标**量子位（本例中为 B）的值会反转或不变。因此，您将获得目标状态 $\alpha \ket{00} + \beta \ket{11}$。

```qsharp
namespace Kata {
    operation EntangleQubits (qs : Qubit[]) : Unit is Adj + Ctl {
        CNOT(qs[0], qs[1]);
    }
}
```
