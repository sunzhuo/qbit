# 任意控制

**输入：**

1. `controls` - 处于任意状态 $\ket{\phi}$ 的 $N$ 个量子位的寄存器。
2. `target` - 处于任意状态 $\ket{\psi}$ 的量子位。
3. `controlBits` - $N$ 布尔值数组，指定每个控制量子位应处于什么状态才能应用门。

**目标：** 应用受控 $X$ 门，以 `controls` 作为控制量子位，以 `target` 作为目标，以 `controlBits` 指定的状态作为控制。如果数组的元素是`true`，则对应的量子位是常规控制（应处于状态$\ket{1}$），如果是`false`，则对应的量子位是反控制（应处于状态$\ket{0}$）。

> 例如，如果 `controlBits = [true, false, true]`，则仅当控制量子位处于状态 $\ket{101}$ 时才应应用受控 $X$ 门。

<details>
    <summary><strong>需要提示吗？</strong></summary>
    <p>考虑使用库操作来执行此任务。如果您想在不进行库操作的情况下完成此操作，请不要忘记将量子位重置回其原始状态。</p>
</details>

## 解决方案注释

您需要在 `target` 量子位上执行 $X$ 门，该门由 `controls` 量子位的状态控制；该状态应对应于 `controlBits` 给出的掩码。

如果 `controlBits` 掩码由所有 `true` 值组成，您可以使用熟悉的 `Controlled X` 门。如果掩码中有一些 `false` 值，您可以做什么？

事实证明，您可以根据 `controlBits` 的相应元素来转换控制量子位的状态：如果该元素是 `false`，则将 $X$ 门应用于 `controls` 数组中的相应量子位。此后，`Controlled X` 门将在您想要的确切情况下应用 $X$ 门。
最后，您需要记住撤消（“取消计算”）第一步，否则您的受控门将影响控制量子位的状态以及目标的状态。

正如您在下面的第一个单元格中看到的，这可能需要相当多的编码。

来源：SolutionA.qs

```qsharp
namespace Kata {
    operation MultiControls (controls : Qubit[], target : Qubit, controlBits : Bool[]) : Unit is Adj + Ctl {
        for index in 0 .. Length(controls) - 1 {
            if controlBits[index] == false {
                X(controls[index]);       
            }
        }
     
        Controlled X(controls,target);
    
        for index in 0 .. Length(controls) - 1 {
            if controlBits[index] == false {
                X(controls[index]);       
            }
        }
    }
}
```

您可以使用 `within ... apply` 构造来稍微缩短代码，该构造负责自动取消计算第一个代码块中完成的步骤：

来源：SolutionB.qs

```qsharp
namespace Kata {
    operation MultiControls (controls : Qubit[], target : Qubit, controlBits : Bool[]) : Unit is Adj + Ctl {
        within {
            for index in 0 .. Length(controls) - 1 {
                if controlBits[index] == false {
                    X(controls[index]);       
                }
            }
        } apply {
            Controlled X(controls,target);
        }
    }
}
```

最后，以下是如何使用库函数 `ApplyControlledOnBitString` 实现完全相同的任务。

来源：SolutionC.qs

```qsharp
namespace Kata {
    operation MultiControls (controls : Qubit[], target : Qubit, controlBits : Bool[]) : Unit is Adj + Ctl {
        ApplyControlledOnBitString(controlBits, X, controls, target);
    }
}
```
