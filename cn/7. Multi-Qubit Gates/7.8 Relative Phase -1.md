# 相对相位-1

**输入：** 处于状态 $\ket{+} \otimes \ket{+} = \frac{1}{2} \big( \ket{00} + \ket{01} + \ket{10} {\color{blue}+} \ket{11} \big)$ 的两个未纠缠的量子位（存储在长度为 2 的数组中）。

**目标：** 将两个量子位状态更改为 $\frac{1}{2} \big( \ket{00} + \ket{01} + \ket{10} {\color{red}-} \ket{11} \big)$。

## 解决方案注释

首先，请注意您正在处理一对未纠缠的量子位。
以向量形式，您需要的转换是
$$
\frac{1}{2}\begin{bmatrix}1 \\ 1 \\ 1 \\ 1 \end{bmatrix}
\rightarrow
\frac{1}{2}\begin{bmatrix}1 \\ 1 \\ 1 \\ -1 \end{bmatrix}
$$

要改变目标的输入，只需将 $\ket{11}$ 基本状态的符号翻转即可。

请记住，泡利 Z 门在单量子位情况下翻转符号，并且 $CZ$ 是该门的 2 量子位版本。事实上，$CZ$ 门的效果正是您在这里寻找的转换。

来源：SolutionA.qs

```qsharp
namespace Kata {
    operation RelativePhaseMinusOne (qs : Qubit[]) : Unit is Adj + Ctl {
        CZ(qs[0], qs[1]);
    }
}
```

或者，您可以使用内在 $Z$ 门及其受控变体（使用 Controlled functor）来表达此门：

来源：SolutionB.qs

```qsharp
namespace Kata {
    operation RelativePhaseMinusOne (qs : Qubit[]) : Unit is Adj + Ctl {
        Controlled Z([qs[0]], qs[1]);
    }
}
```
