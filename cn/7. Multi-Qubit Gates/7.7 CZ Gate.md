# CZ 门

$CZ$（“受控 Z”）门是一个双量子位门，其中一个量子位称为 **控制** 量子位，另一个称为 **目标** 量子位。有趣的是，对于 $CZ$ 门来说，哪个量子位是控制哪个是目标并不重要 - 无论哪种方式，门的效果都是相同的！

$CZ$ 门充当条件门：如果控制量子位处于状态 $\ket{1}$，它将 $Z$ 门应用于目标量子位，否则不执行任何操作。

$CZ$ 门对于创建和操纵纠缠态特别有用，其中量子态的相位至关重要。考虑以下可分离状态：

$$\big(\alpha\ket{0} + \beta\ket{1}\big) \otimes \big(\gamma\ket{0} + \delta\ket{1}\big) = \alpha\gamma\ket{00} + \alpha\delta\ket{01} + \beta\gamma\ket{10} + \beta\delta\ket{11}$$

如果将 $CZ$ 门应用于它，将第一个量子位作为控制，第二个量子位作为目标（反之亦然），您将得到以下状态，该状态无法再分离：

$$\alpha\gamma\ket{00} + \alpha\delta\ket{01} + \beta\gamma\ket{10} - \beta\delta\ket{11}$$

$CZ$ 门也是自伴的：第二次应用它会反转其效果，类似于 $CNOT$ 门。
