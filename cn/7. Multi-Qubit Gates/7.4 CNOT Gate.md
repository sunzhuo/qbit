# CNOT 门

此 kata 引入的第一个多量子位门是 $CNOT$（“受控非”）门。 $CNOT$ 门是一个双量子位门，其中一个量子位称为 **控制** 量子位，另一个量子位称为 **目标** 量子位（通常第一个量子位是控制，第二个量子位是目标）。

$CNOT$ 充当某种条件门：如果控制量子位处于状态 $\ket{1}$，它将 $X$ 门应用于目标量子位，否则不执行任何操作。

> 如果系统处于多个基本状态的叠加状态，则门的效果将是其单独作用于每个基本状态的效果的线性组合。
> 这将是您稍后遇到的所有以基本状态指定的量子门的情况：由于所有酉门都是线性的，因此足以定义它们对基本状态的影响，并使用线性来计算它们对任何状态的影响。

$CNOT$ 门对于准备纠缠态特别有用。考虑以下可分离状态：

$$\big(\alpha\ket{0} + \beta\ket{1}\big) \otimes \ket{0} = \alpha\ket{00} + \beta\ket{10}$$

如果将 $CNOT$ 门应用于它，以第一个量子位作为控制，第二个量子位作为目标，您将得到以下状态，该状态不再可分离：

$$\alpha\ket{00} + \beta\ket{11}$$

$CNOT$ 门是自伴的：第二次应用它会反转其效果。
