# 获胜条件

**输入：**

1. Alice、Bob 和 Charlie 的输入位（r、s 和 t），存储为长度为 3 的数组。
   输入位将有零个或两个位设置为 true。
2. Alice、Bob 和 Charlie 的输出位（a、b 和 c），存储为长度为 3 的数组。

**目标：**

如果 Alice、Bob 和 Charlie 赢得 GHZ 游戏，即 R $\lor$ S $\lor$ T = A $\oplus$ B $\oplus$ C，则为 true，否则为 false。

## 解决方案注释

有四个可能的输入：(0,0,0)、(0,1,1)、(1,0,1) 和 (1,1,0)，每个输入都有 $25\%$ 概率。
因此，为了获胜，如果输入为 (0,0,0)，则输出位的总和必须为偶数，否则为奇数。

要检查获胜条件是否成立，您需要计算表达式 R $\lor$ S $\lor$ T 和 A $\oplus$ B $\oplus$ C 并比较它们：
如果相等，则游戏获胜。要计算表达式，您可以使用[内置运算符](https://learn.microsoft.com/azure/quantum/user-guide/language/expressions/logicalexpressions)：
X $\lor$ Y 为 `x or y`，A $\oplus$ B 为 `a != b`。

来源：Solution.qs

```qsharp
namespace Kata {
    function WinCondition (rst : Bool[], abc : Bool[]) : Bool {
        return (rst[0] or rst[1] or rst[2]) == (abc[0] != abc[1] != abc[2]);
    }
}
```
