# 经典 GHZ 游戏

**输入：**

1. 实现玩家经典策略的三个操作的数组（即获取输入位并产生输出位），
2. 应传递给玩家的 3 个输入位的数组。

**输出：**

如果每个玩家使用他们给定的策略，将产生一个由三位组成的数组。

## 解决方案注释

您会获得输入位和每个玩家正在使用的策略，因此您只需将它们转换为输出位并返回它们即可。

来源：Solution.qs

```qsharp
namespace Kata {
    operation PlayClassicalGHZ (strategies : (Bool => Bool)[], inputs : Bool[]) : Bool[] {
        let r = inputs[0];
        let s = inputs[1];
        let t = inputs[2];
        let a = strategies[0](r);
        let b = strategies[1](s);
        let c = strategies[2](t);
        return [a, b, c];
    }
}
```
