# 爱丽丝和鲍勃的经典策略

在此任务中，您必须实现两个函数，一个用于 Alice 的经典策略，另一个用于 Bob 的策略。 
请注意，一项测试涵盖了它们，因此您必须同时实现这两项测试才能通过测试。

**输入：** 
Alice 和 Bob 的起始位（X 和 Y）。

**输出：**
  Alice 和 Bob 输出位（A 和 B）以最大化他们获胜的机会。

## 解决方案注释

如果 Alice 和 Bob 始终返回 TRUE，则他们将具有 $75\%$ 获胜率，因为 TRUE $\oplus$ TRUE == FALSE，并且其输入位上的 AND 运算将以 $75\%$ 概率为 FALSE。
或者，Alice 和 Bob 可以同意始终返回 FALSE 以实现相同的 $75\%$ 获胜概率。经典策略无法获得更高的成功概率。

来源：Solution.qs

```qsharp
namespace Kata {
    function AliceClassical (x : Bool) : Bool {
        return false;
    }

    function BobClassical (y : Bool) : Bool {
        return false;
    }
}
```
