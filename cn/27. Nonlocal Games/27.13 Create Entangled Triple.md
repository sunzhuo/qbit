# 创建纠缠三元组

在量子版本的游戏中，玩家在游戏过程中仍然无法交流，但可以分享
比赛开始前来自纠缠三元组的量子位。

**输入：**
处于 $\ket{000}$ 状态的三个量子位的数组。

**目标：**
在这些量子位上创建纠缠态 $\ket{\Phi} = \frac{1}{2} \big(\ket{000} - \ket{011} - \ket{101} - \ket{110} \big)$。

>该状态相当于三量子位[GHZ状态](https://en.wikipedia.org/wiki/Greenberger%E2%80%93Horne%E2%80%93Zeilinger_state)
>$$\frac{1}{\sqrt{2}} \big(\ket{000} + \ket{111} \big)$$
> 直至本地统一运营。详情请参阅后续GHZ量子策略讨论。

## 解决方案注释

1. 对第一个和第二个量子位应用 X 门以获得 $\ket{110}$ 状态。
2. 对第一个和第二个量子位应用 H 门以获得以下状态：
$$\frac12 \big( \ket{000} - \ket{010} - \ket{100} + \ket{110} \big)$$
3. 使用受控 Z 门翻转最后一项的符号，其中第一个量子位作为控制，第二个量子位作为目标（反之亦然）：
$$\frac12 \big( \ket{000} - \ket{010} - \ket{100} -{\color{blue}\ket{110}} \big)$$
4. 现在我们有了每个项的正确符号，并且第一个和最后一个项与我们正在准备的状态相匹配，因此我们只需要调整中间的两个项。
为此，如果前两个量子位处于 $\ket{01}$ 或 $\ket{10}$ 状态，我们可以使用 [ControlledOnBitString](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.canon/applycontrolledonbitstring) 操作来翻转最后一个量子位的状态，这给我们：
$$\frac{1}{2} \big(\ket{000} - {\color{blue}\ket{011}} - {\color{blue}\ket{101}} - \ket{110} \big)$$

来源：Solution.qs

```qsharp
namespace Kata {
    operation CreateEntangledTriple (qs : Qubit[]) : Unit is Adj {
        X(qs[0]);
        X(qs[1]);
        H(qs[0]);
        H(qs[1]);
        Controlled Z([qs[0]], qs[1]);
        ApplyControlledOnBitString([false, true], X, [qs[0], qs[1]], qs[2]);
        ApplyControlledOnBitString([true, false], X, [qs[0], qs[1]], qs[2]);
    }
}
```
