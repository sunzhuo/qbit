# 获胜条件

**输入：**

- Alice 和 Bob 的起始位（X 和 Y）。
- Alice 和 Bob 的输出位（A 和 B）。

**输出：**
  如果 Alice 和 Bob 赢得了 CHSH 游戏，即 X $\land$ Y = A $\oplus$ B，则为 true，否则为 false。

## 解决方案注释

有四个可能的输入对 (X, Y)：(0,0)、(0,1)、(1,0) 和 (1,1)，每个输入对都有 $25\%$ 概率。
为了获胜，如果输入是 (1,1)，Alice 和 Bob 必须输出不同的位，否则必须输出相同的位。

要检查获胜条件是否成立，您需要计算 X $\land$ Y 和 A $\oplus$ B 并比较这些值：
如果相等，则爱丽丝和鲍勃获胜。您可以使用以下方法计算这些值
[内置运算符](https://learn.microsoft.com/azure/quantum/user-guide/language/expressions/logicalexpressions)：
X $\land$ Y 为 `x and y`，A $\oplus$ B 为 `a != b`。

来源：Solution.qs

```qsharp
namespace Kata {
    function WinCondition (x : Bool, y : Bool, a : Bool, b : Bool) : Bool {
        return (x and y) == (a != b);
    }
}
```
