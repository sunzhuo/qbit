# 经典策略

在此任务中，您必须实现三个函数，每个函数对应每个玩家的经典策略。
请注意，一项测试涵盖了它们，因此您必须实现所有这些才能通过测试。
在每个函数中，输入是相应玩家的起始位，并且应该返回该玩家选择的输出位。

**输入：**

Alice、Bob 和 Charlie 的起始位（R、S 和 T）。

**输出：**

Alice、Bob 和 Charlie 的输出位（A、B 和 C）以最大化他们获胜的机会。

## 解决方案注释

如果所有三个玩家都返回 TRUE，则 A $\oplus$ B $\oplus$ C == TRUE 必然（因为它们的位之和为奇数）。
这将战胜输入 011、101 和 110，而输给 000。
另一种解决方案是一个玩家返回 TRUE，另外两个玩家返回 FALSE。

由于上述四个输入具有相等的概率，并且代表所有可能的输入，
这些确定性策略中的任何一个都会以 $75\%$ 概率获胜。

来源：Solution.qs

```qsharp
namespace Kata {
    operation AliceClassical (r : Bool) : Bool {
        return true;
    }

    operation BobClassical (s : Bool) : Bool {
        return true;
    }

    operation CharlieClassical (t : Bool) : Bool {
        return true;
    }
}
```
