# 量子策略

在此任务中，您应该实现三个函数，每个函数对应每个玩家的量子策略。
请注意，一项测试涵盖了它们，因此您必须实现所有这些才能通过测试。

**输入：**

1. 每个玩家之一的输入位（分别为 R、S 和 T），
2. 该玩家的纠缠三元组的量子位在玩家之间共享。

**目标：**
如果该位为 0 (FALSE)，则在 Z 基础上测量该量子位；如果该位为 1 (TRUE)，则在 X 基础上测量该量子位，并返回结果。
操作后量子位的状态并不重要。

## 解决方案注释

在 Q# 中，您可以使用以下任一方法在特定基础上执行测量
[测量操作](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.intrinsic/measure)
或测量并重置为 $\ket{0}$ 操作序列的便捷简写
[MResetZ](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.measurement/mresetz) 和
[MResetX](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.measurement/mresetx)。

或者，您可以回忆一下，在 X 基础上测量量子位相当于对其应用 H 门并在 Z 基础上测量它。

来源：Solution.qs

```qsharp
namespace Kata {
    operation AliceQuantum (bit : Bool, qubit : Qubit) : Bool {
        if bit {
            return MResetX(qubit) == One;
        }
        return MResetZ(qubit) == One;
    }

    operation BobQuantum (bit : Bool, qubit : Qubit) : Bool {
        if bit {
            return MResetX(qubit) == One;
        }
        return MResetZ(qubit) == One;
    }

    // alternative implementation
    operation CharlieQuantum (bit : Bool, qubit : Qubit) : Bool {
        if bit {
            H(qubit);
        }    
        return M(qubit) == One;
    }
}
```
