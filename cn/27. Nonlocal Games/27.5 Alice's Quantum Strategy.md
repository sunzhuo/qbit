# 爱丽丝的量子策略

在量子版本的游戏中，玩家在游戏过程中仍然无法交流，
但他们可以在比赛开始前共享贝尔对的量子位。

**输入：**

- 爱丽丝的起始位（X）。
- Alice 与 Bob 共享贝尔配对的一半。

**目标：**
  如果 Alice 的位为 0（假），则在 Z 基础上测量 Alice 的量子位；如果她的位为 1（真），则在 X 基础上测量 Alice 的量子位；
  以布尔值形式返回测量结果：将 `Zero` 映射为 false，将 `One` 映射为 true。
  操作后量子位的状态并不重要。

## 解决方案注释

在 Q# 中，您可以使用以下任一方法在特定基础上执行测量
[测量操作](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.intrinsic/measure)
或测量并重置为 $\ket{0}$ 操作序列的便捷简写
[MResetZ](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.measurement/mresetz) 和
[MResetX](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.measurement/mresetx)。

（有关爱丽丝为何应遵循此策略的详细信息，请参阅下面的课程。）

来源：Solution.qs

```qsharp
namespace Kata {
    operation AliceQuantum (bit : Bool, qubit : Qubit) : Bool {
        if bit {
            let q = MResetX(qubit);
            return q == One;
        }
        else {
            let q = MResetZ(qubit);
            return q == One;
        }
    }
}
```
