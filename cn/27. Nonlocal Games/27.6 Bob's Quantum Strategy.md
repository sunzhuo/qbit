# 鲍勃的量子策略

**输入：**

- Bob 的起始位 (Y)。
- 鲍勃与爱丽丝共享贝尔配对的一半。

**目标：**
如果 Bob 的位为 0（假），则在 $\frac{\pi}{8}$ 基础上测量 Bob 的量子位，或者在 $-\frac{\pi}{8}$ 基础上测量 Bob 的量子位
如果他的位为 1（真）并将测量结果作为布尔值返回：将 `Zero` 映射到 false，将 `One` 映射到 true。
操作后量子位的状态并不重要。

在 $\theta$ 基础上测量量子位与将量子位顺时针旋转 $\theta$，然后在 Z 基础上进行标准测量相同。

## 解决方案注释

可以使用合适的 $R_y$ 旋转从计算基础 $\{ \ket{0}, \ket{1} \}$ 转到 $\{ \ket{\psi_+}, \ket{\psi_-} \}$ 基础，反之亦然。

要在 Q# 中实现所描述的转换，我们需要将量子位顺时针旋转 $\frac{\pi}{8}$（如果 `bit == false`）或逆时针旋转 `bit == true`，然后执行测量。
我们可以使用 $R_y$ 门进行旋转（注意我们需要执行的布尔参数的否定）。

（有关为什么鲍勃应该遵循此策略的详细信息，请参阅下面的课程。）

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    operation BobQuantum(bit : Bool, qubit : Qubit) : Bool {
        let angle = 2.0 * PI() / 8.0;
        Ry(not bit ? -angle | angle, qubit);
        return M(qubit) == One;
    }
}
```
