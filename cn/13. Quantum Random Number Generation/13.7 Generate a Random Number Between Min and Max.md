# 生成一个介于最小值和最大值之间的随机数

在“生成任意大小的数字”练习中，您生成了 $[0, 2^N-1]$ $(1 \leq N \leq 10)$ 范围内的数字。现在让我们创建一个返回 $[min, max]$ 范围内的随机数的操作。

**输入：** 
两个整数 $min$ 和 $max$ ($0 \leq min \leq max \leq 2^{10}-1$)。

**目标：** 生成 $[min, max]$ 范围内的随机数，并以相同的概率获得该范围内的每个数字。

> Q# 命名空间 `Std.Math` 包含有用的函数 `BitSizeI`，用于计算给定数字的二进制表示形式的位数。

## 解决方案注释

您可以重复使用之前练习中的 `RandomBit` 和 `RandomNBits` 操作。

您将通过调用 `RandomNBits` 操作生成 $N$ 位随机数，其中 $N$ 是 $max - min$ 的位大小。您可以重复此过程，直到结果小于或等于 $max - min$，然后返回该数字加上 $min$。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    operation RandomNumberInRange(min : Int, max : Int) : Int {
        let nBits = BitSizeI(max - min);
        mutable output = 0;
        repeat {
            set output = RandomNBits(nBits);
        } until output <= max - min;
        return output + min;
    }

    operation RandomNBits(N : Int) : Int {
        mutable result = 0;
        for i in 0..N - 1 {
            set result = result * 2 + RandomBit();
        }
        return result;
    }

    operation RandomBit() : Int {
        use q = Qubit();
        H(q);
        return MResetZ(q) == Zero ? 0 | 1;
    }
}
```
