# 生成一个随机的两位数

现在您可以生成单个随机位，您可以使用该逻辑来创建随机多位数字。我们首先尝试通过组合两个随机生成的位来生成一个两位数。

**输入：** 无。

**目标：** 生成 $[0, 3]$ 范围内的随机数，并以相同的概率获得四个数字中的每一个。

**扩展目标：** 您可以在不在此操作中分配量子位的情况下完成此操作吗？
<details>
  <summary><b>需要提示吗？</b></summary>
  请记住，您可以使用先前定义的操作来实施您的解决方案。为方便起见，<b>RandomBit</b> 操作已可供您在本练习中使用。
</details>

## 解决方案注释

让我们重用“生成单个随机位”练习中的 `RandomBit` 操作。
您可以通过调用 `RandomBit` 操作两次来生成两个随机位，将最高有效位乘以 2，然后添加最低有效位以生成随机的两位数。

来源：Solution.qs

```qsharp
namespace Kata {
    operation RandomTwoBits() : Int {
        return 2 * RandomBit() + RandomBit();
    }

    operation RandomBit() : Int {
        use q = Qubit();
        H(q);
        return MResetZ(q) == Zero ? 0 | 1;
    }
}
```
