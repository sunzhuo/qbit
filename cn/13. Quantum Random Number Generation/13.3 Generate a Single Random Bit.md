# 生成单个随机位

**输入：** 无。

**目标：** 以相等的概率生成 $0$ 或 $1$。
<details>
  <summary><b>需要提示吗？</b></summary>
  使用分配的量子位，对其应用量子门，测量它并使用结果返回 $0$ 或 $1$。
</details>

**延伸目标：** 你能找到不同的方式来实现这个操作吗？
<details>
  <summary><b>需要提示吗？</b></summary>
  以相同概率产生 $0$ 和 $1$ 测量结果的不同量子态是什么？以不同的基础测量量子位会如何改变结果？
</details>

## 解决方案注释

单个量子位的状态可以表示为二维列向量$\begin{bmatrix} \alpha \\ \beta \end{bmatrix}$，其中$\alpha$和$\beta$是满足$|\alpha|^2 + |\beta|^2 = 1$的复数。当您测量量子位时，您会以 $|\alpha|^2$ 的概率得到 0，或以 $|\beta|^2$ 的概率得到 1。本质上，您可以通过设置正确的基态幅度来控制测量结果的概率。

在 Q# 中分配量子位时，幅度 $\alpha$ 和 $\beta$ 分别为 1 和 0。现在您的目标是为 $\alpha$ 和 $\beta$ 设置相等的幅度以实现绝对随机性。您可以通过简单地将 Hadamard 门应用到初始状态 $\ket{0}$ 来实现这一点：

$$
H\ket{0} =
\frac{1}{\sqrt{2}}
\begin{bmatrix} 1 & 1 \\ 1 & -1 \end{bmatrix}
\begin{bmatrix} 1 \\ 0 \end{bmatrix} =
\frac{1}{\sqrt{2}}
\begin{bmatrix} 1 \cdot 1 + 1 \cdot 0 \\ 1 \cdot 1 + (-1) \cdot 0 \end{bmatrix} =
\frac{1}{\sqrt{2}}
\begin{bmatrix} 1 \\ 1 \end{bmatrix}
$$

现在，0 和 1 测量结果出现的概率均相等 $|\frac{1}{\sqrt{2}}|^2 = \frac{1}{2}$。

> 由于结果概率是幅度绝对值的平方，因此通过测量 $\ket{-}$ 状态，您将获得相同的结果分布，您可以通过将 Hadamard 门应用于基础状态 $\ket{1}$ 来准备。尝试一下以实现伸展目标！

```qsharp
namespace Kata {
    operation RandomBit() : Int {
        // Allocate single qubit.
        use q = Qubit();

        // Convert the qubit state to |+>.
        H(q);

        // Measure the qubit and reset it to |0>.
        let result = MResetZ(q);

        // Return integer value of result.
        if result == One {
            return 1;
        }
        return 0;
    }
}
```
