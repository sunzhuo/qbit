# 生成一个加权位

在上述每个练习中，所有生成的数字都有相同的可能性。现在让我们创建一个操作，该操作将返回具有不同结果概率的随机位。

> 请记住，通过设置基态 $\alpha$ 和 $\beta$ 的幅度，您可以控制测量量子位时获得测量结果 $0$ 和 $1$ 的概率。

**输入：** 
浮点数 $x$、$0 \le x \le 1$。

**目标：** 生成 $0$ 或 $1$，$0$ 的概率等于 $x$，$1$ 的概率等于 $1 - x$。

> Q# 命名空间 `Std.Math` 包括有用的函数 `Sqrt`、`ArcCos` 和 `ArcSin`。

## 解决方案注释

任意单量子位状态可以写为：

$$
\ket{\psi} =
    \cos \frac{\theta}{2} \ket{0 } + e^{i\phi}  \sin \frac{\theta}{2} \ket{1}
$$

这里，$\theta$是状态向量和$Z$轴之间的角度，$\phi$是布洛赫球上相对于$X$轴的经度角。

您的目标是生成 $0$ 或 $1$，生成 $0$ 的概率等于 $x$，生成 $1$ 的概率等于 $1 - x$。这意味着量子位状态应如下所示：

$$
\ket{\psi} =
    \sqrt x \ket{0 } + \sqrt{1 - x} \ket{1}
$$

比较两个方程中状态 $\ket{0 }$ 的幅度，您可以得到

$$
\sqrt x = \cos \frac{\theta}{2} \Rightarrow \theta = 2 \arccos\sqrt x
$$

由于 $\theta$ 是状态向量与 $Z$ 轴之间的角度，因此您需要将 $Ry$ 门与计算出的 $\theta$ 应用于起始状态 $\ket{0 }$ 以获得所需的量子位状态。

$Ry$ 门围绕 $Y$ 轴（即 $ZX$ 平面）应用给定旋转。因此，$\phi$（相对于$X$轴的经度角）始终等于$0^{\circ}$，这意味着相对相位$e^{i\phi}$对生成的量子位状态没有任何影响。

> 您还可以通过比较两个方程中状态$\ket{1 }$的幅度来计算${\theta}$，即$2 \arcsin\sqrt{1.0 - x}$。结果将是等效的。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    operation WeightedRandomBit(x : Double) : Int {
        let theta = 2.0 * ArcCos(Sqrt(x));  // or 2.0 * ArcSin(Sqrt(1.0 - x));

        use q = Qubit();
        Ry(theta, q);

        let result = MResetZ(q);
        return result == Zero ? 0 | 1;
    }
}
```
