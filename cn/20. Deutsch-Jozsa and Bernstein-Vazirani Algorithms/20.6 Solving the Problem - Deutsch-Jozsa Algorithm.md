# 解决问题：Deutsch-Jozsa 算法

现在让我们回到确定给定函数是恒定函数还是平衡函数的问题。
以下各节详细介绍了该算法的步骤。

### 输入

您将获得预言机输入 $N$ 和预言机本身的位数 - 实现经典函数 $f(x)$ 的“黑匣子”操作 $U_f$。您可以保证预言机实现的功能是恒定的或平衡的。

### 起始状态

该算法从处于 $\ket{0...0} = \ket{0}^{\otimes N}$ 状态的 $N$ 量子位开始。

### 步骤 1. 对每个量子位应用 Hadamard 变换

将 $H$ 门应用于处于 $\ket{0}$ 状态的一个量子位，会将其转换为 $\frac{1}{\sqrt2} \big(\ket{0} + \ket{1} \big)$ 状态，这是一个量子位上两个基本状态的相等叠加。

如果将 $H$ 门应用于 $\ket{00}$ 状态下的两个量子位中的每一个，您将得到

$$(H \otimes H) \ket{00} = \big(H \ket{0} \big) \otimes \big(H \ket{0}\big) = \left(\frac{1}{\sqrt2} \big(\ket{0} + \ket{1} \big)\right) \otimes \left(\frac{1}{\sqrt2} \big(\ket{0} + \ket{1} \big)\right) = \frac{1}{2} \big(\ket{00} + \ket{01} + \ket{10} + \ket{11} \big)$$

这只是两个量子位上所有基础状态的相等叠加！ 
您可以将相同的想法扩展到将 $H$ 门应用到 $\ket{0...0}$ 状态中的每个 $N$ 量子位，得出这样的结论：这会将它们转换为 $N$ 量子位上所有基本状态的相等叠加状态。

从数学上讲，“将 $H$ 门应用于每个 $N$ 量子位”的转换可以表示为 $H^{\otimes N}$。应用此转换后，您将获得以下状态：

$$H^{\otimes N} \ket{0}^{\otimes N} = \big( H\ket{0} \big)^{\otimes N} = \left( \frac{1}{\sqrt2} \big(\ket{0} + \ket{1} \big) \right)^{\otimes N} = \frac{1}{\sqrt{2^N}} \sum_{x=0}^{2^N-1} \ket{x}$$

### 步骤 2. 应用预言

此步骤是您使用经典函数知识（作为量子预言给出）的唯一步骤。 
该步骤保持$f(x) = 0$的基态幅度不变，并将$f(x) = 1$的基态幅度乘以$-1$。

从数学上来说，oracle应用的结果可以写成如下：

$$U_f \left(\frac{1}{\sqrt{2^N}} \sum_{x=0}^{2^N-1} \ket{x} \right) = \frac{1}{\sqrt{2^N}} \sum_{x=0}^{2^N-1} U_f\ket{x} = \frac{1}{\sqrt{2^N}} \sum_{x=0}^{2^N-1} (-1)^{f(x)} \ket{x}$$

### 步骤 3. 再次对每个量子位应用 Hadamard 变换

在此步骤中，在将 $H$ 门应用于量子位之后，您无需担心量子位状态的整个表达式；仅计算基本状态 $\ket{0}^{\otimes N}$ 的结果幅度就足够了。

考虑表达式 $\sum_{x=0}^{2^N-1} (-1)^{f(x)} \ket{x}$ 中的基础状态 $\ket{x}$ 之一。  
它可以写为 $\ket{x} = \ket{x_{0} } \otimes \cdots \otimes \ket{x_{N-1}}$，其中每个 $\ket{x_k}$ 是 $\ket{0}$ 或 $\ket{1}$。  
当您将 $H$ 门应用于 $\ket{x}$ 时，我们将得到 $H^{\otimes N} \ket{x} = H\ket{x_{0} } \otimes \cdots \otimes H\ket{x_{N-1}}$，其中张量积的每一项都是 $H\ket{0} = \frac{1}{\sqrt2}\big(\ket{0} + \ket{1} \big) = \ket{+}$ 或 $H\ket{1} = \frac{1}{\sqrt2}\big(\ket{0} - \ket{1} \big) = \ket{-}$。 
如果打开此张量积中的括号，您将得到所有 $N$-量子位基础状态的叠加，每个状态都具有幅度 $\frac{1}{\sqrt{2^N}}$ 或 $-\frac{1}{\sqrt{2^N}}$ — 并且，由于 $\ket{+}$ 和 $\ket{-}$ 中 $\ket{0}$ 状态的幅度均为正，因此您知道幅度基本状态 $\ket{0}^{\otimes N}$ 的结果为正，即 $\frac{1}{\sqrt{2^N}}$。

现在您可以在表达式 $H^{\otimes N} \left( \frac{1}{\sqrt{2^N}} \sum_{x=0}^{2^N-1} (-1)^{f(x)} \ket{x} \right)$ 中计算 $\ket{0}^{\otimes N}$ 状态的幅度：在总和的每个 $2^N$ 项中，其幅度为 $\frac{1}{\sqrt{2^N}}$。因此，您可以得到总振幅

$$\frac{1}{\sqrt{2^N}} \sum_{x=0}^{2^N-1} (-1)^{f(x)} \frac{1}{\sqrt{2^N}} = \frac{1}{2^N} \sum_{x=0}^{2^N-1} (-1)^{f(x)}$$

### 步骤 4. 执行测量并解释结果

到目前为止，您还没有利用给定的函数是恒定或平衡的这一事实。让我们看看这如何影响 $\ket{0}^{\otimes N}$ 状态的幅度。

* 如果函数是常数 $f(x) = C$（始终为 $0$ 或始终 $1$），则得到
  $$\frac{1}{2^N} \sum_{x=0}^{2^N-1} (-1)^{f(x)} = \frac{1}{2^N} \sum_{x=0}^{2^N-1} (-1)^{C} = \frac{1}{2^N} \cdot 2^N (-1)^C = (-1)^C$$
由于所有基础状态的幅度平方和始终等于 $1$，因此其余基础状态的幅度必须为 0 - 这意味着步骤 3 之后的量子位的状态*为* $\ket{0}^{\otimes N}$。

* 如果函数是平衡的，即对于一半输入返回 $0$，对于另一半输入返回 $1$，则总和 $\frac{1}{2^N} \sum_{x=0}^{2^N-1} (-1)^{f(x)}$ 中的一半项将是 $1$，另一半项将是 $-1$，并且它们将全部抵消，留下幅度$\ket{0}^{\otimes N}$ 等于 $0$。

现在，当你测量所有量子位时会发生什么？ （请记住，测量结果得到某种状态的概率等于该状态幅度的平方。）

如果函数恒定，则您可以获得的唯一测量结果全为零 - 获得任何其他结果的概率为 $0$。如果函数是平衡的，则获得全零的概率为 $0$，因此您将获得除此之外的任何测量结果。

这正是算法的最后一步：**测量所有量子位，如果所有测量结果均为 0，则函数为常数，否则为平衡**。

＃＃＃ 概括

最后，算法非常简单：

1. 将 $H$ 门应用于每个量子位。
2.应用神谕。
3. 再次将 $H$ 门应用于每个量子位。
4. 测量所有量子位。
5. 如果所有量子位均在 $\ket{0}$ 状态下测量，则函数为常数，否则为平衡函数。

请注意，该算法仅需要 $1$ oracle 调用，并且始终产生正确的结果！
