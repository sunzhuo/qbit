# 实现 Bernstein-Vazirani 算法

**输入：** 
1. 输入 $N$ ($1 \le N \le 3$) 中的位数。
2. “黑匣子”预言机采用量子位数组作为参数，并将 $f(x)$ 实现为相位预言机。  
  您可以保证，oracle 实现的函数可以表示为某个位串 $s$ 的标量积 $f(x) = x \cdot s$。

**目标：** 返回位串 $s$。
您只能使用一次 oracle 调用！

## 解决方案注释

除了测量步骤之外，该算法的实现与 Deutsch-Jozsa 算法非常相似。

1. 分配 $N$ 量子位。
2. 将 $H$ 门应用于每个量子位。
3.应用神谕。
4. 再次将 $H$ 门应用于每个量子位。
5. 测量每个量子位并将测量结果转换为位 $0$ 和 $1$。

来源：Solution.qs

```qsharp
namespace Kata {
    operation BernsteinVaziraniAlgorithm (N : Int, oracle : Qubit[] => Unit) : Int[] {
        mutable s = [];
        use x = Qubit[N];
        ApplyToEach(H, x);
        oracle(x);
        ApplyToEach(H, x);
        for xi in x {
            set s += [MResetZ(xi) == Zero ? 0 | 1];
        }
        return s;
    }
}
```
