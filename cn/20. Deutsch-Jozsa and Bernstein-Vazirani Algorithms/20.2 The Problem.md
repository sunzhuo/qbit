# 问题

给您一个经典函数，它将 $N$ 位字符串作为输入并返回一位：$f(x): \{0, 1\}^N \to \{0, 1\}$。您保证函数 $f$ 是

- 任一*常量*（为所有输入返回相同的值） 
- 或 *平衡*（一半输入返回值 $0$，另一半输入返回值 $1$）。

任务是弄清楚函数是恒定的还是平衡的。

**示例**

- $f(x) \equiv 0$ 或 $f(x) \equiv 1$ 是常量函数（它们实际上是唯一存在的常量函数）。
- $f(x) = x \bmod 2$（$x$ 的最低有效位）或 $f(x) = 1 \text{ if the binary notation of }x \text{ has odd number of 1s and 0 otherwise}$ 是多位平衡函数的示例。事实上，对于这两个函数，您可以检查对于 $f(x) = 0$ 的每个可能的输入 $x$，都存在一个输入 $x^\prime$（等于 $x$，最低有效位翻转），使得 $f(x^\prime) = 1$，反之亦然，这意味着该函数是平衡的。

如果您以经典方式解决此问题，您将需要对给定函数进行多少次调用？

第一次调用不会给您任何信息 - 无论它返回 $0$ 还是 $1$，该函数仍然可以是常量或平衡的。
在最好的情况下，第二次调用将返回不同的值。您将能够得出结论，该函数仅在 $2$ 调用中达到平衡。 
但是，如果前两次调用获得相同的值，则必须继续查询该函数，直到该函数返回不同的值，或者直到执行返回相同值的 $2^{N-1}+1$ 查询 - 只有在这种情况下，您才能确定该函数是常量。
