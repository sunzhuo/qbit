# Oracle for f(x) = x 中 1 位数的奇偶校验

**输入：** 处于任意状态 $\ket{x}$ 的 $N$ 量子位数组。

**目标：** 将 $f(x) = 1 \text{ if x has odd number of 1s, and } 0 \text{ otherwise }$ 的相位预言 $U_f$ 应用到量子位。
也就是说，将相对相位$(-1)^{f(x)}$应用于每个基础状态$\ket{x}$。

<details>
<summary><strong>需要提示吗？</strong></summary>
您能否将该函数表示为依赖于 $x$ 各个位（模 $2$）的表达式之和？ 
那么，您能否将预言机对量子位数组的影响表示为对数组中每个量子位的影响的乘积？
</details>

## 解决方案注释

在这个预言中，答案取决于输入的所有位。您可以如下编写$f(x)$（这里$\bigoplus$表示求和模$2$）：

$$f(x) = \bigoplus_{k=0}^{N-1} x_k$$ 

让我们将这个表达式替换为量子态预言效应的表达式：

$$U_f \ket{x} = (-1)^{f(x)} \ket{x} = (-1)^{\bigoplus_{k=0}^{N-1} x_k} \ket{x}$$

从 $(-1)^2 = 1$ 开始，您可以将模 $2$ 的总和替换为指数中的常规总和。然后，您将能够将其重写为每个位的各个指数的乘积：

$$U_f \ket{x} = (-1)^{\sum_{k=0}^{N-1} x_k} \ket{x} = \prod_{k=0}^{N-1} {(-1)^{x_k}} \ket{x}$$

现在让我们将系统状态阐明为各个量子位状态的张量积：

$$U_f \ket{x} = \prod_{k=0}^{N-1} {(-1)^{x_k}} \cdot \ket{x_{0} } \otimes \cdots \otimes \ket{x_{N-1}}$$

张量积是线性运算，因此您可以将每个 $(-1)^{x_k}$ 标量因子放在相应的 $\ket{x_k}$ 旁边：

$$U_f \ket{x} = (-1)^{x_0} \ket{x_{k}} \otimes \dots \otimes (-1)^{x_{N-1}} \ket{x_{N-1}} = \bigotimes_{k=0}^{N-1} (-1)^{x_k} \ket{x_{k}}$$

正如您在之前的预言中所看到的，这可以通过对每个量子位应用 $Z$ 门来实现。

```qsharp
namespace Kata {
    operation PhaseOracle_Parity(x : Qubit[]) : Unit is Adj + Ctl {
        for xi in x {
            Z(xi);
        }
    }
}
```
