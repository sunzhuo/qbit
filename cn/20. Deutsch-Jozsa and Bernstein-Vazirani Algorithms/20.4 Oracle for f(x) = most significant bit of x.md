# Oracle for f(x) = x 的最高有效位

**输入：** 处于任意状态 $\ket{x}$ 的 $N$ 量子位数组。

**目标：** 将 $f(x) = \text{most significant bit of } x$ 的相位预言 $U_f$ 应用到量子位。
也就是说，将相对相位$(-1)^{f(x)}$应用于每个基础状态$\ket{x}$。
$x$ 使用大端编码，首先存储最高有效位。

## 解决方案注释

$x$ 的二进制表示为 $x = (x_{0}, x_{1}, \dots, x_{N-1})$，最高有效位编码在第一位（存储在输入数组的第一个量子位中）。然后，您可以将该函数重写为

$$f(x) = x_0$$

及其对量子态的影响为

$$U_f \ket{x} = (-1)^{f(x)} \ket{x} = (-1)^{x_0} \ket{x} = (-1)^{x_0} \ket{x_{0} } \otimes \ket{x_1} \otimes \cdots \otimes \ket{x_{N-1}}$$

正如您在上一个练习中所看到的，这可以通过将 $Z$ 门应用于第一个量子位来实现。

来源：Solution.qs

```qsharp
namespace Kata {
    operation PhaseOracle_MostSignificantBit(x : Qubit[]) : Unit is Adj + Ctl {
        Z(x[0]);
    }
}
```
