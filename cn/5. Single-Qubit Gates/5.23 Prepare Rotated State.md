# 准备旋转状态

**输入：**

1. 实数 $\alpha$ 和 $\beta$ 使得 $\alpha^2 + \beta^2 = 1$。
2. 处于状态 $\ket{0}$ 的量子位。

**目标：** 使用旋转门将量子位转换为状态 $\alpha\ket{0} -i\beta\ket{1}$。

> 您可能需要 `Std.Math` 命名空间中的函数，特别是 <a href="https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.math/arctan2" target="_blank">ArcTan2</a>。
>
> 您可以使用 `let` 关键字在 Q# 中分配变量：`let num = 3;` 或 `let result = Function(input);`

## 解决方案注释

您使用旋转门$R_x(\theta)$。该门将状态 $\ket{0}$ 转变为 $R_x(\theta)\ket{0} = \cos\frac{\theta}{2}\ket{0} - i\sin\frac{\theta}{2}\ket{1}$。
这与您需要的状态类似。您只需找到一个角度 $\theta$，使得 $\cos\frac{\theta}{2}=\alpha$ 和 $\sin\frac{\theta}{2}=\beta$ 一致。您可以使用这两个方程求解 $\theta$: $\theta = 2\arctan\frac{\beta}{\alpha}$。 （*注：已知 $\alpha^2 + \beta^2=1$*）。
因此，所需的门是 $R_x(2\arctan\frac{\beta}{\alpha})$，其矩阵形式是 $\begin{bmatrix} \alpha & -i\beta \\ -i\beta & \alpha \end{bmatrix}$。
该门将 $\ket{0} = \begin{bmatrix} 1 \\ 0\end{bmatrix}$ 转换为 $\begin{bmatrix} \alpha & -i\beta \\ -i\beta & \alpha \end{bmatrix} \begin{bmatrix} 1 \\ 0\end{bmatrix} = \begin{bmatrix} \alpha \\ -i\beta \end{bmatrix} = \alpha\ket{0} -i\beta\ket{1}$。

> Q# 中可通过 `Std.Math` 命名空间使用三角函数。在这种情况下，您将需要 <a href="https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.math/arctan2" target="_blank">ArcTan2</a>。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    operation PrepareRotatedState(alpha : Double, beta : Double, q : Qubit) : Unit is Adj + Ctl {
        let phi = ArcTan2(beta, alpha);
        Rx(2.0 * phi, q);
    }
}
```
