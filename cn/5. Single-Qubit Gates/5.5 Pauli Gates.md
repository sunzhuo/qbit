#保利·盖茨

本节介绍一些常见的单量子位门，包括它们的矩阵形式、ket-bra 分解，以及列出它们对一些常见量子位状态的影响的简短“备忘单”。

您可以使用名为 <a href="https://algassert.com/quirk" target="_blank">Quirk</a> 的工具来可视化这些门如何与各种量子位状态交互。

本节依赖于以下符号：

泡利门以 <a href="https://en.wikipedia.org/wiki/Wolfgang_Pauli" target="_blank">Wolfgang Pauli</a> 命名，基于所谓的 **泡利矩阵**、$X$、$Y$ 和 $Z$。所有三个泡利门都是**自伴随**，这意味着每个泡利门都是其自己的逆门，$XX = \mathbb{I}$。

      <td>$X\ket{\psi} = \alpha\ket{1} + \beta\ket{0}$</td>

        $X\ket{-} = -\ket{-}$<br>
        $X\ket{i} = i\ket{-i}$<br>
        $X\ket{-i} = -i\ket{i}$
      </td>
    </tr>
    <tr>
      <td>$Y$</td>
      <td>$\begin{bmatrix} 0 & -i \\ i & 0 \end{bmatrix}$</td>

      <td>$Y\ket{\psi} = i\big(\alpha\ket{1} - \beta\ket{0}\big)$</td>
      <td>
        $Y\ket{0} = i\ket{1}$<br>
        $Y\ket{1} = -i\ket{0}$<br>
        $Y\ket{+} = -i\ket{-}$<br>
        $Y\ket{-} = i\ket{+}$<br>
        $Y\ket{i} = \ket{i}$<br>
        $Y\ket{-i} = -\ket{-i}$<br>
      </td>
    </tr>
    <tr>
      <td>$Z$</td>

      <td>$Z\ket{\psi} = \alpha\ket{0} - \beta\ket{1}$</td>
      <td>
        $Z\ket{0} = \ket{0}$<br>
        $Z\ket{1} = -\ket{1}$<br>
        $Z\ket{+} = \ket{-}$<br>
        $Z\ket{-} = \ket{+}$<br>
        $Z\ket{i} = \ket{-i}$<br>
        $Z\ket{-i} = \ket{i}$<br>

> $X$ 门有时被称为 **位翻转** 门或 **NOT** 门，因为它在计算基础上的作用类似于经典的非门。
>
> $Z$ 门有时称为 **相位翻转** 门。

以下是泡利门的几个易于验证且方便记忆的属性：

- 不同的泡利门_反通勤_：
  $$XZ = -ZX, XY = -YX, YZ = -ZY$$
- 任意两个泡利门的乘积等于第三个门，并带有额外的 $i$（或 $-i$）相位：
  $$XY = iZ, YZ = iX, ZX = iY$$
- 所有三个泡利门的乘积等于恒等式（带有额外的 $i$ 阶段）：
  $$XYZ = iI$$
