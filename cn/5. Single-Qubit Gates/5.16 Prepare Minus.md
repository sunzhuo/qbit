# 准备减号

**输入：** 处于状态 $\ket{0}$ 的量子位。

**目标：** 将量子位转换为状态 $\ket{-}$。

## 解决方案注释

您知道，在计算基础状态 $\ket{0}$ 和 $\ket{1}$ 上应用 Hadamard 门 $H$ 分别会产生 Hadamard 基础状态 $\ket{+}$ 和 $\ket{-}$。
您将获得一个状态为 $\ket{0}$ 的量子位。您首先应用泡利 $X$ 门将其转变为 $X\ket{0}=\ket{1}$，然后应用 $H$ 门，将量子位转变为所需的 $H\ket{1}=\ket{-}$ 状态。

来源：Solution.qs

```qsharp
namespace Kata {
    operation PrepareMinus (q : Qubit) : Unit is Adj+Ctl {
        X(q);
        H(q);
    }
}
```
