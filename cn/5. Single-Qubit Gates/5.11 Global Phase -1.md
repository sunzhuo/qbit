# 全球阶段-1

**输入：** 处于状态 $\ket{\psi} = \beta \ket{0} + \gamma \ket{1}$ 的量子位。

**目标**：将量子位的状态更改为 $- \beta \ket{0} - \gamma \ket{1}$。

> 这种变化本身是不可观察的 - 您无法对独立的量子位进行实验来确定它是否获得了全局阶段。
> 但是，您可以使用此操作的受控版本来观察它引入的全局阶段。
> 这在后来的型中用作更复杂任务的一部分。

## 解决方案注释

全局相位是乘以整个量子态的相位因子。在测量量子位的状态时无法观察到它，因为概率保持不变。然而，在考虑量子态转换时，它很重要。

您的任务是实现将给定量子位状态从 $\ket{\psi} = \beta \ket{0} + \gamma \ket{1}$ 转换为 $- \beta \ket{0} - \gamma \ket{1}$ 的操作。

为此，您需要使用一系列门。当连续应用两次时，先使用泡利 Z 门，然后使用泡利 X 门，即可达到此效果。

1. **应用泡利 Z 门**：Z 门将 $\ket{1}$ 状态乘以 $-1$，但不更改 $\ket{0}$ 状态，将状态 $\beta \ket{0} + \gamma \ket{1}$ 转换为 $\beta \ket{0} - \gamma \ket{1}$。

Z门的矩阵表示为：

   $$
   Z =
   \begin{bmatrix}1 & 0 \\ 0 & -1 \end{bmatrix}
   $$

2. **应用泡利 X 门**：X 门翻转 $\ket{0}$ 和 $\ket{1}$ 基本状态，将 $\beta \ket{0} - \gamma \ket{1}$ 状态转换为 $\beta \ket{1} - \gamma \ket{0}$。

X门的矩阵表示为：

   $$
   X =
   \begin{bmatrix}0 & 1 \\ 1 & 0\end{bmatrix}
   $$

3. **重复 Z 门和 X 门**：再次应用 Z 门将使 $\ket{1}$ 状态（以前是 $\ket{0}$）相乘，将状态 $\beta \ket{1} - \gamma \ket{0}$ 转换为 $- \beta \ket{1} - \gamma \ket{0}$。最后，第二个 X 门将恢复原始基本状态，但现在两个幅度都已获得 $-1$ 的附加相位。这意味着状态已乘以 $-1$，实现所需的全局相变。

```qsharp
namespace Kata {
    operation GlobalPhaseChange (q : Qubit) : Unit is Adj + Ctl {
        Z(q);
        X(q);
        Z(q);
        X(q);
    }
}
```
