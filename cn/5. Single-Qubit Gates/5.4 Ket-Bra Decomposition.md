#Ket-Bra分解

本节描述了寻找量子门的 ket-bra 分解的更正式的过程。本节对于开始使用量子门来说并不是必需的，因此请暂时跳过它，稍后再回来。

您可以使用 _eigenvalues_ 和 _eigenvectors_ 的属性来查找任何门的 ket-bra 分解。给定门 $A$ 和正交向量 $\ket{\phi}$ 和 $\ket{\psi}$，如果：

$$A\ket{\phi} = x_\phi\ket{\phi}$$
$$A\ket{\psi} = x_\psi\ket{\psi}$$

实数 $x_\phi$ 和 $x_\psi$ 称为特征值，$\ket{\phi}$ 和 $\ket{\psi}$ 是 $A$ 的特征向量。然后：

$$A = x_\phi\ket{\phi}\bra{\phi} + x_\psi\ket{\psi}\bra{\psi}$$

让我们使用 $X$ 门作为一个简单的示例。 $X$ 门有两个特征向量：$\ket{+} = \frac{1}{\sqrt{2}}\big(\ket{0} + \ket{1}\big)$ 和 $\ket{-} = \frac{1}{\sqrt{2}}\big(\ket{0} - \ket{1}\big)$。它们的特征值分别为 $1$ 和 $-1$：

$$X\ket{+} = \ket{+}$$
$$X\ket{-} = -\ket{-}$$

分解如下：
$$X = \ket{+}\bra{+} - \ket{-}\bra{-} =$$
$$= \frac{1}{2}\big[\big(\ket{0} + \ket{1}\big)\big(\bra{0} + \bra{1}\big) - \big(\ket{0} - \ket{1}\big)\big(\bra{0} - \bra{1}\big)\big] =$$
$$= \frac{1}{2}\big(\ket{0}\bra{0} + \ket{0}\bra{1} + \ket{1}\bra{0} + \ket{1}\bra{1} - \ket{0}\bra{0} + \ket{0}\bra{1} + \ket{1}\bra{0} - \ket{1}\bra{1}\big) =$$
$$= \frac{1}{2}\big(2\ket{0}\bra{1} + 2\ket{1}\bra{0}\big) =$$
$$= \ket{0}\bra{1} + \ket{1}\bra{0}$$
