# 相对阶段 i

**输入：** 处于状态 $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$ 的量子位。

**目标：** 将量子位状态更改为 $\alpha \ket{0} + {\color{red}i}\beta \ket{1}$（将相对相位 $i$ 添加到叠加的 $\ket{1}$ 分量）。

## 解决方案注释

#### 解决方案 1

您可以认识到，$S$ 门对 $\ket{1}$ 基础状态执行此特定的相对相位加法。提醒一下，

$$
S =
\begin{bmatrix}1 & 0 \\ 0 & i\end{bmatrix}
$$

让我们看看这个门对一般叠加 $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$ 的影响。

$$
 \begin{bmatrix}1 & 0 \\ 0 & i \end{bmatrix}
 \begin{bmatrix}\alpha \\ \beta \end{bmatrix}=
\begin{bmatrix}1\cdot\alpha + 0\cdot\beta \\ 0\cdot\alpha + i\cdot\beta \end{bmatrix}=
 \begin{bmatrix}\alpha \\ i\beta \end{bmatrix}
$$

因此很容易看出当 $\ket{\psi} = 0.6\ket{0} +  0.8\ket{1}, S\ket{\psi} =  0.6\ket{0} + 0.8i\ket{1}$.

来源：SolutionA.qs

```qsharp
namespace Kata {
    operation PhaseFlip (q : Qubit) : Unit is Adj + Ctl {
        S(q);
    }
}
```

#### 解决方案2

或者，请参阅 kata 后面的复杂相对相位任务，了解使用 $R1$ 门实现相同转换的说明。

来源：SolutionB.qs

```qsharp
namespace Kata {
    operation PhaseFlip(q : Qubit) : Unit is Adj + Ctl {
        import Std.Math.*;
        R1(0.5 * PI(), q);
    }
}
```
