# 概述

超密集编码协议允许我们通过使用先前共享的量子纠缠仅发送一个量子位来传输两位经典信息。我们假设 Alice 是发送者，Bob 是接收者。

- 可以在[维基百科文章](https://en.wikipedia.org/wiki/Superdense_coding)中找到很好的描述。
- 可以在 [Wolfram 演示项目](https://demonstrations.wolfram.com/SuperdenseCoding/) 找到精彩的交互式演示。

我们将超密集编码协议分为几个步骤：

- 准备（创建发送给 Alice 和 Bob 的纠缠量子位对）。
- 对消息进行编码（Alice 的任务）：将消息的经典位编码为 Alice 量子位的状态，然后将其发送给 Bob。
- 解码消息（Bob 的任务）：使用 Bob 的原始量子位和他从 Alice 收到的量子位来解码发送的经典消息。
- 最后，我们将这些步骤组合成完整的超密集编码协议。
