# Alice 的任务：对消息进行编码

**输入**：

1. `qAlice` ：Alice 的纠缠量子位对的一部分。
2. `message`：两个经典位，由两个 `Bool` 变量的元组表示，分别表示 `bit1` 和 `bit2`。

**目标**：
通过操纵爱丽丝的量子位对消息（两个经典位）进行编码。

超密集编码协议根据消息的值将两个量子位的联合状态更改为以下四种状态之一：

- `(0, 0)`：$\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} (\ket{00} + \ket{11})$
- `(0, 1)`：$\ket{\Psi^{+}} = \frac{1}{\sqrt{2}} (\ket{01} + \ket{10})$
- `(1, 0)`：$\ket{\Phi^{-}} = \frac{1}{\sqrt{2}} (\ket{00} - \ket{11})$
- `(1, 1)`：$\ket{\Psi^{-}} = \frac{1}{\sqrt{2}} (\ket{01} - \ket{10})$

请注意，您的解决方案是作为协议端到端实现的一部分进行测试的；目标是正确传输消息。

## 解决方案注释

回想一下，我们在“准备量子态”kata 中学习了如何准备所有贝尔态。这是该任务的稍微高级版本，演示了应用于贝尔状态的一个量子位的操作如何允许我们将其转换为任何其他贝尔状态。

超密集编码协议使用我们在上一个任务中准备的以下贝尔状态：

$$\frac{1}{\sqrt{2}} \big(\ket{00} + \ket{11} \big)$$

我们可以根据`message`的值将其转换为其他所有Bell状态：

- `(0, 0)`：$\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} (\ket{00} + \ket{11})$
- `(0, 1)`：$\ket{\Psi^{+}} = \frac{1}{\sqrt{2}} (\ket{01} + \ket{10})$
- `(1, 0)`：$\ket{\Phi^{-}} = \frac{1}{\sqrt{2}} (\ket{00} - \ket{11})$
- `(1, 1)`：$\ket{\Psi^{-}} = \frac{1}{\sqrt{2}} (\ket{01} - \ket{10})$

以下是我们如何执行此转换：

- 如果 `bits == (0, 0)`，我们什么都不做 - 准备状态已经是 $\ket{\Phi^{+}}$。

- 如果是 `bits ==  (0, 1)`，我们需要更改 $\ket{00}$ 和 $\ket{11}$ 项中的第二个量子位。观察一下，将 $X$ 门应用于 Alice 的量子位确实会起到这样的作用：
  $$(X \otimes I) \ket{\Phi^{+}} = \frac{1}{\sqrt{2}} (\ket{10} + \ket{01})$$

- 如果是`bits == (1, 0)`，我们需要将$-1$的相对相位添加到$\ket{11}$项中。观察一下，将 $Z$ 门应用于 Alice 的量子位确实会实现这一点：
  $$(Z \otimes I) \ket{\Phi^{+}} = \frac{1}{\sqrt{2}} (\ket{00} - \ket{11})$$

- 如果是`bits = (1, 1)`，我们使用相同的逻辑来认识到我们需要应用$Z$和$X$校正来获得$\ket{\Psi^{-}}$状态。
  $$ (Z \otimes I) \cdot (X \otimes I) \ket{\Psi^{+}} = (Z \otimes I) \frac{1}{\sqrt{2}} (\ket{10} + \ket{01}) = \frac{1}{\sqrt{2}} (-\ket{10} + \ket{01}) = \frac{1}{\sqrt{2}} (\ket{01} - \ket{10}) $$

最终的步骤顺序如下：

1. 如果 `bit2 == 1`，则将 $X$ 门应用于 Alice 的量子位。
2. 如果 `bit1 == 1`，则将 $Z$ 门应用于 Alice 的量子位。

来源：Solution.qs

```qsharp
namespace Kata {
    operation EncodeMessageInQubit(qAlice : Qubit, message : (Bool, Bool)) : Unit {
        let (bit1, bit2) = message;

        if bit2 {
            X(qAlice);
        }

        if bit1 {
            Z(qAlice);
        }
    }
}
```
