# 纠缠对

**输入：**

1. `qAlice` ：Alice 的量子位处于 $\ket{0}$ 状态。
2. `qBob` ：Bob 的量子位处于 $\ket{0}$ 状态。

**目标：** 在这些量子位上准备贝尔态 $\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} (\ket{00} + \ket{11})$。

## 解决方案注释

回想一下，这个状态就是我们在“多量子位门”和“准备量子态”套路中已经看到的贝尔状态。

该解决方案需要两个步骤：

1. 对第一个量子位应用 Hadamard 门以获得状态：$\big(\frac{1}{\sqrt{2}} \ket{0} + \frac{1}{\sqrt{2}} \ket{1} \big) \otimes \ket{0}$。
2. 应用 $CNOT$ 门，将第一个量子位作为对照，将第二个量子位作为目标。

来源：Solution.qs

```qsharp
namespace Kata {
    operation CreateEntangledPair(qAlice : Qubit, qBob : Qubit) : Unit is Adj {
        H(qAlice);
        CNOT(qAlice, qBob);
    }
}
```
