# Bob 的任务：解码消息

**输入：**

1. `qAlice` ：从 Alice 收到的量子比特。
2. `qBob` ：Bob 是纠缠对的一部分。

**目标**：使用从 Alice 收到的量子位解码消息。为此，从量子位中检索两位经典数据，并返回 `(Bool, Bool)` 形式的元组来表示 `message`。操作结束时量子位的状态并不重要。

请注意，您的解决方案是作为协议端到端实现的一部分进行测试的；目标是正确传输消息。

## 解决方案注释

回想一下，爱丽丝对量子位进行了如下编码：

- `(0, 0)`：$\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} (\ket{00} + \ket{11})$
- `(0, 1)`：$\ket{\Psi^{+}} = \frac{1}{\sqrt{2}} (\ket{01} + \ket{10})$
- `(1, 0)`：$\ket{\Phi^{-}} = \frac{1}{\sqrt{2}} (\ket{00} - \ket{11})$
- `(1, 1)`：$\ket{\Psi^{-}} = \frac{1}{\sqrt{2}} (\ket{01} - \ket{10})$

为了读出编码消息，鲍勃需要弄清楚他拥有四个贝尔状态中的哪一个。我们可以通过应用第一个量子位作为控制、第二个量子位作为目标的 $CNOT$ 门，然后在第一个量子位上应用 $H$ 门，将贝尔状态映射到基本状态。 （请注意，这正是第一个任务中状态准备操作的 [`Adjoint`](https://learn.microsoft.com/azure/quantum/user-guide/language/expressions/functorapplication#adjoint-functor) 所做的事情。）
  
假设编码步骤后每种可能的量子态，这种转换的结果是什么？

- $\ket{\Phi^{+}} = \frac{1}{\sqrt{2}} (\ket{00} + \ket{11}) \rightarrow \ket{00}$
- $\ket{\Psi^{+}} = \frac{1}{\sqrt{2}} (\ket{01} + \ket{10}) \rightarrow \ket{01}$
- $\ket{\Phi^{-}} = \frac{1}{\sqrt{2}} (\ket{00} - \ket{11}) \rightarrow \ket{10}$
- $\ket{\Psi^{-}} = \frac{1}{\sqrt{2}} (\ket{01} - \ket{10}) \rightarrow \ket{11}$

因此，我们可以仅通过测量比特来检索编码比特。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DecodeMessageFromQubits(qAlice : Qubit, qBob : Qubit) : (Bool, Bool) {
        CNOT(qAlice, qBob);
        H(qAlice);
        return (MResetZ(qAlice) == One, MResetZ(qBob) == One);
    }
}
```
