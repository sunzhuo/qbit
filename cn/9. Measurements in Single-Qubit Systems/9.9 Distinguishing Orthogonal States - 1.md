# 区分正交态：1

**输入：** 保证处于 $\ket {\psi_+}$ 或 $\ket{\psi_-} $ 状态的量子位，其中 $\ket {\psi_+} = 0.6\ket 0 + 0.8 \ket 1 $ 和 $\ket {\psi_-} = -0.8\ket 0 + 0.6 \ket 1$。

**输出：** 如果量子位处于 $\ket {\psi_+}$ 状态，则为 `true`；如果处于 $\ket{\psi_-} $ 状态，则为 `false`。操作结束时量子位的状态并不重要。

<details>
<summary><strong>需要提示吗？</strong></summary>
可以使用合适的 $R_y$ 旋转从计算基础 $\{ \ket 0, \ket 1 \}$ 转到 $\{ \ket{\psi_+}, \ket{\psi_-} \}$ 基础，反之亦然。
</details>

## 解决方案注释

如果我们在 $\{ \ket{\psi_+}, \ket{\psi_-}\}$ 基础上实现测量，我们可以区分状态 $\ket{\psi_\pm}$。如果我们构造一个将 $\ket{\psi_+}$ 状态映射到 $\ket{0}$ 状态，并将 $\ket{\psi_{-}}$ 状态映射到 $\ket{1}$ 状态的酉变换，就可以做到这一点。

我们可以注意到，带有 $\theta = 2 \arctan \frac{0.8}{0.6}$ 的 $R_y$ 旋转门是一个合适的变换：

$$R_y(\theta) \ket 0 = 0.6 \ket 0 + 0.8 \ket 1 = \ket {\psi_+},$$
$$R_y(\theta) \ket 1 = -0.8 \ket 0 + 0.6 \ket 1 = \ket{\psi_-}.$$

因此，逆（伴随）变换 $R_y(-\theta)$ 将 $\ket{\psi_\pm}$ 基础映射到计算基础，即
$$R_y(-\theta) \ket {\psi_+} = \ket 0,$$
$$R_y(-\theta) \ket {\psi_-} = \ket 1.$$

因此，如果我们将 $R_y(-\theta)$ 应用于量子位，其状态将转换为计算基础状态之一，此时我们可以使用 `M` 来测量它。如果 `M` 返回 `Zero`，则旋转状态为 $\ket{0}$，这意味着量子位的原始状态为 $\ket{\psi_+}$。类似地，`One` 的输出指示量子位最初处于状态 $\ket{\psi_-}$。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    operation IsQubitPsiPlus(q : Qubit) : Bool {
        Ry(-2.0 * ArcTan2(0.8, 0.6), q);
        return M(q) == Zero;
    }
}
```
