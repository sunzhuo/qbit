# 使用操作 M 和 MResetZ 在 Q# 中实现测量

在这个演示中，我们准备一个状态为$0.6\ket{0} + 0.8\ket{1}$的量子位，然后在计算基础上对其进行测量。在 Q# 中，可以使用 `M` 操作来实现计算基础上的单量子位测量。如果测量结果为 $0$，则返回常量 `Zero`；如果测量结果为 $1$，则返回常量 `One`。 `Zero` 和 `One` 是 `Result` 类型的常量。

> 如果多次运行此代码，您会注意到每当测量结果为 $1$ 时，量子位的测量后状态为 $\ket 1$，同样，对于结果 $0$，最终状态为 $\ket{0}$。这符合我们的预期，即测量后波函数“崩溃”到相应的状态。

实现计算基础测量的另一种方法是使用 `MResetZ` 操作来测量量子位并将其重置为 $\ket{0}$ 状态。请记住，Q# 要求您在释放所有量子位之前将其重置为 $\ket{0}$，因此，如果您在测量后不需要使用量子位，那么立即使用 `MResetZ` 会很方便，而不是使用 `M` 进行测量，然后单独调用 `Reset`。

来源：Example.qs

```qsharp
namespace Kata {
    import Std.Diagnostics.*;
    import Std.Math.*;

    @EntryPoint()
    operation SimpleMeasurementDemo() : Unit {
        use q = Qubit();
        // Prepare the qubit in the superposition state |𝜓❭ = 0.6 |0❭ + 0.8 |1❭
        Ry(2.0 * ArcTan2(0.8, 0.6), q);
        Message("Qubit in state |𝜓❭:");
        DumpMachine();

        let outcome = M(q) == One ? 1 | 0;
        Message($"The measurement outcome is {outcome}.");
        Message("Post-measurement state of the qubit:");
        DumpMachine();
        Reset(q);
    }
}
```
