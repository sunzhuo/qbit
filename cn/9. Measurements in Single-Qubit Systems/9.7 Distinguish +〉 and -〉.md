# 区分 |+〉 和 |-〉

**输入**：保证处于 $\ket +$ 状态或 $\ket -$ 状态的量子位。

**输出**：如果量子位处于 $\ket -$ 状态，则为 `true`；如果量子位处于 $\ket +$ 状态，则为 `false`。操作结束时量子位的状态并不重要。

> 要使用 `Measure` 操作在特定泡利基中执行单量子位测量，
> 您需要向其传递两个参数：第一，一个 `Pauli` 常量（`PauliX`、`PauliY` 或 `PauliZ`）组成的数组，第二个是您要测量的一个量子位的数组。

## 解决方案注释

输入量子位保证处于基本状态 $\ket{+}$ 或 $\ket{-}$。这意味着当测量泡利X基中的量子位时，测量将毫无疑问地报告输入状态。 （回想一下，这些状态是泡利 X 矩阵的本征态）。

在 Q# 中，操作 `Measure` 可用于测量用户选择的泡利基中的量子位。该操作返回 `Result` 类型的值，如果测量状态对应于泡利算子的特征值 $+1$，则返回 `Zero`；如果测量状态对应于泡利算子的特征值 $-1$，则返回 `One`。

由于状态$\ket +$和$\ket -$对应于Pauli X算子的特征值$+1$和$-1$，因此我们可以返回测量结果与`One`相等比较的结果。
请注意，由于 `Measure` 操作通常使用多个量子位来执行多量子位测量，因此它需要数组参数。要进行单量子位测量，您需要传递一个元素的两个数组 `[PauliX]` 和 `[q]`，而不是单个值。

来源：Solution.qs

```qsharp
namespace Kata {
    operation IsQubitMinus(q : Qubit) : Bool {
        return Measure([PauliX], [q]) == One;
    }
}
```
