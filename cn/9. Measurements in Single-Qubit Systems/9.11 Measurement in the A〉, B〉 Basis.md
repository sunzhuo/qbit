# 以|A〉、|B〉为基础进行测量

**输入：**

1. 角度 $\alpha$，以弧度为单位，表示为 `Double`。
1. 处于某种未知状态的量子位。

**输出：** 在 $\{\ket A, \ket B\}$ 基础上实施测量。与上一个练习相同，$\ket{A} = \cos \alpha \ket{0} - i \sin \alpha \ket{1}$ 和 $\ket{B} = - i \sin \alpha \ket{0} + \cos \alpha \ket{1}$。如果测量结果为 $A$，则返回 `Zero`，如果结果为 $B$，则返回 `One`。
测量后量子位的状态应与测量结果相对应。

<details>
    <summary><strong>需要提示吗？</strong></summary>
    <p>$R_x$ 旋转可用于从计算基础 $\{ \ket 0, \ket 1 \}$ 到 $\{ \ket{A}, \ket{B} \}$ 基础，反之亦然。</p>
</details>

## 解决方案注释

对于这个问题，我们遵循 Kata 的“任意正交基的测量”部分中给出的过程。
如“区分正交状态：2”练习的解决方案中所述，门 $R_x(-\theta)$ 和 $\theta = 2\alpha$ 将 $\ket A/\ket B$ 状态转换为 $\ket 0/\ket 1$ 状态：
$$R_x(-\theta) \ket A = \ket 0,$$
$$R_x(-\theta) \ket B = \ket 1.$$
因此，我们首先将 $R_x(-\theta)$ 应用于量子位。接下来，我们使用 `M` 操作在计算基础上进行测量。
如果 `M` 操作返回 `Zero`，我们将获得测量结果 $A$，如果返回 `One`，我们将获得测量结果 $B$。

测量后，我们应用 $R_x(-\theta)$ 门的逆，即 $R_x(\theta)$ 门。
最终的旋转确保量子位的状态处于与测量结果相对应的状态。

来源：Solution.qs

```qsharp
namespace Kata {
    operation MeasureInABBasis(alpha : Double, q : Qubit) : Result {
        Rx(-2.0 * alpha, q);
        let measurementResult = M(q);
        Rx(2.0 * alpha, q);
        return measurementResult;
    }
}
```
