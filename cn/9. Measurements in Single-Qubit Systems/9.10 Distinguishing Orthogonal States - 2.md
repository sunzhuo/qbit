# 区分正交态：2

**输入：**

1. 角度 $\alpha$，以弧度为单位，表示为 `Double`。
2. 保证处于 $\ket{A}$ 或 $\ket{B}$ 状态的量子位，其中 $\ket{A} = \cos \alpha \ket{0} - i \sin \alpha \ket{1}$ 和 $\ket{B} = - i \sin \alpha \ket{0} + \cos \alpha \ket{1}$。

**输出：** 如果量子位处于 $\ket{A}$ 状态，则为 `true`；如果处于 $\ket{B}$ 状态，则为 `false`。操作结束时量子位的状态并不重要。

<details>
    <summary><strong>需要提示吗？</strong></summary>
    <p>$R_x$ 旋转可用于从计算基础 $\{ \ket 0, \ket 1 \}$ 到 $\{ \ket{A}, \ket{B} \}$ 基础，反之亦然。</p>
</details>

## 解决方案注释

如果我们在 $\{ \ket{A}, \ket{B}\}$ 基础上实现测量，我们可以区分状态 $\ket{A}$ 和 $\ket B$。

我们可以注意到，带有 $\theta = 2 \alpha$ 的 $R_x$ 旋转门是一种适当的变换，它将 $\ket 0 $ 状态映射到 $\ket A$ 状态，将 $\ket 1$ 状态映射到 $\ket B$ 状态：

$$R_x(\theta) \ket 0 = \cos \alpha \ket 0 -i \sin \alpha \ket 1 = \ket {A},$$
$$R_x(\theta) \ket 1 = -i \sin \alpha \ket 0 + \cos \alpha \ket 1 = \ket{B}.$$

因此，逆变换$R_x(-\theta)$将$A/B$基映射到$0/1$基。

因此，如果我们将 $R_x(-\theta)$ 应用于量子位并使用 `M` 对其进行测量，则 `Zero` 的测量结果将对应于量子位的原始状态 $\ket{A}$，而 `One` 的结果将对应于量子位的原始状态 $\ket B$。

来源：Solution.qs

```qsharp
namespace Kata {
    operation IsQubitA(alpha : Double, q : Qubit) : Bool {
        Rx(-2.0 * alpha, q);
        return M(q) == Zero;
    }
}
```
