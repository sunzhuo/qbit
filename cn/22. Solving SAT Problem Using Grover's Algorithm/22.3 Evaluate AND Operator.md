# 评估 AND 运算符

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。

**目标：**
实现一个计算输入 AND 的量子预言机，即 $f(x) = x_0 \wedge x_1 \wedge ... \wedge x_{N-1}$。
    
让输入寄存器中的量子位保持其开始时的状态。
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

目标是当且仅当寄存器 $\ket{x}$ 中的所有量子位都处于状态 $\ket{1}$ 时，翻转量子位 $\ket{y}$。

所需的单一 $U_{and}$ 是这样的：
$$U_{and}\ket{x}\ket{y} = \begin{cases} 
          \ket{x}\ket{y} & \text{if }x \neq 1...1 \\
          \ket{x}X\ket{y} & \text{如果 }x = 1...1 
       \end{cases}$$

此转换可以作为 `Controlled X` 门来实现，以输入寄存器 $\ket{x}$ 作为控制，以目标量子位 $\ket{y}$ 作为目标。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Oracle_And(x : Qubit[], y : Qubit) : Unit is Adj + Ctl {
        Controlled X(x, y);
    }        
}
```
