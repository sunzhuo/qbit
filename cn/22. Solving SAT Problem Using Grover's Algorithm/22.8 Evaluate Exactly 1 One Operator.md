# 评估“Exactly 1 One”运算符

**输入：**

1. 处于任意状态的三个量子位 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。

**目标：**
实现一个量子预言机，计算函数，检查输入之一是否恰好是 $1$，即，如果 $x$ 的一位正好是 $1$，则为 $f(x) = 1$，否则为 $0$。
    
让输入寄存器中的量子位保持其开始时的状态。
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

考虑长度为 $n$ 的所有位串 $x$ 的集合，其中 $x$ 中只有一位等于 $1$。   
这组位串是$S=\{00...01, 00...10, ..., 00..1..00, ..., 01...00, 10...00\}$， 
或者，如果我们将位串转换为整数，
$$S=\{1,2,4,..2^i,..,2^{n-1}\} = \{2^k: 0 \le k \le n-1\}$$

如果输入基础状态 $\ket{x}$ 对应于 $S$ 中的位串之一，您需要实现一个翻转 $\ket{y}$ 的预言机。
最简单的方法是使用 $n$ 控制的 $X$ 门，以 `x` 作为控制，当且仅当控制寄存器处于 $2^k$ 形式的特定状态时，量子位 `y` 才会翻转。
您可以使用库操作 `ApplyControlledOnInt` 来实现此目的，并以整数 $1, 2, 4, ...$ 作为控制。
请注意，此操作使用小端表示法将整数转换为位字符串，但这对于本练习来说并不重要。
您需要迭代所有符号中只有一个 $1$ 的位字符串，并且迭代的顺序并不重要。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Oracle_Exactly1One(x : Qubit[], y : Qubit) : Unit is Adj + Ctl {
        for i in 0 .. Length(x) - 1 {
            ApplyControlledOnInt(2 ^ i, X, x, y);
        }
    }        
}
```
