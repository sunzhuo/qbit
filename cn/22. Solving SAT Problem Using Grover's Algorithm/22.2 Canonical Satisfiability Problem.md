# 规范可满足性问题

布尔可满足性问题是确定布尔变量$x_j$是否存在赋值的问题
给定的布尔公式计算结果为 true。

SAT 问题的规范表示基于布尔公式的合取范式。
为了定义这种形式，我们需要几个定义：

- 布尔变量 $x_0, ..., x_{n-1}$ 是公式的输入。
- _literal_ 可以是变量 $x_j$（正文字），也可以是变量 $\neg x_j$ 的否定（称为负文字）。
- _clause_ 是一个或多个文字的析取（OR 运算），例如 $x_0 \vee \neg x_1$。
  一般来说，
  $$clause(x) = \bigvee_k y_{k},\text{ where }y_{k} =\text{ either }x_j\text{ or }\neg x_j\text{ for some }j \in \{0, \dots, n-1\}$$

- 最后，公式的_合取范式_是多个子句的合取（AND 运算）：
  $$f(x) = \bigwedge_i \big(\bigvee_k y_{ik} \big),\text{ where }y_{ik} =\text{ either }x_j\text{ or }\neg x_j\text{ for some }j \in \{0, \dots, n-1\}$$

在本课程中，您将学习如何为 SAT 问题实现标记预言，这些问题以合取范式描述的形式给出。
