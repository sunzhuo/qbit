# 评估 OR 运算符

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。

**目标：**
实现一个计算输入 OR 的量子预言机，即 $f(x) = x_0 \vee x_1 \vee ... \vee x_{N-1}$。
    
让输入寄存器中的量子位保持其开始时的状态。
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

目标是当且仅当寄存器 $\ket{x}$ 中的至少一个量子位处于状态 $\ket{1}$ 时，翻转量子位 $\ket{y}$。

所需的单一 $U_{or}$ 是这样的：
$$U_{or}\ket{x}\ket{y} = \begin{cases} 
          \ket{x}\ket{y} & \text{如果 }x = 0...0 \\
          \ket{x}X\ket{y} & \text{if }x \neq 0...0
       \end{cases}$$

此转换可以通过两个步骤的序列来实现：

1. 使用零控制 $X$ 门翻转目标量子位 if $x = 0...0$ 的状态。
2. 使用 $X$ 门翻转目标量子位的状态。这将否定上一步的结果，
   确保目标量子位的总体状态在 $x \neq 0...0$ 时被翻转。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Oracle_Or(x : Qubit[], y : Qubit) : Unit is Adj + Ctl {
        ApplyControlledOnInt(0, X, x, y);
        X(y);
    }        
}
```
