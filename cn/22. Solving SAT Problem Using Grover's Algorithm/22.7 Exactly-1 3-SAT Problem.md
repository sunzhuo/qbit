# Exactly-1 3-SAT 问题

Exactly-1 3-SAT 问题（也称为“三分之一 3-SAT”）是一般 SAT 问题的变体。
描述恰好 1 3-SAT 问题实例的公式结构与规范 SAT 问题的结构完全相同，每个子句恰好由三个文字组成。
然而，问题是找到变量的赋值，使每个子句都有*恰好一个*真实的文字， 
而在正常的 SAT 问题中，每个子句可以有一个或多个真实文字来满足公式。

形式上，当且仅当参数之一恰好是 `true` 时，exactly-1 3-SAT 问题的子句可以通过使用三元运算符 `true` 来定义。然而，此 kata 对这些子句使用与规范 SAT 问题相同的公式符号。
