# 评估 SAT 公式

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。
3. 二维元组数组 `formula`，描述 SAT 问题实例 $f(x)$。
   `formula` 的第 $i$ 元素是一个元组数组，使用与上一个练习相同的格式描述 $f(x)$ 的第 $i$ 子句。

例如，评估两个输入 $f(x) = (x_0 \vee x_1) \wedge (\neg x_0 \vee \neg x_1)$ 的 XOR 的二变量 SAT 公式可以表示为 `[[(0, true), (1, true)], [(0, false), (1, false)]]`。

**目标：**
实现一个计算公式 $f(x)$ 的量子预言机。
    
让输入寄存器中的量子位保持其开始时的状态。
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

此任务由多个子句评估结果的合取 (AND) 组成。可以使用您在上一个练习中编写的代码单独评估每个子句。这些子句的计算结果必须临时存储在新分配的量子位中。然后可以使用第一个练习中的 `Oracle_And` 计算这些结果的合取。

我们将公式中的子句数量表示为 $m$。实施 SAT 预言机的步骤如下：

1. 在状态 $\ket{0}$ 中分配 $m$ 量子位 `aux` 的数组。
2. 使用上一个练习中的 `Oracle_SATClause` 评估每个子句，并将 `aux` 的相应元素作为目标量子位。
3. 使用第一个任务中实现的 `Oracle_And` 评估 SAT 公式，其中 `aux` 作为输入寄存器，`target` 作为目标量子位。
4. 撤消步骤 2，将辅助量子位恢复到 $\ket{0}$ 状态，然后再释放它们。

您可以再次使用应用内 Q# 语言构造来执行步骤 2、3 和 4，最后一步将自动生成。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Arrays.*;

    operation Oracle_SATFormula(x : Qubit[], y : Qubit, formula : (Int, Bool)[][]) : Unit is Adj + Ctl {
        use aux = Qubit[Length(formula)];
        within {
            for i in 0..Length(formula) - 1 {
                Oracle_SATClause(x, aux[i], formula[i]);
            }
        } apply {
            Oracle_And(aux, y);
        }
    }

    operation Oracle_SATClause(x : Qubit[], y : Qubit, clause : (Int, Bool)[]) : Unit is Adj + Ctl {
        let clauseQubits = Mapped((ind, _) -> x[ind], clause);
        within {
            for (ind, positive) in clause {
                if not positive {
                    X(x[ind]);
                }
            }
        } apply {
            Oracle_Or(clauseQubits, y);
        }
    }

    operation Oracle_Or(x : Qubit[], y : Qubit) : Unit is Adj + Ctl {
        ApplyControlledOnInt(0, X, x, y);
        X(y);
    }

    operation Oracle_And(x : Qubit[], y : Qubit) : Unit is Adj + Ctl {
        Controlled X(x, y);
    }
}
```
