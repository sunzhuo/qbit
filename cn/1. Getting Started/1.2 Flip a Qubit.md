# 翻转一个量子位

练习是您必须通过编写 Q# 代码来解决的问题。在每个练习中，您都会看到占位符代码，您必须填写正确的解决方案。

**输入：** 处于 $\ket{0}$ 状态的量子位。

**目标：** 将 X 门应用于量子位，即执行“位翻转”，将量子位设置为 $\ket{1}$ 状态。

## 解决方案注释

成功实施解决方案或遇到问题时，您可以查阅参考解决方案，其中包含有关如何解决练习的更多详细信息。

对于本练习，您只需对输入量子位使用 Q# 的 `X` 操作即可解决问题，该操作将执行“位翻转”并将量子位设置为 $\ket{1}$ 状态。

来源：Solution.qs

```qsharp
namespace Kata {
    operation FlipQubit(q : Qubit) : Unit is Adj + Ctl {
        // Perform a "bit flip" on the qubit by applying the X gate.
        X(q);
    }
}
```
