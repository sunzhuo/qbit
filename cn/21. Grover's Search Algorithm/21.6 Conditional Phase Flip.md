# 条件相位翻转

**输入：** 处于任意状态的 $N$ 量子位的寄存器。

**目标：** 如果寄存器状态*不*处于 $\ket{0...0}$ 状态，则翻转寄存器状态的符号。
也就是说，如果寄存器处于状态$\ket{0...0}$，则保持不变，
但如果它处于任何其他基本状态，请将其相位乘以 $-1$。

> 该操作实现了运算符 
> $$2\ket{0...0}\bra{0...0} - I = 
\begin{bmatrix}
  1 & 0 & ... & 0 \\
  0 &-1 & ... & 0 \\
\vdots & \vdots & \ddots & \vdots \\
  0 & 0 & ... & -1
\end{bmatrix}$$

## 解决方案注释

此操作相当于仅翻转 $\ket{0...0}$ 基本状态的符号以及附加全局相位 $-1$ 的操作。
这意味着您可以分两步实施：

1. 翻转 $\ket{0...0}$ 基本状态的符号。
   您可以通过将 $X$ 门应用于所有量子位，然后使用受控 $Z$ 门翻转 $\ket{1...1}$ 的符号，然后再次将 $X$ 门应用于所有量子位来实现此目的。
2. 使用 `R1` 门将全局阶段 $-1$ 应用于整个状态。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;
    operation ConditionalPhaseFlip(qs : Qubit[]) : Unit is Adj + Ctl {
        within {
            ApplyToEachA(X, qs);
        } apply {
            Controlled Z(qs[1...], qs[0]);
        }
        R(PauliI, 2.0 * PI(), qs[0]);
    }
}
```
