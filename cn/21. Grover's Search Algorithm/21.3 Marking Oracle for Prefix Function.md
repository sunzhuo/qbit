# 将 Oracle 标记为前缀函数

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。
3.长度为$P$的位模式$p$表示为`Bool[]`($1 ≤ P ≤ N$)。

**目标：** 
如果模式 $p$ 是输入寄存器 $\ket{x}$ 的前缀，即对于 $0$ 和 $P - 1$ 之间的所有 $j$（包括 $p_j = x_j$），则实现一个标记预言机，该预言机翻转 $\ket{y}$ 的状态。 （`false` 和 `true` 值分别表示状态 $\ket{0}$ 和 $\ket{1}$）。

例如，对于 $N = 3$，两个位串以前缀 `[true, false]` 开头：`[true, false, false]` 和 `[true, false, true]`。

让输入寄存器中的量子位保持其开始时的状态。
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

在此问题中，我们正在评估的函数的值不取决于量子位 $P$ 之后的量子位的状态。这意味着我们可以忽略它们并只考虑重要的量子位 - `x[... Length(p) - 1]`。

一旦我们这样做了，问题就变得更加简单：如果输入量子位处于给定状态，则翻转目标量子位的状态。这就是具有任意控件的受控门的定义，我们可以使用库操作 `ApplyControlledOnBitString` 轻松应用它。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Oracle_StartsWith(x : Qubit[], y : Qubit, p : Bool[]) : Unit is Adj + Ctl {
        ApplyControlledOnBitString(p, X, x[... Length(p) - 1], y);
    }
}
```
