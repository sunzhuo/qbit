# 标记 Oracle 中的阶段 Oracle

**输入：**

1. 标记预言机实现未知的 $N$ 位函数 $f(x)$。
  2. 处于任意状态的 $N$ 量子位（输入/查询寄存器）。
  
**目标：**

翻转 $f(x) = 1$ 的每个基本状态 $\ket{x}$ 的相位。您只能通过为您提供的标记预言机访问 $f(x)$。

您实现的操作是同一功能的阶段预言。

<br/>
<details>
  <summary><b>需要提示吗？</b></summary>
    请记住，您可以分配额外的量子位来协助此操作。  是否有一种状态可以用辅助量子位来准备，以帮助您将标记预言机转换为相位预言机？
</details>

## 解决方案注释

正如我们在 Oracle kata 中看到的，我们可以在 $\ket{-}$ 状态中分配一个额外的量子位，并将其用作标记预言的目标。
这将回弹 $f(x) = 1$ 输入寄存器的基本状态 $\ket{x}$ 的 $-1$ 相对相位。

来源：Solution.qs

```qsharp
namespace Kata {
    operation ApplyMarkingOracleAsPhaseOracle(
        markingOracle : (Qubit[], Qubit) => Unit is Adj + Ctl,
        qubits : Qubit[])
    : Unit is Adj + Ctl {
        use minus = Qubit();
        within {
            X(minus);
            H(minus);
        } apply {
            markingOracle(qubits, minus);
        }
    }
}
```
