# 对任意状态的反思

**输入：**

1. $N$ 量子位处于任意状态。
2.从状态$\ket{0...0}$准备$N$-量子位状态$\ket{\psi}$的操作$P$。
  
**目标：**

反映给定量子位关于状态 $\ket{\psi}$ 的状态。
换句话说，实现操作$2\ket{\psi}\bra{\psi} - I$。

如果您使用准备均值作为输入的操作，则此操作允许您实现“关于均值的反映”操作。

## 解决方案注释

您知道$P\ket{0...0} = \ket{\psi}$。
您可以使用这个和 $I = PP^\dagger$ 的事实来编写您需要实现的操作的以下表示：

$$2\ket{\psi}\bra{\psi} - I = 2P\ket{0...0}\bra{0...}P^\dagger - I =$$
$$= 2P\ket{0...0}\bra{0...0}P^\dagger - PP^\dagger = P(2\ket{0...0}\bra{0...0} - I)P^\dagger$$

中间的操作正是您在上一个练习中实现的条件相位翻转。
整体解决方案是：

1. 应用给定运算 $P$ 的伴随。
2. 应用上一个练习中的条件相位翻转。
3. 应用给定操作$P$。

namespace Kata {
    import Std.Math.*;

    operation ReflectionAboutState(
        qs : Qubit[],
        statePrep : Qubit[] => Unit is Adj + Ctl
    ) : Unit is Adj + Ctl {
        within {
            Adjoint statePrep(qs);
        } apply {
            ConditionalPhaseFlip(qs);
        }
    }

    operation ConditionalPhaseFlip(qs : Qubit[]) : Unit is Adj + Ctl {
        within {
            ApplyToEachA(X, qs);
        } apply {
            Controlled Z(qs[1...], qs[0]);
        }
        R(PauliI, 2.0 * PI(), qs[0]);
    }
}
