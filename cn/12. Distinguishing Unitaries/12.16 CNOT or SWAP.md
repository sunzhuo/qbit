# CNOT 还是交换？

**输入：** 实现两个量子位酉变换的操作：
$CNOT$ 门，以第一个量子位作为控制，以第二个量子位作为目标 ($CNOT_{12}$)
或 $SWAP$ 门。

**输出：** 如果给定操作为 $CNOT_{12}$，则为 0；如果给定操作为 $SWAP$，则为 1。

该操作将接受一组量子位作为输入，但如果该数组为空或具有一个或两个以上量子位，则该操作将会失败。
该操作将定义伴随变量和受控变量。
您可以精确**一次**应用给定操作及其伴随/受控变体。

## 解决方案注释

与上一个任务相同，将这两个门应用于除 $\ket{00}$ 之外的任何基本状态都会产生不同的结果，我们可以使用这些状态中的任何一个来区分幺正。

最简单的解决方案是：准备两个处于 $\ket{01}$ 状态的量子位并对它们应用酉，然后测量第一个量子位；如果仍然是`Zero`，则门是$CNOT_{12}$，否则是$SWAP$。请记住，这次第二个量子位可能最终处于 $\ket{1}$ 状态，因此在释放它之前需要将其重置为 $\ket{0}$。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DistinguishCNOTfromSWAP (unitary : (Qubit[] => Unit is Adj + Ctl)) : Int {
        use qs = Qubit[2];
        X(qs[1]);
        unitary(qs);
        Reset(qs[1]);
        return MResetZ(qs[0]) == Zero ? 0 | 1;
    }
}
```
