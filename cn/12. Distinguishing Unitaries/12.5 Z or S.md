#Z还是S？

**输入：** 实现单量子位酉变换的操作：
$Z$ 门或 $S$ 门。

**输出：** 如果给定操作是 $Z$ 门，则为 0；如果给定操作是 $S$ 门，则为 1。

该操作将定义伴随变量和受控变量。
您最多可以应用给定操作及其伴随/受控变体**两次**。

## 解决方案注释

> 提醒一下，$$S = \begin{bmatrix} 1 & 0 \\ 0 & i \end{bmatrix}$$

此任务与前两个任务的不同之处在于，它允许您应用给定的单一**两次**。 
让我们将其视为一个提示，并检查给定的门在应用两次时的外观。 
如果你对相应的矩阵进行平方（这对于对角矩阵来说非常简单），你会得到

$$Z^2 = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix} = I, S^2 = \begin{bmatrix} 1 & 0 \\ 0 & -1 \end{bmatrix} = Z$$

这意味着识别给定酉变换的*平方*的任务与区分 $I$ 和 $Z$ 门相同 - 这正是前面的任务。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DistinguishZfromS(unitary : (Qubit => Unit is Adj + Ctl)) : Int {
        use q = Qubit();
        H(q);
        unitary(q);
        unitary(q);
        return MResetX(q) == Zero ? 0 | 1;
    }
}
```
