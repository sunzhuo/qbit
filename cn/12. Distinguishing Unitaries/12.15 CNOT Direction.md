# CNOT 方向

**输入：** 实现两个量子位酉变换的操作：
$CNOT$ 门，以第一个量子位作为控制，以第二个量子位作为目标 ($CNOT_{12}$)
或 $CNOT$ 门，以第二个量子位作为控制，第一个量子位作为目标 ($CNOT_{21}$)。

**输出：** 如果给定操作为 $CNOT_{12}$，则为 0；如果给定操作为 $CNOT_{21}$，则为 1。

该操作将接受一组量子位作为输入，但如果该数组为空或具有一个或两个以上量子位，则该操作将会失败。
该操作将定义伴随变量和受控变量。
您可以精确**一次**应用给定操作及其伴随/受控变体。

## 解决方案注释

再次，让我们考虑这些门对基础状态的影响：

我们可以看到，将这两个门应用于除 $\ket{00}$ 之外的任何基本状态都会产生不同的结果，并且我们可以使用这些状态中的任何一个来区分幺正。

因此，最简单的解决方案是：准备两个处于 $\ket{01}$ 状态的量子位并对它们应用酉，然后测量第一个量子位；如果仍然是`Zero`，则门是$CNOT_{12}$，否则是$CNOT_{21}$。

来源：Solution.qs

```qsharp
namespace Kata {
    operation CNOTDirection (unitary : (Qubit[] => Unit is Adj + Ctl)) : Int {
        use qs = Qubit[2];
        within { X(qs[1]); }
        apply { unitary(qs); }
        return MResetZ(qs[0]) == Zero ? 0 | 1;
    }
}
```
