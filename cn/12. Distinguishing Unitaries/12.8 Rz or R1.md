# Rz 还是 R1？

**输入：** 实现单量子位酉变换的操作：
[$R_z$ 门](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.intrinsic/rz) 或 [$R1$ 门](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.intrinsic/r1)。

此操作将采用两个参数：第一个参数是以弧度为单位的旋转角度，第二个参数是应应用门的量子位（与 Q# 中的正常 `Rz` 和 `R1` 门匹配）。

> 提醒一下， $$R_z(\theta) = \begin{bmatrix} e^{-i\theta/2} & 0 \\ 0 & e^{i\theta/2} \end{bmatrix} \text{, } 
R_1(\theta) = \begin{bmatrix} 1 & 0 \\ 0 & e^{i\theta} \end{bmatrix} = e^{i\theta/2} R_z(\theta)$$

**输出：** 如果给定操作是 $R_z$ 门，则为 0；如果给定操作是 $R1$ 门，则为 1。

该操作将定义伴随变量和受控变量。
您可以精确**一次**应用给定操作及其伴随/受控变体。

## 解决方案注释

我们看到这两个门的不同之处在于全局相位$e^{i\theta/2}$。
在这个问题中，我们可以自由选择要传递给门的角度参数，因此我们可以选择一个使全局相位差易于检测的角度：对于 $\theta = 2\pi$ $e^{i\theta/2} = -1$，$R_z(\theta) = -I$ 和 $R_1(\theta) = I$。

现在我们需要区分 $I$ 门和 $-I$ 门，这可以使用门的受控变体来完成，其方式与在任务“Z 或 -Z”中区分 $Z$ 门和 $-Z$ 门完全相同。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;
    operation DistinguishRzFromR1(unitary : ((Double, Qubit) => Unit is Adj + Ctl)) : Int {
        use qs = Qubit[2];
        H(qs[0]);
        Controlled unitary(qs[0..0], (2.0 * PI(), qs[1]));
        return MResetX(qs[0]) == Zero ? 1 | 0;
    }
}
```
