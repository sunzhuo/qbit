# 身份还是X？

**输入：** 实现单量子位酉变换的操作：
身份 $I$ 或泡利 $X$ 门。

**输出：** 如果给定操作是 $I$ 门，则为 0；如果给定操作是 $X$ 门，则为 1。

该操作将定义伴随变量和受控变量。
您可以准确**一次**应用给定操作及其伴随/受控变体。

## 解决方案注释

从量子系统中提取信息的唯一方法是测量。 
测量为我们提供了有关系统状态的信息，因此要获取有关门的信息，我们需要找到一种方法将其转换为有关状态的信息。
如果我们想区分两个门，我们需要弄清楚准备一个状态并对其进行测量，这将给我们一个可以解释的结果。
为此，我们需要找到一个量子位状态，通过对其应用 $I$ 门或 $X$ 门，该状态将转换为可以使用测量来区分的状态，即正交状态。 
让我们找到这样的状态。

> 作为提醒，以下是与给定门对应的矩阵：
> $$I = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix}, X = \begin{bmatrix} 0 & 1 \\ 1 & 0 \end{bmatrix}$$

考虑这些门对基本状态 $\ket{0}$ 的影响。

$$I\ket{0} = \ket{0}$$
$$X\ket{0} = \ket{1}$$

我们看到 $I$ 门保持 $\ket{0}$ 状态不变，而 $X$ 门将其转换为 $\ket{1}$ 状态。 
因此，最简单的事情是准备一个 $\ket{0}$ 状态，将给定的酉应用于它，并在计算基础上测量结果状态：
* 如果测量结果是`Zero`，则测量的状态是$\ket{0}$，并且我们知道应用于它的酉是$I$门。
* 如果测量结果是`One`，则测量的状态是$\ket{1}$，并且我们知道应用于它的酉是$X$门。

> 在 Q# 中，新分配的量子位以 $\ket{0}$ 状态开始，因此在对其应用酉之前，您无需执行任何操作来准备必要的状态。
> 您还必须在释放之前将分配给 $\ket{0}$ 状态的量子位返回。 
> 您可以通过使用 `M` 操作测量量子位并应用 $X$ 门（如果在 $\ket{1}$ 状态下测量）来实现这一点，或者您可以使用 [`MResetZ`](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.measurement/mresetz) 操作将此测量和量子位重置合并为一个操作。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DistinguishIfromX(unitary : (Qubit => Unit is Adj + Ctl)) : Int {
        use q = Qubit();
        unitary(q);
        return MResetZ(q) == Zero ? 0 | 1;
    }
}
```
