# 区分四个泡利酉

**输入：** 实现单量子位酉变换的操作：
恒等门（$I$ 门）或泡利门之一（$X$、$Y$ 或 $Z$ 门）。

**输出：** 
* 0 如果给定操作是 $I$ 门，
* 1 如果给定操作是 $X$ 门，
* 2 如果给定操作是 $Y$ 门，
* 3 如果给定操作是 $Z$ 门。

该操作将定义伴随变量和受控变量。
您可以精确**一次**应用给定操作及其伴随/受控变体。

## 解决方案注释

此任务与本节前面的任务有很大不同； 
乍一看，似乎不可能仅用一个单一应用程序来区分四个不同的单一应用程序（即获取两位信息）！

然而，由于酉式是经过仔细选择的（并且您可以进行的测量次数不受限制），因此这是可能的。 
该解决方案使用贝尔状态：四个正交状态，您可以通过从第一个状态 $\frac{1}{\sqrt2}(\ket{00} + \ket{11})$ 开始，并将门 $I$、$X$、$Z$ 和 $Y$ 分别应用于第一个量子位来准备。 
因此，解决方案变为：准备 $\frac{1}{\sqrt2}(\ket{00} + \ket{11})$ 状态，应用酉并测量贝尔基中的结果状态，以找出它是哪个贝尔状态。有关如何做到这一点的详细信息，请参阅区分量子态类型，任务“区分四个贝尔态”。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DistinguishPaulis (unitary : (Qubit => Unit is Adj + Ctl)) : Int {
        // apply operation to the 1st qubit of a Bell state and measure in Bell basis
        use qs = Qubit[2];
        within {
            H(qs[0]);
            CNOT(qs[0], qs[1]);
        } apply {
            unitary(qs[0]);
        }

        // after this I -> 00, X -> 01, Y -> 11, Z -> 10
        let ind = MeasureInteger(qs);
        let returnValues = [0, 3, 1, 2];
        return returnValues[ind];
    }
}
```
