# Y 或 XZ 或 -Y 或 -XZ？

**输入：** 实现单量子位酉变换的操作：
$Y$ 门（可能具有 $-1$ 的额外全局相位）或 Pauli $Z$ 和 Pauli $X$ 门序列（可能具有 $-1$ 的额外全局相位）。

**输出：** 
* 0 如果给定操作是 $Y$ 门，
* 1 如果给定操作是 $-XZ$ 门，
* 2 如果给定操作是 $-Y$ 门，
* 3 如果给定操作是 $XZ$ 门。

该操作将定义伴随变量和受控变量。
您最多可以应用给定操作及其伴随/受控变体**三次**。

## 解决方案注释

在此任务中，我们必须区分在全局阶段中相同的 4 个门，即门 $Y$、$iY$、$-Y$ 和 $-iY$（按顺序）。

执行此操作的一种方法是“手动”，类似于前面的任务。首先，我们将应用酉的受控变体两次，并检查结果是 $I$ 还是 $-I$（这使我们能够区分 $\pm Y$ 和 $\pm iY$，因为前两个门的平方将等于 $I$ 门，后两个等于 $-I$门）。然后，我们通过应用酉一次的受控变体来区分每组中的门（$Y$ 与 $-Y$ 或 $iY$ 与 $-iY$）。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DistinguishYfromXZWithPhases (unitary : (Qubit => Unit is Adj+Ctl)) : Int {
        use (control, target) = (Qubit(), Qubit());
        // Distinguish Y from iY
        within {
            H(control);
        } apply {
            Controlled unitary([control], target);
            Controlled unitary([control], target);
        }
        Reset(target);
        let isY = MResetZ(control) == Zero;

        // Distinguish Y from -Y and iY from -iY
        within {
            H(control);
        } apply {
            Controlled unitary([control], target);
            // apply controlled variant of the gate we're expecting to compensate effect on target qubit
            if isY {
                CY(control, target);
            } else {
                CZ(control, target);
                CX(control, target);
            }
        }
        let result = isY ? M(control) == Zero ? 0 | 2 
                         | M(control) == Zero ? 1 | 3;
        Reset(control);
        return result;
        
    }
}
```
