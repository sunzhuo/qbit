# 身份还是Z？

**输入：** 实现单量子位酉变换的操作：
身份 $I$ 或泡利 $Z$ 门。

**输出：** 如果给定操作是 $I$ 门，则为 0；如果给定操作是 $Z$ 门，则为 1。

该操作将定义伴随变量和受控变量。
您可以精确**一次**应用给定操作及其伴随/受控变体。

## 解决方案注释

> 提醒一下，$$Z = \begin{bmatrix} 1 & 0 \\ 0 & -1 \end{bmatrix}$$

我们无法通过将 $I$ 与 $Z$ 应用于基础状态来区分它们，因为它们都保持 $\ket{0}$ 状态不变，并向 $\ket{1}$ 状态添加一个相位：

$$I\ket{0} = \ket{0}, I\ket{1} = \ket{1}$$
$$Z\ket{0} = \ket{0}, Z\ket{1} = -\ket{1}$$

然而，如果我们尝试将这些门应用于基础状态的叠加，我们将开始看到结果状态之间的差异：

$$I \big(\frac{1}{\sqrt2}(\ket{0} + \ket{1})\big) = \frac{1}{\sqrt2}(\ket{0} + \ket{1})$$
$$Z \big(\frac{1}{\sqrt2}(\ket{0} + \ket{1})\big) = \frac{1}{\sqrt2}(\ket{0} - \ket{1})$$

这两个状态是正交的，可以通过使用 [`MResetX`](https://learn.microsoft.com/en-us/qsharp/api/qsharp-lang/microsoft.quantum.measurement/mresetx) 操作在 $\{ \ket{+}, \ket{-}\}$ 基础上测量它们来区分（相当于应用 $H$ 门并在计算基础上测量）。

> 区分这两种状态的任务在区分量子态 kata 中有更详细的介绍。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DistinguishIfromZ(unitary : (Qubit => Unit is Adj + Ctl)) : Int {
        use q = Qubit();
        H(q);
        unitary(q);
        return MResetX(q) == Zero ? 0 | 1;
    }
}
```
