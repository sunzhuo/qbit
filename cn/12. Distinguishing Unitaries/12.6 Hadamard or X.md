# 哈达玛还是 X？

**输入：** 实现单量子位酉变换的操作：
$H$ 门或 $X$ 门。

**输出：** 如果给定操作是 $H$ 门，则为 0；如果给定操作是 $X$ 门，则为 1。

该操作将定义伴随变量和受控变量。
您最多可以应用给定操作及其伴随/受控变体**两次**。

## 解决方案注释

> 提醒一下，$$H = \frac{1}{\sqrt2} \begin{bmatrix} 1 & 1 \\ 1 & -1 \end{bmatrix}$$

该解决方案依赖于众所周知的恒等式 $HXH = Z$（即哈达玛门可用于将 $X$ 转换为 $Z$，反之亦然）。如果单一元是 $X$，应用“给定单一元 - $X$ - 给定单一元”门序列将相当于 $XXX = X$ 门，如果单一元是 $H$，则相当于 $HXH = Z$。通过这一观察，我们需要区分 $X$ 和 $Z$，这可以使用 $\ket{0}$ 状态来完成：$X$ 门将其更改为 $\ket{1}$，而 $Z$ 门将保持其不变。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DistinguishHfromX(unitary : (Qubit => Unit is Adj + Ctl)) : Int {
        use q = Qubit();
        within {
            unitary(q);
        } apply {
            X(q);
        }
        return MResetZ(q) == Zero ? 0 | 1;
    }
}
```
