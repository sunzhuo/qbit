# 区分两个量子位酉

**输入：** 实现两个量子位酉变换的操作：
身份 ($I \otimes I$)、$CNOT$ 门（其中一个量子位作为控制，另一个量子位作为目标）或 $SWAP$ 门。

**输出：**

* 0 如果给定操作是 $I \otimes I$， 
* 1 如果给定操作是$CNOT_{12}$，
* 2 如果给定操作是$CNOT_{21}$，
* 3 如果给定操作是 $SWAP$。

该操作将接受一组量子位作为输入，但如果该数组为空或具有一个或两个以上量子位，则该操作将会失败。
该操作将定义伴随变量和受控变量。
您最多可以应用给定操作及其伴随/受控变体**两次**。

## 解决方案注释

在这个问题中，我们可以使用给定的酉两次，因此我们可以将决策过程分为两个阶段：

1. 将酉应用到$\ket{11}$状态； $CNOT_{12}$ 将产生 $\ket{10}$ 状态，$CNOT_{21}$ - $\ket{01}$，$I \otimes I$ 和 $SWAP$ 门都将保持状态不变。
2. 现在为了区分 $I \otimes I$ 和 $SWAP$，我们可以使用 $\ket{01}$ 状态：$I \otimes I$ 门将保持其不变，而 $SWAP$ 将产生 $\ket{10}$。

库操作`MeasureInteger`测量数组的所有量子位，将它们重置为$\ket{0}$，并返回测量结果，使用little-endian将位数组转换为整数。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DistinguishTwoQubitUnitaries (unitary : (Qubit[] => Unit is Adj + Ctl)) : Int {
        // First run: apply to |11⟩;
        // CNOT₁₂ will give |10⟩, CNOT₂₁ will give |01⟩, I⊗I and SWAP will remain |11⟩
        use qs = Qubit[2];
        ApplyToEach(X, qs);
        unitary(qs);
        let ind1 = MeasureInteger(qs);

        // Second run: to distinguish I⊗I from SWAP, apply to |01⟩: 
        // I⊗I will remain |01⟩, SWAP will become |10⟩
        X(qs[1]);
        unitary(qs);
        let ind2 = MeasureInteger(qs);

        if ind1 == 1 or ind1 == 2 {
            // Respective CNOT
            return ind1;
        } else {
            return ind2 == 1 ? 3 | 0;
        }
    }
}
```
