# Y 还是 XZ？

**输入：** 实现单量子位酉变换的操作：
$Y$ 门或 Pauli $Z$ 和 Pauli $X$ 门序列（相当于应用 $Z$ 门，然后应用 $X$ 门）。

**输出：** 如果给定操作是 $Y$ 门，则为 0；如果给定操作是 $XZ$ 门，则为 1。

该操作将定义伴随变量和受控变量。
您最多可以应用给定操作及其伴随/受控变体**两次**。

## 解决方案注释

> 提醒一下，$$Y = \begin{bmatrix} 0 & -i \\ i & 0 \end{bmatrix} = iXZ$$

我们看到这两个门的不同之处在于全局相位$i = e^{i\pi}$。应用门两次将分别得到门 $Y^2 = I$ 和 $(XZ)^2 = XZXZ = -XXZZ = -I$。

现在我们需要区分 $I$ 门和 $-I$ 门，这与我们在上一个任务中所做的相同。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DistinguishYfromXZ (unitary : (Qubit => Unit is Adj + Ctl)) : Int {
        use qs = Qubit[2];
        H(qs[0]);
        Controlled unitary(qs[0..0], qs[1]);
        Controlled unitary(qs[0..0], qs[1]);
        return MResetX(qs[0]) == Zero ? 0 | 1;
    }
}
```
