# Z还是-Z？

**输入：** 实现单量子位酉变换的操作：
$Z$ 门或负 $Z$ 门（即门 $-\ket{0}\bra{0} + \ket{1}\bra{1}$）。

**输出：** 如果给定操作是 $Z$ 门，则为 0；如果给定操作是 $-Z$ 门，则为 1。

该操作将定义伴随变量和受控变量。
您可以精确**一次**应用给定操作及其伴随/受控变体。

## 解决方案注释

这个任务更有趣：给定的门因它们引入的全局阶段而不同（即，它们中的每一个都是另一个的倍数），并且将它们应用于任何单量子位状态的结果将通过您可以设计的任何测量来区分。

幸运的是，我们不仅得到了酉本身，还得到了它的受控变体，即如果控制量子位处于 $\ket{1}$ 状态，则应用给定酉的门，如果控制量子位处于 $\ket{0}$ 状态，则不执行任何操作。
这使我们能够使用所谓的“相位反冲”技巧，其中应用门的受控版本使我们能够观察该门在控制量子位上引入的相位。的确，

我们看到，两个受控门都不会修改控制量子位处于 $\ket{0}$ 状态的状态，但如果控制量子位处于 $\ket{1}$ 状态，它们会向不同的基础状态引入 $-1$ 阶段。 
如果我们将受控门应用于*控制量子位*处于叠加状态，例如$\frac{1}{\sqrt2}(\ket{0} + \ket{1}) \otimes \ket{0}$，我们可以利用这一点：

$$\text{Controlled Z}\frac{1}{\sqrt2}(\ket{0} + \ket{1}) \otimes \ket{0} = \frac{1}{\sqrt2}(\ket{0} + \ket{1}) \otimes \ket{0}$$
$$\text{Controlled }-\text{Z}\frac{1}{\sqrt2}(\ket{0} + \ket{1}) \otimes \ket{0} = \frac{1}{\sqrt2}(\ket{0} - \ket{1}) \otimes \ket{0}$$

之后，我们可以测量第一个量子位以区分 $\frac{1}{\sqrt2}(\ket{0} + \ket{1})$ 和 $\frac{1}{\sqrt2}(\ket{0} - \ket{1})$，就像我们在任务“Identity or Z”中所做的那样。

> 在 Q# 中，我们可以使用 [Controlled functor](https://learn.microsoft.com/en-us/azure/quantum/user-guide/language/expressions/functorapplication#controlled-functor) 表达门的受控版本：结果门的第一个参数将是控制量子位数组，第二个参数是原始门的参数（在本例中只是目标量子位）。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DistinguishZfromMinusZ (unitary : (Qubit => Unit is Adj + Ctl)) : Int {
        use qs = Qubit[2];
        H(qs[0]);
        Controlled unitary(qs[0..0], qs[1]);
        return MResetX(qs[0]) == Zero ? 0 | 1;
    } 
}
```
