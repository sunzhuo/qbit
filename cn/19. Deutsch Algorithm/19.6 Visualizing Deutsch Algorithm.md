# 可视化 Deutsch 算法

You can follow the steps of the algorithm for the constant and the balanced scenarios using a neat visualization.由于 Deutsch 算法仅处理具有实际振幅的状态，因此您可以将所有状态映射到单位圆上，并通过步骤跟踪状态演化。

1. 从 $\ket{0}$ 状态的量子位开始，并将 $H$ 门应用于该量子位。
   <br/>
   ![Plus state](./media/Plus_state.svg)

2.应用神谕。  
   在这里，两种情况之间的差异变得显而易见。在恒定场景中，$\ket{0}$ 和 $\ket{1}$ 状态获得相同的相位（$1$ 或 $-1$），因此状态保持不变或获得 $-1$ 的全局相位，这在物理上是相同的状态。在可变场景中，零和一状态得到不同的阶段，因此状态发生变化！
   <br/>
   ![Apply oracle](./media/Apply_oracle.svg)

3. Apply the $H$ gate to the qubit again.
   现在，您将获得两个恒定场景的 $\ket{0}$ 状态和两个可变场景的 $\ket{1}$ 状态！
   <br/>
   ![Apply hadamard](./media/Apply_hadamard.svg)
