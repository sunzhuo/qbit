# f(x) = 1 - x 的预言机

**输入：** 处于任意状态的量子位 $\ket{\psi} = \alpha\ket{0} + \beta\ket{1}$。

**目标：** 将 $f(x) = 1 - x$ 的相位预言 $U_f$ 应用到量子位。
也就是说，将相对相位$(-1)^{f(x)}$应用于每个基础状态$\ket{x}$。

<details>
<summary><strong>需要提示吗？</strong></summary>
您可以将预言机的效果表示为

$$U_f \ket{x} = (-1)^{1-x} \ket{x} = (-1) \cdot (-1)^x \ket{x}$$

结合之前的一些oracle实现能达到这样的效果吗？
</details>

## 解决方案注释

您可以将预言机的效果表示为

$$U_f \ket{x} = (-1)^{1-x} \ket{x} = (-1) \cdot (-1)^x \ket{x}$$

这种效果可以通过连续应用函数 $f(x) = 1$ 和 $f(x) = x$ 的两个预言机的组合来实现。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    operation PhaseOracle_OneMinusX(x : Qubit) : Unit is Adj + Ctl {
        Z(x);
        R(PauliI, 2.0 * PI(), x);
    }
}
```
