# 单量子位预言机

在量子场景中，您正在使用的经典函数被实现为量子预言机 - 用作另一个算法的输入的“黑匣子”操作。此操作的实现方式不仅允许对单个输入执行计算，而且还允许对输入的叠加执行计算。

预言机必须作用于量子态而不是经典值。 
为了实现这一点，整数输入 $x$ 表示为量子位状态 $\ket{x}$。

本教程中使用的预言机类型称为*阶段预言机*。相位预言机 $U_f$ 对它在量子位状态的相位中实现的经典函数 $f$ 的值进行编码，如下所示：

$$U_f \ket{x} = (-1)^{f(x)} \ket{x}$$

在这种情况下，$f$ 只能返回两个值，0 或 1，这分别导致没有相位变化或乘以相对相位 $-1$。

这种预言机对任何单一基础状态的影响并不是特别有趣：它只是添加了一个全局阶段，而这是您无法观察到的。然而，如果你将这个预言应用到基础状态的“叠加”上，它的效果就会变得显而易见。 
请记住，量子操作是线性的：如果您定义操作对基本状态的影响，您将能够使用其线性推断出其对叠加态（只是基本状态的线性组合）的影响。

只有四个单位函数，因此您可以了解如何在 Q# 中将它们全部实现为阶段预言机。

1.$f(x) \equiv 0$

这是最容易实现的函数：if $f(x) \equiv 0$,

$$U_f \ket{x} \equiv (-1)^0 \ket{x} = \ket{x}$$

这意味着 $U_f$ 是一个恒等式 - 一个绝对不执行任何操作的转换！

2.$f(x) \equiv 1$

第二个常量函数稍微棘手一些： if $f(x) \equiv 1$

$$U_f \ket{x} \equiv (-1)^1 \ket{x} = - \ket{x}$$

现在 $U_f$ 是一个负恒等式，即将 $-1$ 的全局相位应用于状态的变换。 
许多算法只是忽略其中累积的全局相位，因为它是不可观察的。 
但是，如果您想要真正细致，您可以使用 $R$ 门，它围绕给定轴执行给定旋转。 
当使用 `PauliI` 轴调用时，此操作将全局相位应用于给定的量子位。

3.$f(x) = x$

$$U_f \ket{x} = (-1)^{f(x)} \ket{x} = (-1)^{x} \ket{x}$$

这意味着，如果量子位处于 $\ket{0}$ 状态，则无需执行任何操作；如果量子位处于 $\ket{1}$ 状态，则应用 $-1$ 阶段。这正是 $Z$ 门的效果！

在下面的演示中，您将了解如何将前三个一位函数实现为量子预言机，以及它们对量子位状态的影响。
之后，您将尝试自己实现第四个功能的预言机！

来源：OracleImplementationDemo.qs

```qsharp
namespace Kata {
    import Std.Diagnostics.*;
    import Std.Math.*;

    operation PhaseOracle_Zero(x : Qubit) : Unit {
        // Do nothing...
    }

    operation PhaseOracle_One(x : Qubit) : Unit {
        // Apply a global phase of -1
        R(PauliI, 2.0 * PI(), x);
    }

    operation PhaseOracle_X(x : Qubit) : Unit {
        Z(x);
    }

    @EntryPoint()
    operation OracleImplementationDemo() : Unit {
        use q = Qubit();
        Ry(2.0 * ArcCos(0.6), q);
        Message("The qubit state before oracle application is 0.6|0⟩ + 0.8|0⟩:");
        DumpMachine();

        // Apply the oracle.
        // Experiment with using other oracles to see their behavior!
        // (Note that the -1 global phase might not show up in simulation)
        PhaseOracle_X(q);

        Message("The qubit state after oracle application:");
        DumpMachine();

        Reset(q);
    }
}
```
