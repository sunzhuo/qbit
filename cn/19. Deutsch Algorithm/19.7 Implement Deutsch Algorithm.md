# 实施 Deutsch 算法

**输入：** “黑匣子”预言机实现 $f(x)$，定义为采用单个量子位作为参数并返回 `Unit` 的操作。

**目标：** 如果函数是常量 ($f(0) = f(1)$)，则返回 `true`，如果函数是变量 ($f(0) \neq f(1)$)，则返回 `false`。
您只能使用一次 oracle 调用！

## 解决方案注释

遵循上一节中概述的算法：

1. 分配一个量子位 - 它从 $\ket{0}$ 状态开始。
2. 将 $H$ 门应用于量子位。
3.应用预言机。应用预言的语法与应用任何其他门或操作的语法相同。
4. 再次将 $H$ 门应用于量子位。
5. 测量量子位：如果测量结果为`Zero`，则函数为常数，否则为变量。
由于如果函数为常数则需要返回`true`，因此只需返回测量结果与`Zero`比较的结果即可。

来源：Solution.qs

```qsharp
namespace Kata {
    operation DeutschAlgorithm (oracle : Qubit => Unit) : Bool {
        use x = Qubit();
        H(x);
        oracle(x);
        H(x);
        return MResetZ(x) == Zero;
    }
}
```
