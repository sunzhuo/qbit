# 解决问题：Deutsch 算法

现在让我们回到确定给定函数对于单位函数是常量还是变量的问题。
如果我们有一个实现函数 $f(x)$ 的量子预言机 $U_f$，我们能做什么？

该函数有两个可能的输入：$\ket{0}$ 和 $\ket{1}$。让我们看看如果将预言应用到它们的叠加上会发生什么：

$$U_f \left( \frac{1}{\sqrt2} \big( \ket{0} + \ket{1} \big) \right) 
= \frac{1}{\sqrt2} \big( U_f \ket{0} + U_f \ket{1} \big) 
= \frac{1}{\sqrt2} \big( (-1)^{f(0)} \ket{0} + (-1)^{f(1)} \ket{1} \big)$$。

- 如果为$f(0) = f(1)$，则两个基本状态的相对相位相同，并且结果状态为$\ket{+} = \frac{1}{\sqrt2} \big( \ket{0} + \ket{1} \big)$（直到全局相位）。 
- 如果为$f(0) \neq f(1)$，则两个基本状态的相对相位相差$-1$倍，并且结果状态为$\ket{-} = \frac{1}{\sqrt2} \big( \ket{0} - \ket{1} \big)$（直到全局相位）。

现在，可以使用测量来区分状态 $\ket{+}$ 和 $\ket{-}$：如果将 H 门应用于每个状态，则如果 $f(0) = f(1)$ 则为 $H\ket{+} = \ket{0}$，如果 $f(0) \neq f(1)$ 则为 $H\ket{-} = \ket{1}$。这意味着一次 oracle 调用不会让您同时计算 $f(0)$ 和 $f(1)$，但它可以让您计算是否 $f(0) = f(1)$！

总的来说，该算法非常简单：

1. 从 $\ket{0}$ 状态的量子位开始。
2. 将 $H$ 门应用于量子位。
3.应用预言机。
4. 再次将 $H$ 门应用于量子位。
5. 测量量子位：如果处于$\ket{0}$状态，则函数为常数，否则为变量。

请注意，该算法仅需要 **1** 次 oracle 调用，并且始终产生正确的结果（该算法是确定性的）。
