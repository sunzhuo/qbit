# 极数相乘

**输入：**

1. 复极数 $x = r_{1}e^{i\theta_1}$
 其中 $r_{1} \geq 0$ 和 $-\pi < \theta_1 \leq \pi$。
2. 复极数 $y = r_{2}e^{i\theta_2}$
 其中 $r_{2} \geq 0$ 和 $-\pi < \theta_2 \leq \pi$。

**目标：**
返回 $x$ 和 $y$ 的乘积作为复极数 $x \cdot y = r_{3}e^{i\theta_3}$。

* $r_3$ 应该是非负数：$r_3 \geq 0$
* $\theta_3$ 应介于 $-\pi$ 和 $\pi$ 之间：$-\pi < \theta_3 \leq \pi$
* 尽量避免将数字转换为笛卡尔形式。

> `Std.Math` 命名空间中的 Q# 函数 `TimesCP` 将两个复数相乘，但不会规范化结果数的参数。出于教育目的，请尝试手动完成此任务。

## 解决方案注释

两个极坐标形式的复数相乘可以通过以下方式有效地完成：

$$ z = x \cdot y = r_{1}e^{\theta_1 i} \cdot r_{2}e^{\theta_2 i} = r_{1}r_{2} \cdot e^{(\theta_1 + \theta_2)i} $$  

> 这是将数字转换为笛卡尔坐标并在其中进行乘法的较长方法：  
> $$ x = r_{1}e^{i\theta_1} = r_{1}(\cos \theta_1 + i \sin \theta_1) $$
> $$ y = r_{2}e^{i\theta_2} = r_{2}(\cos \theta_2 + i \sin \theta_2) $$
> $$ z = x \cdot y = r_1r_2 \cdot \left( \cos \theta_1 \cos \theta_2 − \sin \theta_1 \sin \theta_2 + 
i (\sin \theta_1 \cos \theta_2 + \sin \theta_2 \cos \theta_1 ) \right) $$
>
> 您可以使用以下三角恒等式来简化此过程：
> * $\cos a \cos b  \mp \sin a \sin b = \cos(a \pm b)$
> * $\sin a \cos b  \pm \sin b \cos a = \sin(a \pm b)$
>
> 最后，这个解决方案给出了与上面简短的解决方案相同的答案：
>$$r_{1}r_{2}(\cos(\theta_1 + \theta_2) + i \sin(\theta_1 + \theta_2)) = r_{1}r_{2} \cdot e^{(\theta_1 + \theta_2)i} $$

```qsharp
namespace Kata {
    import Std.Math.*;

    function ComplexPolarMult(x : ComplexPolar, y : ComplexPolar) : ComplexPolar {
        mutable theta = x.Argument + y.Argument;
        if theta > PI() {
            set theta -= 2.0 * PI();
        }
        if theta <= -PI() {
            set theta += 2.0 * PI();
        }
        return ComplexPolar(x.Magnitude * y.Magnitude, theta);
    }
}
```
