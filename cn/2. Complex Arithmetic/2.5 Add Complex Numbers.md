# 添加复数

**输入：**

1. 复数 $x = a + bi$。
2. 复数 $y = c + di$。

**目标：**
以复数形式返回 $x$ 和 $y$ 之和。

<details>
  <summary><b>需要提示吗？</b></summary>
  
复数相加就像多项式相加一样。添加相同类型的组件 - 将实部添加到实部，将虚部添加到虚部。

有关复数相加的视频说明可以在[此处](https://www.youtube.com/watch?v=SfbjqVyQljk)找到。
</details>

> `Std.Math` 命名空间中的 Q# 函数 `PlusC` 添加两个复数。出于教育目的，请尝试手动完成此任务。

## 解决方案注释

两个复数的加法可以通过分别将数字的实部和虚部相加来完成：

$$x + y = (a + bi) + (c + di) = \underset{real}{\underbrace{(a + c)}} + \underset{imaginary}{\underbrace{(b + d)}}i $$

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function ComplexAdd(x : Complex, y : Complex) : Complex {
        Complex(x.Real + y.Real, x.Imag + y.Imag)
    }
}
```
