# 将笛卡尔坐标转换为极坐标

**输入：**
复数 $x = a + bi$。

**目标：**
返回 $x = re^{i\theta}$ 的极坐标表示，即距原点 $r$ 和相位 $\theta$ 的距离作为 `ComplexPolar`。

* $r$ 应该是非负数：$r \geq 0$
* $\theta$ 应介于 $-\pi$ 和 $\pi$ 之间：$-\pi < \theta \leq \pi$

> `Std.Math` 命名空间中的 Q# 函数 `ComplexAsComplexPolar` 将 `Complex` 类型的复数转换为 `ComplexPolar` 类型的复数。出于教育目的，请尝试手动完成此任务。

## 解决方案注释

您需要计算复平面中的 $r$ 和 $\theta$ 值。 
$r$ 您应该已经很熟悉了，因为它是数字的模数（练习 6）：

$$ r = \sqrt{a^2 + b^2} $$

$\theta$ 可以使用三角学来计算：因为您知道数字的极坐标和笛卡尔形式表示相同的值，所以您可以写

$$ re^{i \theta} = a + bi $$

欧拉公式允许我们将方程的左边部分表示为

$$ re^{i \theta} = r \cos \theta + i r \sin \theta $$

要使两个复数相等，它们的实部和虚部必须相等。这给出了以下方程组：

$$ \begin{cases} a = r \cos \theta \\ b = r \sin \theta \end{cases} $$

要计算 $\theta$，您可以将第二个方程除以第一个方程得到

$$ \tan \theta = \frac{b}{a} $$

$$ \theta = \arctan \left(\frac{b}{a}\right) $$

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function ComplexToComplexPolar(x : Complex) : ComplexPolar {
        let (a, b) = (x.Real, x.Imag);
        return ComplexPolar(Sqrt(a * a + b * b), ArcTan2(b, a));
    }
}
```
