# 将极坐标转换为笛卡尔坐标

**输入：**
极坐标形式的复数 $x = re^{i\theta}$。

**目标：**
将 $x = a + bi$ 的笛卡尔表示形式返回为 `Complex`。

> `Std.Math` 命名空间中的 Q# 函数 `ComplexPolarAsComplex` 将 `ComplexPolar` 类型的复数转换为 `Complex` 类型的复数。出于教育目的，请尝试手动完成此任务。

## 解决方案注释

欧拉公式允许您将复极数表示为

$$ re^{i \theta} = \underset{real}{\underbrace{r \cos \theta}} + i \underset{imaginary}{\underbrace{r \sin \theta}}$$

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function ComplexPolarToComplex(x : ComplexPolar) : Complex {
        let (r, theta) = (x.Magnitude, x.Argument);
        return Complex(r * Cos(theta), r * Sin(theta));
    }
}
```
