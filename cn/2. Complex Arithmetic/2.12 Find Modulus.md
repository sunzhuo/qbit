# 求模数

**输入：** 复数 $x = a + bi$。

**目标：** 返回该数字的模数，$|x|$。

## 解决方案注释

复数的模可以看作是复平面中从原点 0 到点 $x$ 的距离。这可以使用毕达哥拉斯定理来计算：$c^2 = a^2 + b^2$，这意味着$ c = \sqrt{a^2 + b^2}$

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function ComplexModulus(x : Complex) : Double {
        let (a, b) = (x.Real, x.Imag);
        return Sqrt(a * a + b * b);
    }
}
```
