# 求复数指数

**输入：** 复数 $x = a + bi$。

**目标：** 返回复数 $e^x = e^{a + bi}$。

<details>
  <summary><b>需要提示吗？</b></summary>
  Q# 命名空间 <code>Std.Math</code> 包含一个函数 <code>E()</code>，该函数返回常量 $e$ 的值。
</details>

## 解决方案注释

首先，您将表达式 $e^{a + bi}$ 重写为两个更简单表达式的乘积：$ e^a \cdot\ e^{bi} $。
第一部分是实数。
第二部分可以使用公式$e^{i\theta} = \cos \theta + i\sin \theta$来表达。
将其代入表达式可得出：
$$ e^a(\cos b + i\sin b) = \underset{real}{\underbrace{e^a \cos b}} + \underset{imaginary}{\underbrace{e^a \sin b}}i  $$

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function ComplexExponent(x : Complex) : Complex {
        Complex(E()^x.Real * Cos(x.Imag), E()^x.Real * Sin(x.Imag))
    }
}
```
