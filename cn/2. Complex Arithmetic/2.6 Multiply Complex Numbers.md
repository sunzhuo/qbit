# 复数相乘

**输入：**

1. 复数 $x = a + bi$。
2. 复数 $y = c + di$。

**目标：**
以复数形式返回 $x$ 和 $y$ 的乘积。

<details>
  <summary><b>需要提示吗？</b></summary>
  
复数相乘就像多项式相乘一样。分配复数之一：
$$(a + bi)(c + di) = a(c + di) + bi(c + di)$$ 
然后相乘，记住 $i^2=-1$，并将实部和虚部组合在一起。

## 解决方案注释

复数相乘就像多项式相乘一样。分配一个复数，然后相乘（记住 $i^2 =-1$），并将实部和虚部组合在一起：

$x \cdot y = (a + bi)(c + di) = a \cdot c + a \cdot di + c \cdot bi + bi \cdot di = \underset{real}{\underbrace{a \cdot c - b \cdot d}} + \underset{imaginary}{\underbrace{(a \cdot d + c \cdot b)}}i $

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function ComplexMult(x : Complex, y : Complex) : Complex {
        let (a, b) = (x.Real, x.Imag);
        let (c, d) = (y.Real, y.Imag);
        return Complex(a * c - b * d, a * d + b * c);
    }
}
```
