# 求共轭

**输入：**
复数 $x = a + bi$。

**目标：**
返回 $x$ $\overline{x} = a - bi$ 的复共轭。

<details>
  <summary><b>需要提示吗？</b></summary>
  
可以在[此处](https://www.youtube.com/watch?v=BZxZ_eEuJBM)找到复数共轭的视频说明。

</details>

## 解决方案注释

要获得复数的复共轭，请将该复数的虚部乘以 $-1$：

$$\overline{x} = a - bi$$

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function ComplexConjugate(x : Complex) : Complex {
        Complex(x.Real, -x.Imag)
    }
}
```
