# 求实数的复幂

**输入：**

1. 非负实数 $r$。
2. 复数 $x = a + bi$。

**目标：** 返回复数 $r^x = r^{a + bi}$。

<details>
  <summary><b>需要提示吗？</b></summary>
  您可以使用 $r = e^{\ln r}$ 来转换指数基数。但请记住，$\ln r$ 仅针对正数定义 - 请务必单独检查 $r = 0$！

Q# 命名空间 `Std.Math` 包括有用的函数 `Log()`、`Sin()` 和 `Cos()`。
</details>

## 解决方案注释

重写表达式 $r^x$ 以使用欧拉常数，这将允许您使用类似于上一个练习的解决方案的方法。

首先，将 $r^x$ 重写为两次幂的乘积：$$ r^{a+bi} = r^a \cdot r^{bi} $$

鉴于 $r = e^{\ln r} $（$\ln$ 是自然对数），您可以将乘积的第二部分重写如下：
    $$ r^{bi} =  e^{bi\ln r} $$

现在，给定 $e^{i\theta} = \cos \theta + i\sin \theta$，您可以进一步重写它​​，如下所示：
    $$ e^{bi\ln r} = \cos( b \cdot \ln r) + i \sin(b \cdot \ln r) $$

将其代入原始表达式时，您将得到：
    $$ \underset{real}{\underbrace{r^a \cos(b \cdot \ln r)}} + \underset{imaginary}{\underbrace{r^a \sin(b \cdot \ln r)}} i $$

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function ComplexExpReal(r : Double, x : Complex) : Complex {
        if AbsD(r) < 1e-9 {
            return Complex(0., 0.);
        }

        let ra = r^x.Real;
        let lnr = Log(r);
        return Complex(ra * Cos(x.Imag * lnr), ra * Sin(x.Imag * lnr));
    }
}
```
