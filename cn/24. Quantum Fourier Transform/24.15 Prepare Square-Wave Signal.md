# 准备方波信号

**输入：** $n \ge 2$ 量子位处于 $\ket{0 \dots 0}$ 状态。

**目标：** 将量子位的状态更改为具有交替 $1, 1, -1, -1$ 基础状态振幅的周期性状态：

$$\frac1{\sqrt{2^n}} \big(\ket{0} + \ket{1} - \ket{2} - \ket{3} + ... - \ket{2^n-2} - \ket{2^n-1} \big)$$

> 例如，对于 $n = 2$，目标状态为 $\frac12 \big(\ket{0} + \ket{1} - \ket{2} - \ket{3}\big)$。

## 解决方案注释

用二进制表示法而不是整数编写的目标状态可以将其术语分为四组，如下所示：

$$ \ket{...00} + \ket{...01} - \ket{...10} - \ket{...11} $$

以 $1$ 作为第二最低有效位的状态具有 $-1$ 相对相位，并且该位中具有 $0$ 的状态具有 $1$ 相对相位。目标状态写为单量子位状态的张量积，如下所示：

$$\frac1{\sqrt{2^n}} (\ket{0} + \ket{1}) \otimes (\ket{0} + \ket{1}) \otimes ... \otimes (\ket{0} - \ket{1}) \otimes (\ket{0} + \ket{1})$$

您可以尝试通过仅将 $j_2$ 设置为 $1$ 并将 $j$ 的其余位设置为 $0$ 来准备此状态，然后应用 QFT。这将在倒数第二个量子位上添加 $-1$ 所需的相对相位，但也会在最后一个量子位上添加 $i$ 的相对相位：

$$QFT(\ket{010...0}) = \frac1{\sqrt{2^n}} (\ket{0} + \ket{1}) \otimes (\ket{0} + \ket{1}) \otimes ... \otimes (\ket{0} - \ket{1}) \otimes (\ket{0} + i\ket{1})$$

要修复最后一个量子位状态，您需要以某种形式取消最后一个 $i$ 相对相位。这可以通过使用以下状态作为 QFT 的输入来实现：

$$\frac1{\sqrt2} \big(e^{-i\pi/4} \ket{010...0} + e^{i\pi/4} \ket{110...0}\big)$$

您可以使用 $e^{\pm i\pi/4} = \frac{1 \pm i}{\sqrt2}$ 的事实，写出将 QFT 应用于此状态的结果，如下所示：

$$QFT\bigg(\frac1{\sqrt2} \big(e^{-i\pi/4} \ket{010...0} + e^{i\pi/4} \ket{110...0}\big)\bigg) =$$
$$= \frac1{\sqrt2}\bigg( \frac{1-i}{\sqrt2} QFT\ket{010...0} + \frac{1+i}{\sqrt2} QFT\ket{110...0} \bigg) = $$
$$= \frac{(1-i)}{2\sqrt{2^n}} (\ket{0} + \ket{1}) \otimes (\ket{0} + \ket{1}) \otimes ... \otimes (\ket{0} - \ket{1}) \otimes (\ket{0} + i\ket{1}) + $$
$$+ \frac{(1+i)}{2\sqrt{2^n}} (\ket{0} + \ket{1}) \otimes (\ket{0} + \ket{1}) \otimes ... \otimes (\ket{0} - \ket{1}) \otimes (\ket{0} - i\ket{1}) =$$
$$= \frac{1}{\sqrt{2^{n}}} (\ket{0} + \ket{1}) \otimes (\ket{0} + \ket{1}) \otimes ... \otimes (\ket{0} - \ket{1}) \otimes (\ket{0} + \ket{1})$$
 
除了更标准的 $X$ 和 $H$ 门之外，还可以使用 $T$ 门及其伴随门来创建所需的初始叠加状态：

1. 将 $H$ 门应用于第一个量子位，将 $X$ 门应用于第二个量子位：
   $$\frac1{\sqrt2} \big(\ket{010...0} + \ket{110...0}\big)$$
2. 通过将 $T$ 门应用于第一个量子位，将相对相位 $e^{i\pi/4}$ 添加到第二项：
   $$\frac1{\sqrt2} \big(\ket{010...0} + e^{i\pi/4} \ket{110...0}\big)$$
3. 通过将包裹在 $X$ 门中的伴随 $T$ 门应用到第一个量子位，将相对相位 $e^{-i\pi/4}$ 添加到第一项（这样，相对相位将应用于该量子位的 $\ket{0}$ 基础状态）：
   $$\frac1{\sqrt2} \big(e^{-i\pi/4} \ket{010...0} + e^{i\pi/4} \ket{110...0}\big)$$

```qsharp
namespace Kata {
    import Std.Arrays.*;

    operation SquareWave(qs : Qubit[]) : Unit is Adj + Ctl {
        X(qs[1]);
        H(qs[0]);
        T(qs[0]);
        X(qs[0]);
        Adjoint T(qs[0]);
        X(qs[0]);
        QFT(qs);
    }

    operation QFT(qs : Qubit[]) : Unit is Adj + Ctl {
        ApplyQFT(Reversed(qs));
        SwapReverseRegister(qs);
    }
}
```
