# 获取信号频率

**输入：**
$n$ 状态中的量子位

$$\frac1{\sqrt{2^n}} \sum_{k=0}^{2^n-1} e^{2\pi i \cdot \frac{Fk}{2^{n}}} \ket{k}$$

**输出：** 在此状态 ($0\leq F\leq 2^n-1$) 中编码的“信号”的频率 $F$。

> 例如，对于 $n = 2$ 和状态 $\frac12\big(\ket{0} + i\ket{1} - \ket{2} - i\ket{3} \big)$，输出应为 $F = 1$。

操作结束时量子位的状态并不重要。

## 解决方案注释

经典傅里叶变换产生时域信号在频域的分解，而逆傅里叶变换产生频域信号的时域表示。

请记住这一点，如果您的状态是将 QFT 应用于基础状态 $\ket{F}$，则可以通过将逆 QFT 应用于该状态，然后测量量子位以找到结果基础状态来恢复值 $F$。

请记住，QFT 以大端表示法作用于数组，并且 `MeasureInteger` 假设输入采用小端表示法，因此您需要在测量寄存器并将结果转换为整数之前反转寄存器。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Arrays.*;

    operation SignalFrequency(qs : Qubit[]) : Int {
        Adjoint QFT(qs);
        return MeasureInteger(Reversed(qs));
    }

    operation QFT(qs : Qubit[]) : Unit is Adj + Ctl {
        ApplyQFT(Reversed(qs));
        SwapReverseRegister(qs);
    }
}
```
