# 旋转门

**输入**：

1. 处于状态 $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$ 的量子位。
2. 整数 $k \geq 0$。

**目标**： 
将量子位的状态更改为 $\alpha \ket{0} + \beta \cdot e^{\frac{2\pi i}{2^{k}}} \ket{1}$

> 注意不要引入额外的全局阶段！ 
这在后面的练习中很重要。

## 解决方案注释

为了避免添加额外的全局阶段，我们必须使用不修改 $\ket{0}$ 状态且仅影响 $\ket{1}$ 的门：即 $U\ket{\psi} = \alpha\ket{0} + \beta \cdot U\ket{1}$。

内置的 [R1Frac 门](https://learn.microsoft.com/qsharp/api/qsharp-lang/microsoft.quantum.intrinsic/r1frac) 正是这样做的：

$$\textrm{R1Frac}(n,k) = \begin{bmatrix} 1 & 0 \\ 0 & e^{i\pi n/2^{k}} \end{bmatrix} $$

我们指定 $n=2$ 来获取所需的转换：

$$\textrm{R1Frac}(2,k) = \begin{bmatrix} 1 & 0 \\ 0 & e^{2\pi i/2^{k}} \end{bmatrix} $$

来源：Solution.qs

```qsharp
namespace Kata {
    operation Rotation(q : Qubit, k : Int) : Unit is Adj + Ctl {
        R1Frac(2, k, q);
    }
}
```
