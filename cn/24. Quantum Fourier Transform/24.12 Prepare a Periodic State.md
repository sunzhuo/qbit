# 准备一个周期性状态

**输入：**

1. $n$ 个量子位处于 $\ket{0 \dots 0}$ 状态。
2. 整数频率 $F$ ($0 \leq F \leq 2^{n}-1$)。

**目标：** 将量子位的状态更改为频率为 $F$ 的周期性状态：

$$\frac1{\sqrt{2^n}} \sum_{k=0}^{2^n-1} e^{2\pi i \cdot \frac{Fk}{2^{n}}} \ket{k}$$

> 例如，对于 $n = 2$ 和 $F = 1$，目标状态为 $\frac12\big(\ket{0} + i\ket{1} - \ket{2} - i\ket{3} \big)$。

## 解决方案注释

回想一下 QFT 的定义： 
对于基本状态 $\ket{j}$，QFT 定义为

$$QFT\ket{j}= \frac1{\sqrt{2^n}} \sum_{k=0}^{2^n-1} e^{2\pi i \cdot j k/2^{n}} \ket{k}$$

您可以看到使用 $j = F$ 将准确产生所需的状态！

要在输入寄存器上准备状态 $\ket{F}$，您需要将整数值 $F$ 转换为大端二进制表示形式并将其编码到寄存器中。

> 编写代码时，一定要注意库函数是使用小端还是大端！  特别是，`IntAsBoolArray`使用little endian编码，而QFT运算使用big endian，因此必须先将整数转换为位串的结果反转，然后再将其编码到寄存器中。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Arrays.*;
    import Std.Convert.*;

    operation PeriodicState(qs : Qubit[], F : Int) : Unit is Adj + Ctl {
        let bitsBE = Reversed(IntAsBoolArray(F, Length(qs)));
        ApplyPauliFromBitString(PauliX, true, bitsBE, qs);
        QFT(qs);
    }

    operation QFT(qs : Qubit[]) : Unit is Adj + Ctl {
        ApplyQFT(Reversed(qs));
        SwapReverseRegister(qs);
    }
}
```
