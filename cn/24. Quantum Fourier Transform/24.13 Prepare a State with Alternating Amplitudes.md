# 准备一个具有交替振幅的状态

**输入：** $n$ 量子位处于 $\ket{0 \dots 0}$ 状态。

**目标：** 将量子位的状态更改为周期状态，并具有交替的 $1$ 和 $-1$ 基本状态幅度：

$$\frac1{\sqrt{2^n}} \big(\ket{0} - \ket{1} + \ket{2} - \ket{3} + ... - \ket{2^n-1} \big)$$

> 例如，对于 $n = 2$，目标状态为 $\frac12 \big(\ket{0} - \ket{1} + \ket{2} - \ket{3}\big)$。

## 解决方案注释

由于该状态的幅度没有虚部，因此很明显，任意幅度 $e^{2\pi i \cdot \frac{Fk}{2^n}}$ 的指数值必须是 $2\pi i$ 才能获得等于 $1$ 的幅度，或者对于 $-1$ 的幅度而言 $\pi i$。 
这意味着只有 $j_1$ 可以是 $1$：$F$ 的所有其他位必须是 $0$。 
事实上，您可以检查使用 $F = 10...0 = 2^{n - 1}$ 和上一个任务的解决方案是否会产生所需的状态。

您可以简化此状态的准备工作：您可以将 $X$ 门应用于寄存器的第一个量子位，而不是在寄存器中使用 $F$ 的通用编码。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Arrays.*;

    operation AlternatingAmplitudes(qs : Qubit[]) : Unit is Adj + Ctl {
        X(qs[0]);
        QFT(qs);
    }

    operation QFT(qs : Qubit[]) : Unit is Adj + Ctl {
        ApplyQFT(Reversed(qs));
        SwapReverseRegister(qs);
    }
}
```
