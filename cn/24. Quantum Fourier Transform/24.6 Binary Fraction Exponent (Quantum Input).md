# 二进制分数指数（量子输入）

**输入**：

1. 处于状态 $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$ 的量子位。
2. 处于状态 $\ket{j_1 j_2 ... j_n}$ 的 $n$ 量子位 $j$ 数组。

**目标**： 
将输入量子位的状态从 $(\alpha \ket{0} + \beta \ket{1}) \otimes \ket{j_1 j_2 ... j_n}$ 更改为 $(\alpha \ket{0} + \beta \cdot e^{2\pi i \cdot 0.j_1 j_2 ... j_n} \ket{1}) \otimes \ket{j_1 j_2 ... j_n}$，其中 $0.j_1 j_2 ... j_n$ 是与量子位数组 $j$ 的基本状态 $j_1 j_2 ... j_n$ 相对应的二进制分数。

> 量子位数组也可以叠加；
在这种情况下，变换的行为由其基于基态的行为和酉变换的线性度来定义。

## 解决方案注释

从练习的目标来看，您可以看到寄存器 $j$ 必须保持不变，而第一个输入量子位应获得一个取决于寄存器 $j$ 中量子位值的相位。

由于 $j$ 是一个量子寄存器，并且可以处于基本状态的叠加状态，因此您不能只测量寄存器，然后使用测量结果作为第二个参数来应用上一个任务中的操作。相反，您必须将上一个任务的解决方案从使用具有经典条件的 $\textrm{R1Frac}$ 门转换为将它们用作受控操作，并将寄存器 $j$ 的量子位作为量子条件。您可以使用 `Controlled` 函子来完成此操作。

来源：Solution.qs

```qsharp
namespace Kata {
    operation BinaryFractionQuantum(q : Qubit, j : Qubit[]) : Unit is Adj + Ctl {
        for ind in 0 .. Length(j) - 1 {
            Controlled R1Frac([j[ind]], (2, ind + 1, q));
        }
    }
}
```
