# 二进制分数指数（经典输入）

**输入**：

1. 处于状态 $\ket{\psi} = \alpha \ket{0} + \beta \ket{1}$ 的量子位。
2. $n$ 位 $[j_1, j_2, ..., j_n]$ 的数组，存储为 `Bool[]`。

**目标**： 
将量子位的状态更改为 $\alpha \ket{0} + \beta \cdot e^{2\pi i \cdot 0.j_1 j_2 ... j_n} \ket{1}$，其中 $0.j_1 j_2 ... j_n$ 是大端表示法中的二进制分数，类似于十进制分数：

$$0.j_1 j_2 ... j_n = j_1 \cdot \frac{1}{2^1} + j_2 \cdot \frac{1}{2^2} + ... + j_n \cdot \frac{1}{2^n}$$

## 解决方案注释

由于您可以将总和的指数表示为指数 ($e^{a+b} = e^a \cdot e^b$) 的乘积，因此您可以将所需的转换实现为来自上一个任务的旋转门序列，并增加 $k$。

每个单独的旋转将使用位 $j_k$ 来决定是否应用旋转（您只想在 $j_k$ 为 true 时应用旋转），并使用该位的索引 $k$ 来定义旋转角度。
应用于每个 $k$ 的门将是：

$$U_k = \begin{cases}
I, j_k=0 \\
\textrm{R1Frac}(2, k), j_k=1
\end{cases}$$

回想一下

$$ \textrm{R1Frac}(2, k)( \alpha \ket{0} + \beta \ket{1}) = \alpha \ket{0} + \beta \cdot e^{2\pi i/2^{k}} \ket{1} = \alpha \ket{0} + \beta \cdot e^{2\pi i \cdot 0. \underset{k-1}{\underbrace{0\dots0}} 1} \ket{1}$$

这意味着每个 $k$ 的门 $U_k$ 的总体效果是：

$$U_k (\alpha \ket{0} + \beta \ket{1}) = \alpha \ket{0} + \beta \cdot e^{2\pi i \cdot 0. \underset{k-1}{\underbrace{0\dots0}} j_k} \ket{1}$$

当您迭代 $k$ 时，结果状态将越来越接近所需的状态：

来源：SolutionA.qs

```qsharp
namespace Kata {
    operation BinaryFractionClassical(q : Qubit, j : Bool[]) : Unit is Adj + Ctl {
        for ind in 0 .. Length(j) - 1 {
            if j[ind] {
                R1Frac(2, ind + 1, q);
            }
        }
    }
}
```

或者，如果将数组 $j$ 转换为旋转角度，则可以使用 $R1$ 门在单次旋转中完成此操作。您需要角度 $2\pi \cdot 0.j_1j_2 \dots j_n$，并且可以通过将位字符串转换为整数 $j$（使用大端表示法）并将其除以 $2^n$ 来计算该分数。

单独考虑时，该解决方案更好，因为它只涉及一次旋转而不是一系列旋转。然而，一旦你开始下一个任务，它就不会像第一个任务那么有帮助了！

来源：SolutionB.qs

```qsharp
namespace Kata {
    import Std.Arrays.*;
    import Std.Convert.*;
    import Std.Math.*;

    operation BinaryFractionClassical(q : Qubit, j : Bool[]) : Unit is Adj + Ctl {
        let n = Length(j);
        let jIntBE = BoolArrayAsInt(Reversed(j));
        R1(2.0 * PI() * IntAsDouble(jIntBE) / IntAsDouble(1 <<< n), q);
    }
}
```
