# 就地二进制分数指数（量子输入）

**输入**：
处于状态 $\ket{j_1 j_2 ... j_n}$ 的 $n$ 量子位寄存器。

**目标**： 
将寄存器的状态从 $\ket{j_1} \otimes \ket{j_2 ... j_n}$ 更改为 $\frac1{\sqrt2}(\ket{0} + e^{2\pi i \cdot 0.j_1 j_2 ... j_n} \ket{1}) \otimes \ket{j_2 ... j_n}$。

<details>
  <summary><b>需要提示吗？</b></summary>
  
此任务与上一个任务非常相似，但数字 $j_1$ 必须就地编码。您可以使用 kata 的第一个任务“单量子位 QFT”来完成此操作。
</details>

## 解决方案注释

首先，让我们回顾一下 kata 的第一个任务：应用于单个量子位 $H\ket{j_1}$ 的 Hadamard 门将给出
$\frac1{\sqrt2}(\ket{0} + \ket{1})$ 或 $\frac{1}{\sqrt{2}}(\ket{0} - \ket{1})$ 取决于 $\ket{j_1}$ 的状态。这个操作也可以写成
$$H\ket{j_1} = \frac1{\sqrt2} \big(\ket{0} + e^{2\pi i \cdot \frac{j_1}{2}}\ket{1} \big)= \frac1{\sqrt2} \big(\ket{0} + e^{2\pi i \cdot 0.j_1}\ket{1} \big)$$

因此，如果起始寄存器状态为 $\ket{j_1 j_2 ... j_n}$，则将 Hadamard 门应用于第一个量子位将导致：

$$\big(H_1\otimes I_{n-1} \big) \big(\ket{j_1} \otimes \ket{j_2 ... j_n} \big)= \frac1{\sqrt2} \big(\ket{0} + e^{2\pi i \cdot 0.j_1}\ket{1} \big) \otimes \ket{j_2 ... j_n} $$

之后，我们可以重复我们在上一个任务中对量子位 $\ket{j_2 ... j_n}$ 使用的循环，以第一个量子位为目标，通过受控的 $\textrm{R1Frac}$ 门调整剩余的相位项。

来源：Solution.qs

```qsharp
namespace Kata {
    operation BinaryFractionQuantumInPlace(j : Qubit[]) : Unit is Adj + Ctl {
        H(j[0]);
        for ind in 1 .. Length(j) - 1 {
            Controlled R1Frac([j[ind]], (2, ind + 1, j[0]));
        }
    }
}
```
