# 量子傅立叶变换

**输入**：
处于状态 $\ket{j_1 j_2 ... j_n}$ 的 $n$ 量子位寄存器。

**目标**： 
对输入寄存器应用量子傅里叶变换，即将其变换为一个状态

$$\begin{align*}
&\frac1{\sqrt2} \big(\ket{0} + e^{2\pi i \cdot 0.j_n} \ket{1} \big) \otimes \\
\otimes &\frac1{\sqrt2} \big(\ket{0} + e^{2\pi i \cdot 0.j_{n-1} j_n} \ket{1} \big) \otimes ... \otimes \\
\otimes &\frac1{\sqrt2} \big(\ket{0} + e^{2\pi i \cdot 0.j_1 j_2 ... j_{n-1} j_n} \ket{1} \big)
\end{align*}$$

量子位的寄存器也可以是叠加的；
这种情况下的行为是由基态行为和酉变换的线性度定义的。

> 请注意，此实现将允许您使用逆 QFT，而无需编写任何额外的代码，只需使用此操作的伴随即可！

<details>
  <summary><b>需要提示吗？</b></summary>
  
考虑首先准备一个状态，其中各个量子位的状态相反，然后将其转换为目标状态。
</details>

## 解决方案注释

让我们使用提示并首先准备所描述的状态并反转量子位：

$$\begin{align*}
&\frac1{\sqrt2} \big(\ket{0} + e^{2\pi i \cdot 0.j_1 j_2 ... j_{n-1} j_n} \ket{1} \big) \otimes \\
&\frac1{\sqrt2} \big(\ket{0} + e^{2\pi i \cdot 0.j_2 ... j_{n-1} j_n} \ket{1} \big) \otimes ... \otimes \\
\otimes &\frac1{\sqrt2} \big(\ket{0} + e^{2\pi i \cdot 0.j_{n-1} j_n} \ket{1} \big) \otimes \\
\otimes &\frac1{\sqrt2} \big(\ket{0} + e^{2\pi i \cdot 0.j_n} \ket{1} \big)
\end{align*}$$

您已经找到了一种在上一个任务中就地准备二进制分数指数（第一个量子位的所需状态）的方法。可以通过以下方式准备状态 $\frac1{\sqrt2} \big(\ket{0} + e^{2\pi i \cdot 0.j_1 j_2 ... j_{n-1} j_n} \ket{1} \big) \otimes \ket{j_2 ... j_n}$：首先将 Hadamard 门应用于状态 $\ket{j_1}$ 中的第一个量子位，然后使用量子位 $\ket{j_k}$ 应用一系列受控旋转，并以递增的 $k$ 值作为控制，以便从 $e^{2\pi i \cdot j_2/2^2}$ 一直到额外的相位项每次旋转都会添加 $e^{2\pi i \cdot j_n/2^{n}}$。

这将使第一个量子位处于正确的状态。您可以看到 $j_1$ 没有出现在目标状态表达式的其余部分中，因此您不需要在代码的其余部分中使用它。

为了将剩余的量子位准备在正确的状态，您可以从该状态开始向后工作。
序列中的第二个量子位需要在状态 $\frac{1}{\sqrt{2}} \big(\ket{0} + e^{2\pi i \cdot 0.j_2 ... j_{n-1} j_n} \ket{1} \big)$ 下准备。 
与第一个量子位类似，您可以通过将 Hadamard 门应用到量子位 $\ket{j_2}$，然后使用量子位 $\ket{j_3}$ 到 $\ket{j_n}$ 将 $n-2$ 控制的旋转门应用到第二个量子位来实现此目的。 
执行此操作后，系统总状态将是：

$$\frac{1}{\sqrt{2}} \big(\ket{0} + e^{2\pi i \cdot 0.j_1 j_2 ... j_{n-1} j_n} \ket{1} \big)\otimes
\frac{1}{\sqrt{2}} \big(\ket{0} + e^{2\pi i \cdot 0.j_2 ... j_{n-1} j_n} \ket{1} \big) \otimes \ket{j_3 ... j_n}$$

这两个步骤让我们看到一个模式：对于每个量子位 $j_k, k = 1 .. n$ ：

1. 将 Hadamard 门应用到 $|j_k\rangle$。
2. 将受控旋转运算符 $n-k$ 次应用于量子位 $|j_k\rangle$，使用量子位 $|j_{k+1}\rangle$ 到 $|j_n\rangle$ 作为对照，相位对应于从 $2^2$ 到 $2^{n-k+1}$ 的分数。

这些步骤的效果将是准备几乎与所需状态匹配的状态，但与它相比量子位顺序相反。 
您可以通过使用一系列 $\textrm{SWAP}$ 门来反转状态中量子位的顺序来解决此问题。

来源：Solution.qs

```qsharp
namespace Kata {
    operation QuantumFourierTransform(j : Qubit[]) : Unit is Adj + Ctl {
        let n = Length(j);
        for ind in 0 .. n - 1 {
            BinaryFractionQuantumInPlace(j[ind ...]);
        }
        for ind in 0 .. n / 2 - 1 {
            SWAP(j[ind], j[n - 1 - ind]);
        }        
    }

    operation BinaryFractionQuantumInPlace(j : Qubit[]) : Unit is Adj + Ctl {
        H(j[0]);
        for ind in 1 .. Length(j) - 1 {
            Controlled R1Frac([j[ind]], (2, ind + 1, j[0]));
        }
    }
}
```
