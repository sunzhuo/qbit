# 准备所有基向量的均匀叠加

**输入：** $n$ 量子位处于 $\ket{0 \dots 0}$ 状态。

**目标：** 将量子位的状态更改为所有基向量 $\frac{1}{\sqrt{2^N}} \big (\ket{0 \dots 0} + \dots + \ket{1 \dots 1}\big)$ 的相等叠加。

## 解决方案注释

您可以将所有基础状态所需的相等叠加视为状态 $\ket{0...0}$ 的 QFT。事实上，如果 $j_1 = j_2 = ... = j_n = 0$, $j = 0$，并且您会得到以下状态：

$$\frac1{\sqrt{2^n}} \sum_{k=0}^{2^n-1} e^{2\pi i \cdot \frac{jk}{2^{n}}} \ket{k} = \frac{1}{\sqrt{2^n}} \sum_{k=0}^{2^n-1} e^{0} \ket{k} = \frac{1}{\sqrt{2^n}} \sum_{k=0}^{2^n-1} \ket{k}$$

这意味着您只需将 QFT 应用于给定的量子位寄存器即可解决此任务。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Arrays.*;

    operation AllBasisVectors(qs : Qubit[]) : Unit is Adj + Ctl {
        QFT(qs);
    }

    operation QFT(qs : Qubit[]) : Unit is Adj + Ctl {
        ApplyQFT(Reversed(qs));
        SwapReverseRegister(qs);
    }
}
```
