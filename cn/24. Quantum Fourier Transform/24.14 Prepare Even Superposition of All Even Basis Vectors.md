# 准备所有偶数基向量的偶数叠加

**输入：** $n$ 量子位处于 $\ket{0 \dots 0}$ 状态。

**目标：** 将量子位的状态更改为所有偶数基向量的相等叠加：

$$\frac1{\sqrt{2^{n-1}}} \big(\ket{0} + \ket{2} + ... + \ket{2^n-2} \big)$$

> 例如，对于 $n = 2$，目标状态为 $\frac1{\sqrt2} \big(\ket{0} + \ket{2}\big)$。

## 解决方案注释

您可以看到，本课第一个和第三个任务中的两个状态的均匀叠加将给出所需的结果。的确，

$$\frac1{\sqrt{2^{n-1}}} \big(\ket{0} + \ket{2} + ... + \ket{2^n-2}\big) =$$
$$=\frac{1}{\sqrt{2}} \bigg( \frac1{\sqrt{2^n}} \big(\ket{0} - \ket{1} + \ket{2} - \ket{3} + ... - \ket{2^n-1}\big) + \frac1{\sqrt{2^n}} \big(\ket{0} + \ket{1} + \ket{2} + \ket{3} + ... + \ket{2^n-1}\big)\bigg)$$

现在，您可以利用 QFT 是线性变换这一事实。
早期任务中的两个周期性状态是通过将 QFT 应用于状态 $\ket{100...0}$ 和 $\ket{000...0}$ 状态而创建的。 
要准备这两种状态的相等叠加，您可以将哈达玛门应用于寄存器中的第一个量子位：

$$ \frac1{\sqrt2} \big(\ket{0} + \ket{1}\big) \otimes \ket{00...0} $$

然后将 QFT 应用到寄存器：

$$ QFT\big(\frac1{\sqrt2} (\ket{0} + \ket{1}) \otimes \ket{00...0}\big) = \frac1{\sqrt2} \big( QFT\ket{000...0} + QFT\ket{100...0}\big)$$

这会产生所需的最终状态。

```qsharp
namespace Kata {
    import Std.Arrays.*;

    operation AllEvenVectors(qs : Qubit[]) : Unit is Adj + Ctl {
        H(qs[0]);
        QFT(qs);
    }

    operation QFT(qs : Qubit[]) : Unit is Adj + Ctl {
        ApplyQFT(Reversed(qs));
        SwapReverseRegister(qs);
    }
}
```
