# 产品功能

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。
3.长度为$N$的位向量$r$，表示为`Bool[]`。

**目标：** 
实现一个计算标量积函数的标记预言机
$$f(x) = \bigoplus\limits_{i=0}^{N-1} r_i x_i$$

换句话说，对于每个基础状态 $\ket{x}$，如果 $f(x) = 1$，则翻转目标量子位 $\ket{y}$ 的状态，否则保持不变。
让输入寄存器中的量子位保持其开始时的状态。 
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

此问题与上一问题类似，但这次仅当经典输入 `r[i]` 设置为 `true` 时，输入量子位 `x[j]` 才会影响目标量子位的状态。 
我们也可以使用类似的方法来解决：迭代所有输入量子位，检查相应的输入位 `r[i]` 是否为 `true`，如果是，则应用 $CNOT$ 门，以量子位 `x[j]` 作为控制，以量子位 `y` 作为目标。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Oracle_Product(x : Qubit[], y : Qubit, r : Bool[]) : Unit is Adj + Ctl {
        for i in 0 .. Length(x) - 1 {
            if r[i] {
                CNOT(x[i], y);
            }
        }
    }
}
```
