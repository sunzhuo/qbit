# 第 K 位

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。
3. 整数 $k$ 使得 $0 \le k < N$。

**目标：** 
Implement a marking oracle which calculates the $k$-th bit of the input array, that is, the function
$$f(x) = x_k$$

换句话说，对于每个基础状态 $\ket{x}$，如果 $f(x) = 1$，则翻转目标量子位 $\ket{y}$ 的状态，否则保持不变。
让输入寄存器中的量子位保持其开始时的状态。 
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

由于该预言的效果仅取决于第 $k$ 个量子位的值，因此我们可以忽略其余量子位，而只关注 `x[k]`。如果输入量子位 `x[k]` 处于 $\ket{1}$ 状态，我们需要翻转目标量子位的状态，否则保持不变 - 这正是 $CNOT$ 门的效果，其中量子位 `x[k]` 用作控制，量子位 `y` 用作目标。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Oracle_Kth_Bit(x : Qubit[], y : Qubit, k : Int) : Unit is Adj + Ctl {
        CNOT(x[k], y);
    }
}
```
