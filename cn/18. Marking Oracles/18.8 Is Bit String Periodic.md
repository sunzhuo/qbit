# 位串是周期性的吗？

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。

**目标：** 
实现一个量子预言机，检查输入寄存器是否具有任何周期的周期性，即是否存在一个值 $P < N$，使得对于 $0$ 和 $N - P - 1$ 之间的所有 $j$（含）$x_j = x_{j+P}$。

例如，对于$N = 3$，位串`[false, true, false]`是周期性的，周期为2，因此该位串是周期性的。

让输入寄存器中的量子位保持其开始时的状态。
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

我们需要检查对于 $P$ 的任何值，位串是否具有周期 $P$。 
我们可以使用上一个任务的解决方案作为构建块来检查特定 $P$ 的周期性，但是我们如何从这些块中构建完整的解决方案？

您可以将我们正在评估的函数表示为“位串是周期性的，周期为 $1$”或
“位串是周期性的，周期为$2$”或...或“位串是周期性的，周期为$N − 1$”。
然后，您必须分配 $N - 1$ 辅助量子位，并使用它们存储 $1$ 到 $N - 1$ 期间每个值的条件评估结果。

之后，您需要计算这些辅助量子位的状态的 OR：如果其中至少一个是 $1$，则该位串是周期性的。
您可以使用 Oracles kata 中的“实现 OR Oracle”练习来完成此操作，方法是检查所有辅助量子位的值是否为 $0$，然后对结果求反。

最后，您必须取消计算对辅助量子位所做的更改，以便在释放它们之前将它们返回到 $\ket{0}$ 状态。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Oracle_Periodic (x : Qubit[], y : Qubit) : Unit is Adj + Ctl {
        let N = Length(x);
        use aux = Qubit[N - 1];
        within {
            for P in 1 .. N - 1 {
                Oracle_PeriodicGivenPeriod(x, aux[P - 1], P);
            }
        } apply {
            ApplyControlledOnInt(0, X, aux, y);
            X(y);
        }
    }

    operation Oracle_PeriodicGivenPeriod (x : Qubit[], y : Qubit, p : Int) : Unit is Adj + Ctl {
        let n = Length(x);
        within {
            for i in 0 .. n - p - 1 {
                CNOT(x[i + p], x[i]);
            }
        } apply {
            ApplyControlledOnInt(0, X, x[... n - p - 1], y);
        }
    }   
}
```
