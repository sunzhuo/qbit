# 奇偶校验函数

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。

**目标：** 
实现一个计算输入数组奇偶校验的标记预言机，即函数
$$f(x) = 1 \text{ if } x \text{ has an odd number of 1s, and } 0 \text{ otherwise }$$

换句话说，对于每个基础状态 $\ket{x}$，如果 $f(x) = 1$，则翻转目标量子位 $\ket{y}$ 的状态，否则保持不变。
让输入寄存器中的量子位保持其开始时的状态。 
您的解决方案应该适用于叠加输入，而不是使用任何测量。

<details>
<summary><strong>需要提示吗？</strong></summary>
$f(x)$ 可以表示为 $x_0 \oplus x_1 \oplus ... \oplus x_{N-1}$。
</details>

## 解决方案注释

我们可以这样写$f(x)$：

$$f(x) = x_0 \oplus x_1 \oplus ... \oplus x_{N-1}$$

让我们将这个表达式替换为量子态预言效应的表达式：

$$U_f \ket{x} \ket{y} = \ket{x} \ket{y \oplus f(x)} = \ket{x} \ket{y \oplus x_0 \oplus x_1 \oplus ... \oplus x_{N-1}}$$

现在，我们可以将目标量子位的最终状态表示为一系列按顺序应用的 $N$ 标记预言机的结果，如果输入量子位 $j$ $\ket{x_j}$ 的状态为 $\ket{1}$，则标记预言机 $j$ 翻转其状态。
正如我们在上一个任务中看到的，每个标记预言机都可以实现为单个 $CNOT$ 门。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Oracle_Parity(x : Qubit[], y : Qubit) : Unit is Adj + Ctl {
        for xi in x {
            CNOT(xi, y);
        }
    }
}
```
