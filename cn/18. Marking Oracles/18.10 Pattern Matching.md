# 模式匹配

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。
3. 输入寄存器 ($1 ≤ K ≤ N$) 中 $K$ 个不同索引的数组 $a$。
4.长度为$K$的位模式$r$表示为`Bool[]`。

**目标：** 
实现一个量子预言机，检查输入寄存器是否与给定模式匹配，即给定索引处的位与模式中的相应位匹配（“假”和“真”值分别表示状态 $\ket{0}$ 和 $\ket{1}$）。
输入寄存器 $x_{a_j}$ 的值应与 $0$ 和 $K - 1$（含）之间的所有 $j$ 的模式元素 $r_j$ 匹配。

例如，对于 $N = 3$，索引 `[0, 2]` 处的模式 `[false, true]` 将匹配两个基本状态：$\ket{001}$ 和 $\ket{011}$。

让输入寄存器中的量子位保持其开始时的状态。
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

此任务与上一个任务类似：我们正在评估的函数仅取决于量子位子集的状态，因此我们可以忽略其余的量子位。这里的主要区别是获取我们需要用作控制的量子位集。

`Std.Arrays` 命名空间中的 Q# 库函数 `Subarray` 提取给定数组中给定索引处的元素，这正是我们所需要的。

来源：Solution.qs

```qsharp
namespace Kata {
    import Std.Arrays.*;

    operation Oracle_PatternMatching(x : Qubit[], y : Qubit, a : Int[], r : Bool[]) : Unit is Adj + Ctl {
        let ctrl = Subarray(a, x);
        ApplyControlledOnBitString(r, X, ctrl, y);
    }
}
```
