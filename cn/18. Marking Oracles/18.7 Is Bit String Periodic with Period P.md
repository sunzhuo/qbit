# 位串是否具有周期 P 的周期性？

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。
3. 整数 $P$ 使得 $P < N$。

**目标：** 
实现一个量子预言机，检查输入寄存器是否是周期性的，周期为 $P$，即，对于 $0$ 和 $N - P - 1$ 之间的所有 $j$（包括 $x_j = x_{j+P}$）。

例如，对于 $N = 3$，位串 `[false, true, false]` 是周期性的，周期为 2。

让输入寄存器中的量子位保持其开始时的状态。
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

此问题与上一个问题类似：在该问题中，您必须比较对称位置的量子位的状态，而在这一问题中，您必须比较彼此相距 $P$ 的量子位的状态。
为了使基础状态 $\ket{x_0 ... x_{N-1}}$ 具有周期 $P$，您需要检查量子位对 $x_0$ 和 $x_P$、$x_1$ 和 $x_{P + 1}$ 等是否相同。

我们也可以使用类似的方法来解决：使用 $CNOT$ 门进行就地比较。

如果您按从左到右的顺序迭代这些对，从 $x_0$ 和 $x_P$ 对开始，您需要确保将 XOR 存储在该对的左侧量子位中。当比较量子位 $x_j$ 和 $x_{j + P}$ 的状态时，如果位置 $j + 2P$ 在位串内，则右侧量子位 $x_{j + P}$ 可能会参与稍后的另一次比较，因此您不应在之前的比较中修改其状态。但是，量子位 $x_j$ 已经与量子位 $x_{j-P}$ 进行了比较，因此现在可以安全地修改其状态。

与上一个任务相同，然后我们使用零控制 $X$ 门（以第一个 $N-P$ 量子位作为控制）检查所有 XOR 是否都是 $0$，最后取消计算我们对输入寄存器所做的任何更改。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Oracle_PeriodicGivenPeriod (x : Qubit[], y : Qubit, p : Int) : Unit is Adj + Ctl {
        let n = Length(x);
        within {
            for i in 0 .. n - p - 1 {
                CNOT(x[i + p], x[i]);
            }
        } apply {
            ApplyControlledOnInt(0, X, x[... n - p - 1], y);
        }
    }    
}
```
