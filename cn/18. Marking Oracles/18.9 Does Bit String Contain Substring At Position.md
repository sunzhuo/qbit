# 位串在该位置是否包含子串？

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。
3.长度为$K$的位模式$r$表示为`Bool[]`($1 ≤ K ≤ N$)。
4. 整数 $P$ 使得 $0 ≤ P < N - K$。

**目标：** 
实现一个量子预言机，检查输入寄存器是否包含从给定位置开始的给定模式，即，对于 $0$ 和 $K - 1$ 之间的所有 $j$（包括 $r_j = x_{j+P}$）。 （“假”和“真”值分别表示状态 $\ket{0}$ 和 $\ket{1}$）。

例如，对于 $N = 3$，位串 `[false, true, false]` 包含从索引 1 开始的模式 `[true, false]`。

让输入寄存器中的量子位保持其开始时的状态。
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

在此问题中，我们正在评估的函数的值不取决于 qubit $P$ 之前或 qubit $P + K - 1$ 之后的量子位的状态。这意味着我们可以忽略它们并只考虑重要的量子位 - `x[p .. p + Length(r) - 1]`。

一旦我们这样做了，问题就变得更加简单：如果输入量子位处于给定状态，则翻转目标量子位的状态。这就是具有任意控件的受控门的定义，我们可以使用库操作 `ApplyControlledOnBitString` 轻松应用它。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Oracle_ContainsSubstringAtPosition (x : Qubit[], y : Qubit, r : Bool[], p : Int) : Unit is Adj + Ctl {
        ApplyControlledOnBitString(r, X, x[p .. p + Length(r) - 1], y);
    }
}
```
