# 位串是否包含子串？

**输入：**

1. $N$ 量子位处于任意状态 $\ket{x}$（输入/查询寄存器）。
2. 处于任意状态 $\ket{y}$ 的量子位（输出/目标量子位）。
3.长度为$K$的位模式$r$表示为`Bool[]`($1 ≤ K ≤ N$)。

**目标：** 
实现一个量子预言机，检查输入寄存器是否包含给定模式，即是否存在位置 $P$，使得对于 $0$ 和 $K - 1$（含）之间的所有 $j$，$r_j = x_{j+P}$。

例如，对于 $N = 3$，位串 `[false, true, false]` 包含模式 `[true, false]`（从索引 1 开始）。

让输入寄存器中的量子位保持其开始时的状态。
您的解决方案应该适用于叠加输入，而不是使用任何测量。

## 解决方案注释

此任务与我们检查字符串是否周期性的任务非常相似。 
与那里相同，我们有一个构建块来检查我们的条件（位串是否在特定位置包含给定模式），并且我们的函数是应用不同参数的条件的或（给定模式可以从位置 $0$、$1$、$2$ 等开始）。

解决方案也类似：

1. 分配 $N - K + 1$ 辅助量子位，每个可以作为模式开头的位置一个。
2. 以每个可能的位置作为模式的开始来评估条件，并将结果存储在这些量子位中。
3. 将整体函数评估为辅助量子位值的 OR。
4. 在释放辅助量子位之前取消计算对辅助量子位的状态所做的更改。

来源：Solution.qs

```qsharp
namespace Kata {
    operation Oracle_ContainsSubstring (x : Qubit[], y : Qubit, r : Bool[]) : Unit is Adj + Ctl {
        let N = Length(x);
        let K = Length(r);
        use aux = Qubit[N - K + 1];
        within {
            for P in 0 .. N - K {
                Oracle_ContainsSubstringAtPosition(x, aux[P], r, P);
            }
        } apply {
            ApplyControlledOnInt(0, X, aux, y);
            X(y);
        }
    }

    operation Oracle_ContainsSubstringAtPosition (x : Qubit[], y : Qubit, r : Bool[], p : Int) : Unit is Adj + Ctl {
        ApplyControlledOnBitString(r, X, x[p .. p + Length(r) - 1], y);
    }    
}
```
