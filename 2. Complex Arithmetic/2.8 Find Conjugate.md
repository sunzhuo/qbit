# Find Conjugate

**Input:**
A complex number $x = a + bi$.

**Goal:**
Return the complex conjugate of $x$ $\overline{x} = a - bi$.

  <summary><b>Need a hint?</b></summary>
  
A video explanation of the complex conjugate can be found [here](https://www.youtube.com/watch?v=BZxZ_eEuJBM).

</details>

## Solution Notes

To get the complex conjugate of a complex number, you multiply the imaginary part of that complex number by $-1$:

$$\overline{x} = a - bi$$

Source: Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function ComplexConjugate(x : Complex) : Complex {
        Complex(x.Real, -x.Imag)
    }
}
```
