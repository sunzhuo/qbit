# Find Complex Power of Real Number

**Inputs:**

1. A non-negative real number $r$.
2. A complex number $x = a + bi$.

**Goal:** Return the complex number $r^x = r^{a + bi}$.

<details>
  <summary><b>Need a hint?</b></summary>
  You can use the fact that $r = e^{\ln r}$ to convert exponent bases. Remember though, $\ln r$ is only defined for positive numbers - make sure to check for $r = 0$ separately!

  Q# namespace `Std.Math` includes useful functions `Log()`, `Sin()`, and `Cos()`.
</details>

## Solution Notes

Rewrite the expression $r^x$ to use Euler's constant, which will allow you to use an approach similar to the solution to the previous exercise.

First, rewrite $r^x$ into a product of two powers: $$ r^{a+bi} = r^a \cdot r^{bi} $$

Given that $r = e^{\ln r} $ ($\ln$ is the natural logarithm), you can rewrite the second part of the product as follows: 
    $$ r^{bi} =  e^{bi\ln r} $$

Now, given $e^{i\theta} = \cos \theta + i\sin \theta$, you can rewrite it further as follows: 
    $$ e^{bi\ln r} = \cos( b \cdot \ln r) + i \sin(b \cdot \ln r) $$

When substituting this into the original expression, you get:
    $$ \underset{real}{\underbrace{r^a \cos(b \cdot \ln r)}} + \underset{imaginary}{\underbrace{r^a \sin(b \cdot \ln r)}} i $$

Source: Solution.qs

```qsharp
namespace Kata {
    import Std.Math.*;

    function ComplexExpReal(r : Double, x : Complex) : Complex {
        if AbsD(r) < 1e-9 {
            return Complex(0., 0.);
        }

        let ra = r^x.Real;
        let lnr = Log(r);
        return Complex(ra * Cos(x.Imag * lnr), ra * Sin(x.Imag * lnr));
    }
}
```
