# Prepare Even Superposition of All Basis Vectors

**Input:** $n$ qubits in the $\ket{0 \dots 0}$ state.

**Goal:**  Change the state of the qubits to an equal superposition of all basis vectors $\frac{1}{\sqrt{2^N}} \big (\ket{0 \dots 0} + \dots + \ket{1 \dots 1}\big)$.

## Solution Notes

You can think of the required equal superposition of all basis states as the QFT of the state $\ket{0...0}$. Indeed, if $j_1 = j_2 = ... = j_n = 0$, $j = 0$, and you get the following state:

$$\frac1{\sqrt{2^n}} \sum_{k=0}^{2^n-1} e^{2\pi i \cdot \frac{jk}{2^{n}}} \ket{k} = \frac{1}{\sqrt{2^n}} \sum_{k=0}^{2^n-1} e^{0} \ket{k} = \frac{1}{\sqrt{2^n}} \sum_{k=0}^{2^n-1} \ket{k}$$

This means that you can solve this task by simply applying the QFT to the given qubit register.

Source: Solution.qs

```qsharp
namespace Kata {
    import Std.Arrays.*;

    operation AllBasisVectors(qs : Qubit[]) : Unit is Adj + Ctl {
        QFT(qs);
    }

    operation QFT(qs : Qubit[]) : Unit is Adj + Ctl {
        ApplyQFT(Reversed(qs));
        SwapReverseRegister(qs);
    }
}
```
